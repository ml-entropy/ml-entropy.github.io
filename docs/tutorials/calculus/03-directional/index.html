03. Directional Derivatives

Partial derivatives measure change along coordinate axes. Directional derivatives generalize this to any direction—and reveal why the gradient is so special.

Theory Code Exercises

Definition

The directional derivative of $f$ at point $\mathbf{a}$ in direction $\mathbf{u}$ (a unit vector) is:

Directional Derivative

$$D_{\mathbf{u}} f(\mathbf{a}) = \lim_{h \to 0} \frac{f(\mathbf{a} + h\mathbf{u}) - f(\mathbf{a})}{h}$$ This measures the rate of change of $f$ as we move from $\mathbf{a}$ in direction $\mathbf{u}$.

Notice: partial derivatives are just directional derivatives along coordinate axes!

$$\frac{\partial f}{\partial x} = D_{\mathbf{e}_1} f, \quad \frac{\partial f}{\partial y} = D_{\mathbf{e}_2} f$$

Connection to the Gradient

Here's the key insight: directional derivatives can be computed from the gradient!

Computing Directional Derivatives

$$D_{\mathbf{u}} f = \nabla f \cdot \mathbf{u} = \|\nabla f\| \cos\theta$$ where $\theta$ is the angle between $\nabla f$ and $\mathbf{u}$.

This formula reveals everything:

  • Maximum when $\theta = 0$ (direction of $\nabla f$): $D_{\mathbf{u}} f = \|\nabla f\|$
  • Minimum when $\theta = \pi$ (opposite to $\nabla f$): $D_{\mathbf{u}} f = -\|\nabla f\|$
  • Zero when $\theta = \pi/2$ (perpendicular to $\nabla f$)
The Gradient's Secret

The gradient points in the direction of maximum increase, and its magnitude is that maximum rate of increase. This is why gradient descent works—we simply go the opposite way!

Worked Example

Let $f(x, y) = x^2 - xy + y^2$. Find the directional derivative at $(1, 2)$ in direction $\mathbf{v} = (3, 4)$.

Solution
1
Compute the gradient: $\nabla f = (2x - y, -x + 2y)$
2
Evaluate at $(1, 2)$: $\nabla f(1, 2) = (2 - 2, -1 + 4) = (0, 3)$
3
Normalize direction: $\mathbf{u} = \frac{(3, 4)}{5} = (3/5, 4/5)$
4
Compute: $D_{\mathbf{u}} f = (0, 3) \cdot (3/5, 4/5) = 0 + 12/5 = \boxed{2.4}$

Level Curves and Gradients

A level curve is the set of points where $f(x, y) = c$ for some constant $c$.

Gradient is Perpendicular to Level Curves

Along a level curve, $f$ doesn't change, so the directional derivative along the curve is zero. Since $D_{\mathbf{u}} f = \nabla f \cdot \mathbf{u} = 0$, the gradient must be perpendicular to the level curve.

This is why gradient descent always moves perpendicular to contour lines—it's taking the most direct path downhill!

Applications

Constrained Optimization

In Lagrange multipliers, we find where the gradient of the objective is parallel to the gradient of the constraint—where moving along the constraint doesn't improve the objective.

Normal Vectors

For a surface $f(x, y, z) = 0$, the gradient $\nabla f$ gives the normal vector at any point. This is used in computer graphics for lighting calculations.

Feature Importance

In ML, directional derivatives tell us how sensitive the output is to changes in input features. Large gradients indicate important features.
