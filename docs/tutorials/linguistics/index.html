Computational Linguistics Tutorials

How language works, how machines process it, and the deep connections to information theory. From phonemes to parse trees, from n-grams to word embeddings.

The Entropy Thread

Entropy weaves through every tutorial in this section. Language has structure (low entropy) embedded in apparent randomness (high entropy). Information theory gives us the tools to measure, model, and exploit this structure.

  • Tutorial 04 connects perplexity to cross-entropy
