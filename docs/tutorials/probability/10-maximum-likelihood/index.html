10. Maximum Likelihood Estimation

Maximum likelihood estimation (MLE) is the most widely used method for fitting statistical models to data. Given a parametric family of distributions, the MLE picks the parameter value that makes the observed data most probable. This tutorial derives MLE from first principles, works through key examples, establishes its optimality through Fisher information and the Cramér-Rao bound, and connects it to modern machine learning.

The Estimation Problem

Suppose we observe data $x_1, x_2, \ldots, x_n$ and believe they were generated by some probability distribution $f(x \mid \theta)$, where $\theta$ is an unknown parameter (or vector of parameters). The estimation problem is: given the data, what is the best guess for $\theta$?

This question arises everywhere:

  • A biologist measures the heights of 100 plants and wants to estimate the population mean and variance.
  • A data scientist fits a logistic regression model and needs to estimate the weight vector.
  • A physicist measures the decay times of radioactive atoms to estimate the decay rate.

There are two major philosophical frameworks for estimation:

Approach View of $\theta$ Method
Frequentist $\theta$ is a fixed but unknown constant MLE, method of moments, UMVUE
Bayesian $\theta$ is a random variable with a prior distribution MAP, posterior mean, full posterior

MLE is the crown jewel of frequentist estimation. It is popular because it is intuitive (choose the parameter that best explains the data), general (works for any parametric model), and optimal (achieves the lowest possible variance among a broad class of estimators, at least asymptotically).

The Likelihood Function

The likelihood function is the key object in MLE. Given observed data $\mathbf{x} = (x_1, \ldots, x_n)$, the likelihood is the joint density (or mass function) viewed as a function of the parameter:

$$L(\theta) = L(\theta \mid \mathbf{x}) = f(\mathbf{x} \mid \theta) = \prod_{i=1}^{n} f(x_i \mid \theta)$$

The last equality holds when the observations are independent and identically distributed (i.i.d.). This is a crucial conceptual shift:

  • Probability: $\theta$ is fixed, and we ask how probable different data outcomes are. $f(x \mid \theta)$ is a function of $x$.
  • Likelihood: the data $\mathbf{x}$ is fixed (we observed it), and we ask which parameter values make this data most probable. $L(\theta)$ is a function of $\theta$.

Important: The likelihood is not a probability distribution over $\theta$. It does not integrate to 1 over the parameter space, and in general $\int L(\theta) \, d\theta \neq 1$. It is a measure of relative plausibility: if $L(\theta_1) = 3 \, L(\theta_2)$, then $\theta_1$ makes the observed data three times more probable than $\theta_2$.

Because products are unwieldy (numerically and analytically), we almost always work with the log-likelihood:

$$\ell(\theta) = \log L(\theta) = \sum_{i=1}^{n} \log f(x_i \mid \theta)$$

The log is a strictly increasing function, so maximizing $\ell(\theta)$ and maximizing $L(\theta)$ give the same answer. The log-likelihood turns products into sums, which are far easier to differentiate and optimize.

Maximum Likelihood Estimator

The maximum likelihood estimator (MLE) is the parameter value that maximizes the likelihood (equivalently, the log-likelihood):

$$\hat{\theta}_{\text{MLE}} = \arg\max_{\theta \in \Theta} \, L(\theta) = \arg\max_{\theta \in \Theta} \, \ell(\theta)$$

To find the MLE, we typically follow a three-step recipe:

  1. Write the log-likelihood. Compute $\ell(\theta) = \sum_{i=1}^n \log f(x_i \mid \theta)$.
  2. Solve the score equation. The score function is $S(\theta) = \frac{\partial \ell}{\partial \theta}$. Set it to zero: $$S(\hat{\theta}) = \frac{\partial \ell}{\partial \theta}\bigg|_{\theta = \hat{\theta}} = 0$$
  3. Verify it is a maximum. Check that $\frac{\partial^2 \ell}{\partial \theta^2}\big|_{\theta = \hat{\theta}} < 0$, or use other arguments (e.g., concavity of the entire log-likelihood).

For vector parameters $\boldsymbol{\theta} = (\theta_1, \ldots, \theta_k)$, the score equation becomes a system: $\nabla_{\boldsymbol{\theta}} \ell = \mathbf{0}$, and the second-order condition requires the Hessian matrix to be negative definite.

Caveat: Not all MLEs come from solving $S(\theta) = 0$. Sometimes the MLE occurs at the boundary of the parameter space (as in the Uniform example below), and sometimes the likelihood has no closed-form maximum, requiring numerical optimization.

MLE Examples

Let us work through the MLE for five fundamental distributions. These examples build core skill and illustrate the variety of behaviors MLEs can have.

Example 1: Bernoulli Distribution

Data: $x_1, \ldots, x_n \overset{\text{iid}}{\sim} \text{Bernoulli}(p)$, so $x_i \in \{0, 1\}$. Let $k = \sum_{i=1}^n x_i$ be the number of successes.

Likelihood:

$$L(p) = \prod_{i=1}^n p^{x_i}(1-p)^{1-x_i} = p^k (1-p)^{n-k}$$

Log-likelihood:

$$\ell(p) = k \log p + (n-k) \log(1-p)$$

Score equation:

$$\frac{d\ell}{dp} = \frac{k}{p} - \frac{n-k}{1-p} = 0 \implies k(1-p) = (n-k)p \implies k = np$$
$$\boxed{\hat{p}_{\text{MLE}} = \frac{k}{n} = \bar{x}}$$

The MLE for the Bernoulli parameter is the sample proportion—exactly what intuition suggests.

Example 2: Normal Distribution

Data: $x_1, \ldots, x_n \overset{\text{iid}}{\sim} \mathcal{N}(\mu, \sigma^2)$. Both $\mu$ and $\sigma^2$ are unknown.

Log-likelihood:

$$\ell(\mu, \sigma^2) = -\frac{n}{2}\log(2\pi) - \frac{n}{2}\log\sigma^2 - \frac{1}{2\sigma^2}\sum_{i=1}^n (x_i - \mu)^2$$

Score equations:

$$\frac{\partial \ell}{\partial \mu} = \frac{1}{\sigma^2}\sum_{i=1}^n (x_i - \mu) = 0 \implies \hat{\mu} = \frac{1}{n}\sum_{i=1}^n x_i = \bar{x}$$
$$\frac{\partial \ell}{\partial \sigma^2} = -\frac{n}{2\sigma^2} + \frac{1}{2\sigma^4}\sum_{i=1}^n (x_i - \mu)^2 = 0 \implies \hat{\sigma}^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2$$
$$\boxed{\hat{\mu}_{\text{MLE}} = \bar{x}, \qquad \hat{\sigma}^2_{\text{MLE}} = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2}$$

Note that $\hat{\sigma}^2_{\text{MLE}}$ divides by $n$, not $n-1$. This makes it a biased estimator of $\sigma^2$ (it slightly underestimates the true variance). The unbiased sample variance $s^2 = \frac{1}{n-1}\sum(x_i - \bar{x})^2$ is preferred for small samples, but the bias vanishes as $n \to \infty$.

Example 3: Poisson Distribution

Data: $x_1, \ldots, x_n \overset{\text{iid}}{\sim} \text{Poisson}(\lambda)$.

Log-likelihood:

$$\ell(\lambda) = \sum_{i=1}^n \left( x_i \log \lambda - \lambda - \log(x_i!) \right) = \left(\sum x_i\right) \log \lambda - n\lambda - \sum \log(x_i!)$$

Score equation:

$$\frac{d\ell}{d\lambda} = \frac{\sum x_i}{\lambda} - n = 0$$
$$\boxed{\hat{\lambda}_{\text{MLE}} = \frac{1}{n}\sum_{i=1}^n x_i = \bar{x}}$$

Example 4: Exponential Distribution

Data: $x_1, \ldots, x_n \overset{\text{iid}}{\sim} \text{Exp}(\lambda)$, with density $f(x \mid \lambda) = \lambda e^{-\lambda x}$ for $x \geq 0$.

Log-likelihood:

$$\ell(\lambda) = n \log \lambda - \lambda \sum_{i=1}^n x_i$$

Score equation:

$$\frac{d\ell}{d\lambda} = \frac{n}{\lambda} - \sum x_i = 0$$
$$\boxed{\hat{\lambda}_{\text{MLE}} = \frac{n}{\sum_{i=1}^n x_i} = \frac{1}{\bar{x}}}$$

Example 5: Uniform Distribution (Boundary MLE)

Data: $x_1, \ldots, x_n \overset{\text{iid}}{\sim} \text{Uniform}(0, \theta)$, with density $f(x \mid \theta) = 1/\theta$ for $0 \leq x \leq \theta$.

Likelihood:

$$L(\theta) = \prod_{i=1}^n \frac{1}{\theta} \cdot \mathbf{1}(0 \leq x_i \leq \theta) = \frac{1}{\theta^n} \cdot \mathbf{1}(\theta \geq x_{(n)})$$

where $x_{(n)} = \max(x_1, \ldots, x_n)$ is the sample maximum. For $\theta \geq x_{(n)}$, $L(\theta) = 1/\theta^n$ is strictly decreasing, so the maximum occurs at the smallest allowable value:

$$\boxed{\hat{\theta}_{\text{MLE}} = x_{(n)} = \max(x_1, \ldots, x_n)}$$

This MLE does not come from setting a derivative to zero. The maximum is at the boundary of the parameter space. Also, this estimator is biased downward: $E[\hat{\theta}] = \frac{n}{n+1}\theta < \theta$, though the bias vanishes as $n \to \infty$.

Properties of MLE

MLE is not just a convenient recipe—it has remarkable theoretical properties that justify its widespread use. Under regularity conditions (the parameter space is open, the density is smooth, identifiable, etc.), the MLE satisfies:

1. Consistency

As $n \to \infty$, the MLE converges in probability to the true parameter value:

$$\hat{\theta}_{\text{MLE}} \xrightarrow{P} \theta_0$$

Intuition: with more data, the log-likelihood concentrates ever more sharply around the true value. Formally, by the law of large numbers, $\frac{1}{n}\ell(\theta) \to E[\log f(X \mid \theta)]$, and this expectation is uniquely maximized at $\theta_0$ (by the information inequality / Gibbs' inequality).

2. Asymptotic Normality

The MLE is approximately normal for large $n$:

$$\sqrt{n}\left(\hat{\theta}_{\text{MLE}} - \theta_0\right) \xrightarrow{d} \mathcal{N}\!\left(0, \frac{1}{I(\theta_0)}\right)$$

where $I(\theta_0)$ is the Fisher information (defined in the next section). Equivalently, for large $n$:

$$\hat{\theta}_{\text{MLE}} \;\dot{\sim}\; \mathcal{N}\!\left(\theta_0, \frac{1}{nI(\theta_0)}\right)$$

This gives us an automatic recipe for confidence intervals without knowing the exact finite-sample distribution of $\hat{\theta}$.

3. Invariance (Equivariance)

If $\hat{\theta}$ is the MLE of $\theta$, then for any function $g$:

$$\widehat{g(\theta)}_{\text{MLE}} = g(\hat{\theta}_{\text{MLE}})$$

For example, if $\hat{\lambda} = 1/\bar{x}$ is the MLE of the Exponential rate, then the MLE of the mean $1/\lambda$ is just $\bar{x}$. This seems obvious but is not true for many other estimators (e.g., the unbiased estimator of $\sigma^2$ is $s^2$, but $s$ is not an unbiased estimator of $\sigma$).

4. Asymptotic Efficiency

Among all consistent, asymptotically normal estimators, the MLE has the smallest asymptotic variance. It achieves the Cramér-Rao lower bound (defined below) in the limit, meaning no other "well-behaved" estimator can do better for large $n$.

Fisher Information

The Fisher information quantifies how much information a single observation carries about the unknown parameter $\theta$. It is defined as:

$$I(\theta) = E\!\left[\left(\frac{\partial}{\partial \theta} \log f(X \mid \theta)\right)^{\!2}\right] = -E\!\left[\frac{\partial^2}{\partial \theta^2} \log f(X \mid \theta)\right]$$

The two expressions are equal under standard regularity conditions (interchange of differentiation and integration). Let us unpack both:

  • First form (variance of the score): The score $S(\theta) = \frac{\partial}{\partial \theta}\log f(X \mid \theta)$ has $E[S(\theta)] = 0$ at the true $\theta$. So $I(\theta) = \text{Var}(S(\theta))$. High Fisher information means the score is highly variable—the data are informative about $\theta$.
  • Second form (expected curvature): $I(\theta) = -E[\ell''(\theta)]$. A sharply curved log-likelihood (large $|{\ell''}|$) means the peak is narrow, so $\theta$ is precisely determined. This is the form used most for computation.

Fisher information for $n$ i.i.d. observations:

$$I_n(\theta) = n \, I(\theta)$$

Information grows linearly with sample size. This is why standard errors decrease like $1/\sqrt{n}$.

Figure 2. Fisher information as curvature. A narrow (sharply curved) log-likelihood indicates high Fisher information and a precise estimate. A wide (gently curved) log-likelihood indicates low Fisher information and an uncertain estimate.

Fisher Information Examples

Bernoulli($p$):

$$\log f(x \mid p) = x \log p + (1-x)\log(1-p)$$
$$\frac{\partial^2}{\partial p^2}\log f = -\frac{x}{p^2} - \frac{1-x}{(1-p)^2} \implies I(p) = \frac{1}{p(1-p)}$$

Information is lowest at $p = 1/2$ (hardest to estimate) and diverges as $p \to 0$ or $p \to 1$ (easy to tell a very biased coin from a fair one).

Normal($\mu, \sigma^2$ known):

$$I(\mu) = \frac{1}{\sigma^2}$$

Smaller variance (less noise) means more information about the mean per observation.

Exponential($\lambda$):

$$I(\lambda) = \frac{1}{\lambda^2}$$

Fisher Information Matrix

For a vector parameter $\boldsymbol{\theta} = (\theta_1, \ldots, \theta_k)$, Fisher information becomes a $k \times k$ matrix:

$$[\mathbf{I}(\boldsymbol{\theta})]_{jl} = -E\!\left[\frac{\partial^2 \ell}{\partial \theta_j \, \partial \theta_l}\right]$$

For the Normal with both $\mu$ and $\sigma^2$ unknown:

$$\mathbf{I}(\mu, \sigma^2) = \begin{pmatrix} 1/\sigma^2 & 0 \\ 0 & 1/(2\sigma^4) \end{pmatrix}$$

The diagonal structure shows that $\mu$ and $\sigma^2$ are "informationally orthogonal"—knowing one does not help estimate the other.

Cramér-Rao Lower Bound

The Cramér-Rao inequality sets a fundamental limit on how precisely any unbiased estimator can estimate a parameter:

$$\text{Var}(\hat{\theta}) \geq \frac{1}{n \, I(\theta)}$$

for any unbiased estimator $\hat{\theta}$ of $\theta$ based on $n$ i.i.d. observations.

Proof sketch:

The proof uses the Cauchy-Schwarz inequality. Let $T = \hat{\theta}(\mathbf{X})$ be any unbiased estimator and $S = \sum_{i=1}^n \frac{\partial}{\partial \theta}\log f(X_i \mid \theta)$ be the total score. Since $E[S] = 0$ and $E[T] = \theta$:

$$1 = \frac{\partial}{\partial \theta} E[T] = \text{Cov}(T, S)$$

By Cauchy-Schwarz:

$$1 = [\text{Cov}(T, S)]^2 \leq \text{Var}(T) \cdot \text{Var}(S) = \text{Var}(T) \cdot n I(\theta)$$

Rearranging gives the bound. Equality holds if and only if $T$ is a linear function of $S$, which happens precisely for the exponential family.

Definitions:

  • An estimator that achieves the Cramér-Rao bound is called efficient.
  • The efficiency of an estimator is $e(\hat{\theta}) = \frac{1/(nI(\theta))}{\text{Var}(\hat{\theta})} \leq 1$.
  • A UMVUE (uniformly minimum variance unbiased estimator) has the smallest variance among all unbiased estimators. If an efficient estimator exists, it is the UMVUE.

Example: For $X_1, \ldots, X_n \overset{\text{iid}}{\sim} \mathcal{N}(\mu, \sigma^2)$ with $\sigma^2$ known, $\text{Var}(\bar{X}) = \sigma^2/n = 1/(nI(\mu))$. So $\bar{X}$ is efficient—it achieves the Cramér-Rao bound exactly.

Sufficient Statistics

A statistic $T(\mathbf{X})$ is sufficient for $\theta$ if it captures all the information in the data about $\theta$. Formally:

$$T(\mathbf{X}) \text{ is sufficient for } \theta \iff f(\mathbf{x} \mid T(\mathbf{x}), \theta) = f(\mathbf{x} \mid T(\mathbf{x}))$$

The conditional distribution of the data given $T$ does not depend on $\theta$. Once you know $T$, the remaining data contains no additional information about $\theta$.

Fisher-Neyman Factorization Theorem

The most practical tool for finding sufficient statistics:

$$T(\mathbf{X}) \text{ is sufficient for } \theta \iff f(\mathbf{x} \mid \theta) = g(T(\mathbf{x}), \theta) \cdot h(\mathbf{x})$$

where $g$ depends on $\mathbf{x}$ only through $T(\mathbf{x})$, and $h$ does not depend on $\theta$.

Examples:

  • Bernoulli($p$): $L(p) = p^{\sum x_i}(1-p)^{n-\sum x_i}$. Take $g = p^t(1-p)^{n-t}$ with $T = \sum x_i$, and $h = 1$. So $T = \sum X_i$ is sufficient for $p$.
  • Normal($\mu, \sigma^2$): $T = (\sum X_i, \sum X_i^2)$ is jointly sufficient for $(\mu, \sigma^2)$.
  • Uniform($0, \theta$): $T = X_{(n)} = \max(X_i)$ is sufficient for $\theta$.

MLE and Sufficiency

A beautiful fact: the MLE is always a function of any sufficient statistic. This follows because the likelihood depends on $\mathbf{x}$ only through $T(\mathbf{x})$ (by factorization), so maximizing $L(\theta)$ over $\theta$ can only involve $T$.

Minimal sufficiency: A sufficient statistic $T$ is minimal if it achieves the maximum data reduction—it is a function of every other sufficient statistic. The MLE is typically a function of the minimal sufficient statistic.

MLE vs MAP

The maximum a posteriori (MAP) estimator comes from Bayesian statistics. Given a prior $\pi(\theta)$ on the parameter, the MAP estimate maximizes the posterior distribution:

$$\hat{\theta}_{\text{MAP}} = \arg\max_\theta \, p(\theta \mid \mathbf{x}) = \arg\max_\theta \, L(\theta) \cdot \pi(\theta)$$

On a log scale:

$$\hat{\theta}_{\text{MAP}} = \arg\max_\theta \left[ \ell(\theta) + \log \pi(\theta) \right]$$

Key observations:

  • MLE = MAP with uniform prior: If $\pi(\theta) = \text{const}$, then $\log \pi(\theta)$ is constant, and MAP reduces to MLE. MLE is Bayesian with a flat prior.
  • Regularization as prior: L2 regularization $\|\theta\|^2$ corresponds to a Gaussian prior; L1 regularization $\|\theta\|_1$ corresponds to a Laplace prior. So regularized MLE is MAP estimation.
  • Small data vs big data: When $n$ is small, the prior $\log \pi(\theta)$ has a big effect (MAP $\neq$ MLE). As $n \to \infty$, the likelihood dominates: $\ell(\theta) = O(n)$ while $\log\pi(\theta) = O(1)$, so MAP $\to$ MLE.

Figure 3. MLE vs MAP. The prior pulls the MAP estimate toward the prior mode, away from the MLE. With more data, the likelihood sharpens and the MAP converges to the MLE.

The EM Algorithm

Sometimes direct maximization of the likelihood is intractable, often because the model involves latent (hidden) variables. The Expectation-Maximization (EM) algorithm, introduced by Dempster, Laird, and Rubin (1977), handles this elegantly.

Setup: We observe data $\mathbf{x}$ but there are hidden variables $\mathbf{z}$. The complete-data log-likelihood $\ell(\theta \mid \mathbf{x}, \mathbf{z})$ would be easy to maximize if we knew $\mathbf{z}$. The incomplete-data (marginal) log-likelihood is:

$$\ell(\theta \mid \mathbf{x}) = \log \sum_{\mathbf{z}} f(\mathbf{x}, \mathbf{z} \mid \theta)$$

The sum (or integral) inside the log makes direct optimization difficult.

The Algorithm: Starting from an initial guess $\theta^{(0)}$, repeat:

  1. E-step: Compute the expected complete-data log-likelihood, averaging over the current posterior of the latent variables: $$Q(\theta \mid \theta^{(t)}) = E_{\mathbf{z} \mid \mathbf{x}, \theta^{(t)}}\!\left[\log f(\mathbf{x}, \mathbf{z} \mid \theta)\right]$$
  2. M-step: Maximize $Q$ over $\theta$: $$\theta^{(t+1)} = \arg\max_\theta \, Q(\theta \mid \theta^{(t)})$$

Key property: Each EM iteration is guaranteed to increase (or maintain) the observed-data log-likelihood:

$$\ell(\theta^{(t+1)} \mid \mathbf{x}) \geq \ell(\theta^{(t)} \mid \mathbf{x})$$

This follows because the M-step maximizes a lower bound on $\ell$, and the E-step makes the bound tight at the current $\theta^{(t)}$. However, EM only converges to a local maximum (or saddle point), so multiple initializations are common.

Application: Gaussian Mixture Model

A GMM with $K$ components has density $f(x \mid \boldsymbol{\theta}) = \sum_{k=1}^K \pi_k \, \mathcal{N}(x \mid \mu_k, \sigma_k^2)$, where $\boldsymbol{\theta} = (\pi_k, \mu_k, \sigma_k^2)_{k=1}^K$. The latent variable $z_i \in \{1, \ldots, K\}$ indicates which component generated observation $x_i$.

  • E-step: Compute responsibilities $\gamma_{ik} = P(z_i = k \mid x_i, \theta^{(t)}) = \frac{\pi_k^{(t)} \, \mathcal{N}(x_i \mid \mu_k^{(t)}, {\sigma_k^2}^{(t)})}{\sum_j \pi_j^{(t)} \, \mathcal{N}(x_i \mid \mu_j^{(t)}, {\sigma_j^2}^{(t)})}$
  • M-step: Update parameters: $$\pi_k^{(t+1)} = \frac{N_k}{n}, \quad \mu_k^{(t+1)} = \frac{\sum_i \gamma_{ik} x_i}{N_k}, \quad {\sigma_k^2}^{(t+1)} = \frac{\sum_i \gamma_{ik}(x_i - \mu_k^{(t+1)})^2}{N_k}$$ where $N_k = \sum_i \gamma_{ik}$ is the effective number of points assigned to component $k$.

Connections to Machine Learning

MLE is not just a statistical concept—it is the engine behind most supervised learning algorithms. When you train a neural network by minimizing a loss function, you are often performing MLE without realizing it.

Cross-Entropy Loss = Negative Log-Likelihood

In classification with a model $p_\theta(y \mid x)$, the cross-entropy loss over a dataset $\{(x_i, y_i)\}$ is:

$$\mathcal{L}_{\text{CE}}(\theta) = -\frac{1}{n}\sum_{i=1}^n \log p_\theta(y_i \mid x_i) = -\frac{1}{n}\ell(\theta)$$

Minimizing cross-entropy loss is exactly maximizing the log-likelihood. This is true regardless of the model architecture (logistic regression, random forest, transformer, etc.).

Logistic Regression as MLE

In logistic regression, $P(Y = 1 \mid \mathbf{x}) = \sigma(\mathbf{w}^\top \mathbf{x})$ where $\sigma$ is the sigmoid function. The negative log-likelihood is the binary cross-entropy:

$$-\ell(\mathbf{w}) = -\sum_{i=1}^n \left[y_i \log \sigma(\mathbf{w}^\top \mathbf{x}_i) + (1-y_i)\log(1 - \sigma(\mathbf{w}^\top \mathbf{x}_i))\right]$$

This is a concave function of $\mathbf{w}$, so gradient descent finds the global MLE.

MSE Loss and Gaussian Likelihood

If we model $y = f_\theta(x) + \epsilon$ with $\epsilon \sim \mathcal{N}(0, \sigma^2)$, the log-likelihood is:

$$\ell(\theta) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum_{i=1}^n (y_i - f_\theta(x_i))^2$$

Maximizing over $\theta$ is equivalent to minimizing $\sum(y_i - f_\theta(x_i))^2$, the mean squared error. MSE loss assumes Gaussian noise.

Regularization = Prior = MAP

When we add L2 regularization $\lambda \|\mathbf{w}\|^2$ to the loss:

$$\mathcal{L}_{\text{reg}}(\mathbf{w}) = -\ell(\mathbf{w}) + \lambda \|\mathbf{w}\|^2$$

This is the negative log-posterior with a Gaussian prior $\mathbf{w} \sim \mathcal{N}(\mathbf{0}, \frac{1}{2\lambda}\mathbf{I})$. Minimizing the regularized loss performs MAP estimation, not MLE. The regularization strength $\lambda$ controls how much the prior influences the estimate.

1. MLE for Normal Distribution Parameters

We derive the MLE for $\mu$ and $\sigma^2$ and verify them with numerical optimization. 

import numpy as np
from scipy.optimize import minimize_scalar, minimize
from scipy.stats import norm

np.random.seed(42)

# True parameters
mu_true, sigma_true = 5.0, 2.0
n = 100
data = np.random.normal(mu_true, sigma_true, size=n)

# --- Analytical MLE ---
mu_mle = np.mean(data)
sigma2_mle = np.mean((data - mu_mle) ** 2)  # divides by n, not n-1

print("Analytical MLE:")
print(f"  mu_hat  = {mu_mle:.4f}  (true: {mu_true})")
print(f"  sigma2_hat = {sigma2_mle:.4f}  (true: {sigma_true**2})")

# --- Numerical MLE via log-likelihood ---
def neg_log_likelihood(params, data):
    mu, log_sigma2 = params  # use log(sigma^2) to enforce positivity
    sigma2 = np.exp(log_sigma2)
    n = len(data)
    ll = -n/2 * np.log(2 * np.pi * sigma2) - np.sum((data - mu)**2) / (2 * sigma2)
    return -ll  # minimize negative log-likelihood

result = minimize(neg_log_likelihood, x0=[0.0, 0.0], args=(data,), method='Nelder-Mead')
mu_num, sigma2_num = result.x[0], np.exp(result.x[1])

print("\nNumerical MLE (scipy.optimize):")
print(f"  mu_hat  = {mu_num:.4f}")
print(f"  sigma2_hat = {sigma2_num:.4f}")

# --- Log-likelihood surface ---
print("\nLog-likelihood at MLE:", -neg_log_likelihood([mu_mle, np.log(sigma2_mle)], data))

# --- Unbiased estimator comparison ---
sigma2_unbiased = np.var(data, ddof=1)  # divides by n-1
print(f"\nUnbiased sigma^2 = {sigma2_unbiased:.4f}")
print(f"MLE sigma^2      = {sigma2_mle:.4f}")
print(f"Ratio MLE/unbiased = {sigma2_mle/sigma2_unbiased:.4f}  (should be ~ {n}/{n+1} = {n/(n+1):.4f})")

2. MLE for Poisson Regression

Poisson regression models count data: $Y_i \sim \text{Poisson}(\lambda_i)$ with $\log \lambda_i = \mathbf{w}^\top \mathbf{x}_i$. We implement MLE via gradient descent. 

import numpy as np
from scipy.special import gammaln

np.random.seed(42)

# Generate synthetic Poisson regression data
n, d = 200, 3
X = np.column_stack([np.ones(n), np.random.randn(n, d - 1)])
w_true = np.array([1.0, 0.5, -0.3])

# True rates and observed counts
log_lambda = X @ w_true
y = np.random.poisson(np.exp(log_lambda))

# Log-likelihood for Poisson regression
def poisson_log_likelihood(w, X, y):
    eta = X @ w
    return np.sum(y * eta - np.exp(eta) - gammaln(y + 1))

# Gradient of log-likelihood
def poisson_gradient(w, X, y):
    eta = X @ w
    return X.T @ (y - np.exp(eta))

# Hessian of log-likelihood
def poisson_hessian(w, X, y):
    eta = X @ w
    W = np.diag(np.exp(eta))
    return -X.T @ W @ X

# Newton-Raphson optimization (IRLS)
w = np.zeros(d)

4. EM Algorithm for Gaussian Mixture Model

A from-scratch implementation of EM for fitting a mixture of Gaussians. 

import numpy as np
from scipy.stats import norm as sp_norm

np.random.seed(42)

# ============================================================
# Generate data from a 3-component Gaussian mixture
# ============================================================
true_params = {
    'pi': np.array([0.3, 0.5, 0.2]),
    'mu': np.array([-2.0, 1.0, 5.0]),
    'sigma': np.array([0.8, 1.2, 0.5])
}

n = 500
K = 3
z_true = np.random.choice(K, size=n, p=true_params['pi'])
data = np.array([np.random.normal(true_params['mu'][z], true_params['sigma'][z]) for z in z_true])

# ============================================================
# EM Algorithm
# ============================================================
def em_gmm(data, K, max_iter=100, tol=1e-8):
    n = len(data)

    # Random initialization
    pi = np.ones(K) / K
    mu = np.random.choice(data, size=K, replace=False)
    sigma = np.ones(K) * np.std(data)

    log_likelihoods = []

    for iteration in range(max_iter):
        # --- E-step: compute responsibilities ---
        # gamma[i, k] = P(z_i = k | x_i, theta)
        gamma = np.zeros((n, K))
        for k in range(K):
            gamma[:, k] = pi[k] * sp_norm.pdf(data, mu[k], sigma[k])
        gamma_sum = gamma.sum(axis=1, keepdims=True)
        gamma /= gamma_sum

        # Log-likelihood
        ll = np.sum(np.log(gamma_sum.ravel()))
        log_likelihoods.append(ll)

        # Check convergence
        if iteration > 0 and abs(log_likelihoods[-1] - log_likelihoods[-2]) < tol:
            print(f"Converged at iteration {iteration}")
            break

        # --- M-step: update parameters ---
        N_k = gamma.sum(axis=0)  # effective counts per component

        pi = N_k / n
        mu = (gamma.T @ data) / N_k
        for k in range(K):
            diff = data - mu[k]
            sigma[k] = np.sqrt(np.sum(gamma[:, k] * diff**2) / N_k[k])

    return pi, mu, sigma, log_likelihoods

# Run EM
pi_hat, mu_hat, sigma_hat, lls = em_gmm(data, K)

# Sort by means for comparison
order = np.argsort(mu_hat)
print("EM Results (sorted by mean):")
print(f"{'Component':>10} {'pi_hat':>8} {'mu_hat':>8} {'sigma_hat':>10}")
for k in order:
    print(f"{k:10d} {pi_hat[k]:8.3f} {mu_hat[k]:8.3f} {sigma_hat[k]:10.3f}")

print("\nTrue Parameters:")
print(f"{'Component':>10} {'pi':>8} {'mu':>8} {'sigma':>10}")
for k in range(K):
    print(f"{k:10d} {true_params['pi'][k]:8.3f} {true_params['mu'][k]:8.3f} {true_params['sigma'][k]:10.3f}")

print(f"\nFinal log-likelihood: {lls[-1]:.4f}")
print(f"Number of iterations: {len(lls)}")

5. Comparing MLE and MAP Estimates

We compare MLE and MAP for a Bernoulli parameter with a Beta prior, showing how the prior affects the estimate (especially with small samples). 

import numpy as np

np.random.seed(42)

p_true = 0.7

# Beta(alpha, beta) prior on p
# MAP for Bernoulli with Beta prior: (k + alpha - 1) / (n + alpha + beta - 2)
# Posterior mean: (k + alpha) / (n + alpha + beta)

priors = {
    'Uniform (alpha=1, beta=1)': (1, 1),          # flat prior -> MAP = MLE
    'Weak (alpha=2, beta=2)': (2, 2),              # mild shrinkage to 0.5
    'Strong (alpha=10, beta=10)': (10, 10),         # strong pull to 0.5
    'Informative (alpha=7, beta=3)': (7, 3),        # prior centered at 0.7
}

sample_sizes = [5, 20, 100, 1000]

print(f"True p = {p_true}\n")
print(f"{'Prior':<35} {'n':>5} {'k':>5} {'MLE':>8} {'MAP':>8} {'Post. Mean':>11} {'|MLE-p|':>8} {'|MAP-p|':>8}")
print("-" * 100)

for prior_name, (alpha, beta) in priors.items():
    for n in sample_sizes:
        data = np.random.binomial(1, p_true, size=n)
        k = data.sum()

        mle = k / n
        # MAP for Beta-Bernoulli
        if alpha + beta > 2:
            map_est = (k + alpha - 1) / (n + alpha + beta - 2)
        else:
            map_est = mle  # uniform prior: MAP = MLE
        post_mean = (k + alpha) / (n + alpha + beta)

        print(f"{prior_name:<35} {n:5d} {k:5d} {mle:8.4f} {map_est:8.4f} {post_mean:11.4f} {abs(mle-p_true):8.4f} {abs(map_est-p_true):8.4f}")

Exercise 1: MLE for Geometric Distribution

Let $X_1, \ldots, X_n \overset{\text{iid}}{\sim} \text{Geometric}(p)$, where $P(X = k) = (1-p)^{k-1}p$ for $k = 1, 2, 3, \ldots$. Find the MLE of $p$.

Show Solution

Likelihood:

$$L(p) = \prod_{i=1}^n (1-p)^{x_i - 1} p = p^n (1-p)^{\sum x_i - n}$$

Log-likelihood:

$$\ell(p) = n \log p + \left(\sum x_i - n\right) \log(1 - p)$$

Score equation:

$$\frac{d\ell}{dp} = \frac{n}{p} - \frac{\sum x_i - n}{1-p} = 0$$ $$n(1-p) = (\sum x_i - n)p \implies n = p \sum x_i$$

Answer: $\displaystyle\hat{p}_{\text{MLE}} = \frac{n}{\sum_{i=1}^n x_i} = \frac{1}{\bar{x}}$.

This is intuitive: the expected value of a Geometric($p$) is $1/p$, so the MLE inverts the sample mean.

Exercise 6: Fisher Information for the Normal

For $X \sim \mathcal{N}(\mu, \sigma^2)$ with $\sigma^2$ known: (a) Compute the Fisher information $I(\mu)$. (b) Verify that $\text{Var}(\bar{X}) = 1/(nI(\mu))$ (i.e., $\bar{X}$ is efficient). (c) For $\sigma^2$ unknown (and $\mu$ known), compute $I(\sigma^2)$ and the Cramér-Rao bound for estimating $\sigma^2$.

Show Solution

(a) $\log f(x \mid \mu) = -\frac{1}{2}\log(2\pi\sigma^2) - \frac{(x-\mu)^2}{2\sigma^2}$

$$\frac{\partial}{\partial \mu}\log f = \frac{x - \mu}{\sigma^2}, \qquad \frac{\partial^2}{\partial \mu^2}\log f = -\frac{1}{\sigma^2}$$ $$I(\mu) = -E\!\left[-\frac{1}{\sigma^2}\right] = \frac{1}{\sigma^2}$$

(b) $\text{Var}(\bar{X}) = \sigma^2/n$. The Cramér-Rao bound is $\frac{1}{nI(\mu)} = \frac{\sigma^2}{n}$. Since they are equal, $\bar{X}$ achieves the bound and is efficient.

(c) With $\mu$ known, let $\tau = \sigma^2$:

$$\log f(x \mid \tau) = -\frac{1}{2}\log(2\pi\tau) - \frac{(x-\mu)^2}{2\tau}$$ $$\frac{\partial}{\partial \tau}\log f = -\frac{1}{2\tau} + \frac{(x-\mu)^2}{2\tau^2}$$ $$\frac{\partial^2}{\partial \tau^2}\log f = \frac{1}{2\tau^2} - \frac{(x-\mu)^2}{\tau^3}$$ $$I(\tau) = -E\!\left[\frac{1}{2\tau^2} - \frac{(x-\mu)^2}{\tau^3}\right] = -\frac{1}{2\tau^2} + \frac{\tau}{\tau^3} = \frac{1}{2\tau^2} = \frac{1}{2\sigma^4}$$

Answer: $I(\sigma^2) = \frac{1}{2\sigma^4}$. Cramér-Rao bound: $\text{Var}(\hat{\sigma}^2) \geq \frac{2\sigma^4}{n}$.

Exercise 7: Cramér-Rao Bound Verification

For $X_1, \ldots, X_n \overset{\text{iid}}{\sim} \text{Poisson}(\lambda)$: (a) Compute $I(\lambda)$. (b) The MLE is $\hat{\lambda} = \bar{X}$. Compute $\text{Var}(\hat{\lambda})$ exactly. (c) Does $\bar{X}$ achieve the Cramér-Rao bound?

Show Solution

(a) $\log f(x \mid \lambda) = x \log \lambda - \lambda - \log(x!)$

$$\frac{\partial^2}{\partial \lambda^2}\log f = -\frac{x}{\lambda^2} \implies I(\lambda) = -E\!\left[-\frac{X}{\lambda^2}\right] = \frac{E[X]}{\lambda^2} = \frac{\lambda}{\lambda^2} = \frac{1}{\lambda}$$

(b) $\text{Var}(\bar{X}) = \text{Var}(X)/n = \lambda/n$.

(c) Cramér-Rao bound: $1/(nI(\lambda)) = \lambda/n$. Since $\text{Var}(\bar{X}) = \lambda/n$, the MLE $\bar{X}$ achieves the bound exactly and is efficient.

Answer: $I(\lambda) = 1/\lambda$. Yes, $\bar{X}$ is efficient for estimating the Poisson rate.

Exercise 8: Two-Parameter MLE

Let $X_1, \ldots, X_n \overset{\text{iid}}{\sim} \text{Gamma}(\alpha, \beta)$ with density $f(x \mid \alpha, \beta) = \frac{\beta^\alpha}{\Gamma(\alpha)} x^{\alpha-1} e^{-\beta x}$ for $x > 0$. (a) Write the log-likelihood. (b) Show that the MLE of $\beta$ given $\alpha$ is $\hat{\beta}(\alpha) = n\alpha / \sum x_i$. (c) Why can't we find a closed-form MLE for $\alpha$?

Show Solution

(a)

$$\ell(\alpha, \beta) = n\alpha \log \beta - n \log \Gamma(\alpha) + (\alpha - 1)\sum \log x_i - \beta \sum x_i$$

(b) Setting $\partial \ell / \partial \beta = 0$:

$$\frac{n\alpha}{\beta} - \sum x_i = 0 \implies \hat{\beta} = \frac{n\alpha}{\sum x_i}$$

(c) Setting $\partial \ell / \partial \alpha = 0$:

$$n \log \beta - n \psi(\alpha) + \sum \log x_i = 0$$

where $\psi(\alpha) = \Gamma'(\alpha)/\Gamma(\alpha)$ is the digamma function. Substituting $\hat{\beta}$:

$$\log \alpha - \psi(\alpha) + \log\!\left(\frac{\sum x_i}{n}\right) - \frac{1}{n}\sum \log x_i + \log n - \log(\sum x_i) = 0$$

This involves the digamma function $\psi(\alpha)$, which has no closed-form inverse. The MLE for $\alpha$ must be found numerically (e.g., Newton's method).

Answer: $\hat{\beta} = n\alpha/\sum x_i$ (conditional on $\alpha$). No closed form for $\hat{\alpha}$ because the score equation involves the digamma function.

Exercise 9: MLE for the Uniform—Bias and Consistency

Let $X_1, \ldots, X_n \overset{\text{iid}}{\sim} \text{Uniform}(0, \theta)$. The MLE is $\hat{\theta} = X_{(n)} = \max(X_i)$. (a) Show that $P(\hat{\theta} \leq t) = (t/\theta)^n$ for $0 \leq t \leq \theta$. (b) Compute $E[\hat{\theta}]$ and show it is biased. (c) Find a constant $c$ such that $c \, \hat{\theta}$ is unbiased. (d) Prove consistency: show $\hat{\theta} \xrightarrow{P} \theta$.

Show Solution

(a) $P(X_{(n)} \leq t) = P(\text{all } X_i \leq t) = \prod_{i=1}^n P(X_i \leq t) = (t/\theta)^n$.

(b) The PDF of $X_{(n)}$ is $f_{(n)}(t) = n t^{n-1}/\theta^n$ for $0 \leq t \leq \theta$.

$$E[\hat{\theta}] = \int_0^\theta t \cdot \frac{n t^{n-1}}{\theta^n} dt = \frac{n}{\theta^n} \cdot \frac{\theta^{n+1}}{n+1} = \frac{n}{n+1}\theta$$

Since $E[\hat{\theta}] = \frac{n}{n+1}\theta < \theta$, the MLE is biased downward.

(c) $c \cdot E[\hat{\theta}] = \theta \implies c = \frac{n+1}{n}$. So $\frac{n+1}{n}X_{(n)}$ is unbiased.

(d) For any $\epsilon > 0$:

$$P(|\hat{\theta} - \theta| > \epsilon) = P(\hat{\theta} < \theta - \epsilon) = \left(\frac{\theta - \epsilon}{\theta}\right)^n = \left(1 - \frac{\epsilon}{\theta}\right)^n \to 0 \text{ as } n \to \infty$$

Answer: $E[\hat{\theta}] = \frac{n}{n+1}\theta$ (biased). Unbiased version: $\frac{n+1}{n}X_{(n)}$. Consistency follows because $P(\hat{\theta} < \theta - \epsilon) \to 0$ exponentially fast.

Exercise 10: MLE and Sufficient Statistics

For $X_1, \ldots, X_n \overset{\text{iid}}{\sim} \text{Exp}(\lambda)$: (a) Use the Fisher-Neyman factorization theorem to find a sufficient statistic for $\lambda$. (b) Show that the MLE $\hat{\lambda} = n/\sum X_i$ is a function of the sufficient statistic. (c) Is $\sum X_i$ a minimal sufficient statistic? Justify.

Show Solution

(a) The joint density is:

$$f(\mathbf{x} \mid \lambda) = \prod_{i=1}^n \lambda e^{-\lambda x_i} = \lambda^n e^{-\lambda \sum x_i}$$

Take $g(T, \lambda) = \lambda^n e^{-\lambda T}$ with $T = \sum x_i$, and $h(\mathbf{x}) = 1$. By Fisher-Neyman, $T = \sum X_i$ is sufficient for $\lambda$.

(b) $\hat{\lambda} = n / \sum X_i = n / T$, which is indeed a function of $T$.

(c) Yes. $T = \sum X_i$ is minimal sufficient. To prove this, consider the ratio $f(\mathbf{x} \mid \lambda) / f(\mathbf{y} \mid \lambda) = e^{-\lambda(\sum x_i - \sum y_i)}$. Consider a mixture of two Gaussians: $f(x) = \pi \mathcal{N}(x \mid \mu_1, 1) + (1-\pi)\mathcal{N}(x \mid \mu_2, 1)$, with $\sigma^2 = 1$ known. (a) Write the complete-data log-likelihood $\ell_c(\pi, \mu_1, \mu_2 \mid \mathbf{x}, \mathbf{z})$. (b) Derive the E-step: compute $E[z_i \mid x_i, \theta^{(t)}]$. $$r_i^{(t)} = E[\gamma_i \mid x_i, \theta^{(t)}] = P(z_i = 1 \mid x_i, \theta^{(t)}) = \frac{\pi^{(t)} \phi(x_i - \mu_1^{(t)})}{\pi^{(t)}\phi(x_i - \mu_1^{(t)}) + (1-\pi^{(t)})\phi(x_i - \mu_2^{(t)})}$$

$$\pi^{(t+1)} = \frac{1}{n}\sum_{i=1}^n r_i^{(t)}$$ $$\mu_1^{(t+1)} = \frac{\sum_i r_i^{(t)} x_i}{\sum_i r_i^{(t)}}, \qquad \mu_2^{(t+1)} = \frac{\sum_i (1 - r_i^{(t)}) x_i}{\sum_i (1 - r_i^{(t)})}$$

These are weighted averages, with weights given by the responsibilities.

Exercise 12: Proving Consistency of MLE

Let $X_1, \ldots, X_n \overset{\text{iid}}{\sim} f(x \mid \theta_0)$ and define $M_n(\theta) = \frac{1}{n}\ell(\theta) = \frac{1}{n}\sum_{i=1}^n \log f(X_i \mid \theta)$. (a) Show that $M(\theta) := E[\log f(X \mid \theta)]$ is uniquely maximized at $\theta = \theta_0$ using the Kullback-Leibler divergence. (a) Consider:

$$M(\theta_0) - M(\theta) = E_{\theta_0}\!\left[\log \frac{f(X \mid \theta_0)}{f(X \mid \theta)}\right] = D_{KL}(f_{\theta_0} \| f_\theta)$$

By Gibbs' inequality (or the non-negativity of KL divergence), $D_{KL} \geq 0$ with equality if and only if $f_{\theta_0} = f_\theta$ a.e., which (under identifiability) means $\theta = \theta_0$. Therefore $M(\theta)$ is uniquely maximized at $\theta_0$.

(b) By the strong law of large numbers, for each $\theta$:

$$M_n(\theta) = \frac{1}{n}\sum_{i=1}^n \log f(X_i \mid \theta) \xrightarrow{a.s.} M(\theta)$$

Under uniform convergence (which holds under compactness and regularity), the maximizer of $M_n$ converges to the maximizer of $M$:

$$\hat{\theta}_n = \arg\max M_n(\theta) \xrightarrow{P} \arg\max M(\theta) = \theta_0$$

This is the "M-estimation" consistency argument. The key insight is that the MLE maximizes an empirical approximation to a function whose true maximum is at $\theta_0$, and as the approximation gets better (by LLN), the maximizer converges to the truth. $\square$

Exercise 13: Sufficient Statistic Proofs

For the Normal family $\mathcal{N}(\mu, \sigma^2)$ with both parameters unknown: (a) Prove that $T = (\sum X_i, \sum X_i^2)$ is sufficient using the factorization theorem. (b) Prove that $T$ is minimal sufficient by showing the ratio $f(\mathbf{x} \mid \mu, \sigma^2)/f(\mathbf{y} \mid \mu, \sigma^2)$ is free of $(\mu, \sigma^2)$ if and only if $T(\mathbf{x}) = T(\mathbf{y})$. (c) Can you reduce $T$ to a one-dimensional sufficient statistic? Why or why not?

Show Solution

(a) The joint density is:

$$f(\mathbf{x} \mid \mu, \sigma^2) = (2\pi\sigma^2)^{-n/2} \exp\!\left(-\frac{1}{2\sigma^2}\sum(x_i - \mu)^2\right)$$ $$= (2\pi\sigma^2)^{-n/2} \exp\!\left(-\frac{\sum x_i^2 - 2\mu \sum x_i + n\mu^2}{2\sigma^2}\right)$$

This is $g(\sum x_i, \sum x_i^2, \mu, \sigma^2) \cdot h(\mathbf{x})$ where $h(\mathbf{x}) = 1$. The density depends on $\mathbf{x}$ only through $(\sum x_i, \sum x_i^2)$, so this is sufficient by Fisher-Neyman. $\square$

(b) The likelihood ratio is:

$$\frac{f(\mathbf{x})}{f(\mathbf{y})} = \exp\!\left(-\frac{(\sum x_i^2 - \sum y_i^2) - 2\mu(\sum x_i - \sum y_i)}{2\sigma^2}\right)$$

This is free of $(\mu, \sigma^2)$ for all $(\mu, \sigma^2)$ iff both $\sum x_i^2 = \sum y_i^2$ and $\sum x_i = \sum y_i$, i.e., $T(\mathbf{x}) = T(\mathbf{y})$. So $T$ is minimal sufficient. $\square$

(c) No. With two unknown parameters, we need at least a two-dimensional sufficient statistic. Intuitively, the data must carry separate information about $\mu$ and $\sigma^2$, and no one-dimensional summary can capture both. Formally, the Normal family with both parameters unknown is a two-parameter exponential family, so the minimal sufficient statistic has dimension 2.

Exercise 14: Fisher Information Matrix and Multiparameter Cramér-Rao

For $X \sim \mathcal{N}(\mu, \sigma^2)$ with both parameters unknown, let $\boldsymbol{\theta} = (\mu, \sigma^2)$. (a) Compute the full $2 \times 2$ Fisher information matrix $\mathbf{I}(\boldsymbol{\theta})$. (b) State the multiparameter Cramér-Rao bound. (c) From the inverse of $n\mathbf{I}$, read off the asymptotic variance of $\hat{\mu}$ and $\hat{\sigma}^2$ and the asymptotic covariance between them.

Show Solution

(a) Let $\tau = \sigma^2$. The log-density is $\log f = -\frac{1}{2}\log(2\pi\tau) - \frac{(x-\mu)^2}{2\tau}$.

Second derivatives:

$$\frac{\partial^2 \log f}{\partial \mu^2} = -\frac{1}{\tau}, \qquad \frac{\partial^2 \log f}{\partial \mu \, \partial \tau} = -\frac{x - \mu}{\tau^2}$$ $$\frac{\partial^2 \log f}{\partial \tau^2} = \frac{1}{2\tau^2} - \frac{(x-\mu)^2}{\tau^3}$$

Taking negative expectations:

$$\mathbf{I}(\mu, \tau) = \begin{pmatrix} 1/\tau & 0 \\ 0 & 1/(2\tau^2) \end{pmatrix} = \begin{pmatrix} 1/\sigma^2 & 0 \\ 0 & 1/(2\sigma^4) \end{pmatrix}$$

(b) The multiparameter Cramér-Rao bound states that for any unbiased estimator $\hat{\boldsymbol{\theta}}$:

$$\text{Cov}(\hat{\boldsymbol{\theta}}) \succeq [n\mathbf{I}(\boldsymbol{\theta})]^{-1}$$

where $\succeq$ means the difference is positive semidefinite.

(c)

$$[n\mathbf{I}]^{-1} = \frac{1}{n}\begin{pmatrix} \sigma^2 & 0 \\ 0 & 2\sigma^4 \end{pmatrix}$$

Reading off the diagonal:

  • Asymptotic $\text{Var}(\hat{\mu}) = \sigma^2/n$
  • Asymptotic $\text{Var}(\hat{\sigma}^2) = 2\sigma^4/n$
  • Asymptotic $\text{Cov}(\hat{\mu}, \hat{\sigma}^2) = 0$

The zero covariance means the estimation of $\mu$ and $\sigma^2$ are asymptotically independent—an important property of the normal distribution.

Exercise 15: MLE, MAP, and Regularization in Logistic Regression

Consider logistic regression: $P(Y=1 \mid \mathbf{x}) = \sigma(\mathbf{w}^\top \mathbf{x})$ where $\sigma(z) = 1/(1+e^{-z})$. (a) Show that the negative log-likelihood is convex in $\mathbf{w}$. (b) When the data are linearly separable, show that the MLE $\hat{\mathbf{w}}$ does not exist (it diverges to infinity). (c) Show that adding an L2 penalty $\lambda\|\mathbf{w}\|^2$ corresponds to a Gaussian prior on $\mathbf{w}$, and that the regularized problem always has a finite solution. (d) What prior does L1 regularization correspond to, and why does it produce sparse solutions?

Show Solution

(a) The negative log-likelihood is:

$$-\ell(\mathbf{w}) = \sum_{i=1}^n \left[-y_i \mathbf{w}^\top\mathbf{x}_i + \log(1 + e^{\mathbf{w}^\top\mathbf{x}_i})\right]$$

The Hessian of $-\ell$ is:

$$\mathbf{H} = \sum_{i=1}^n \sigma_i(1-\sigma_i) \mathbf{x}_i \mathbf{x}_i^\top$$

where $\sigma_i = \sigma(\mathbf{w}^\top\mathbf{x}_i) \in (0,1)$. Each term is a PSD matrix (outer product scaled by positive weight), so $\mathbf{H} \succeq 0$, proving convexity. $\square$

(b) If the data are linearly separable, there exists $\mathbf{v}$ such that $y_i \mathbf{v}^\top\mathbf{x}_i > 0$ for all $i$. Consider $\mathbf{w} = c \mathbf{v}$ as $c \to \infty$:

  • For $y_i = 1$: the term $\log \sigma(c\mathbf{v}^\top\mathbf{x}_i) \to 0$.
  • For $y_i = 0$: the term $\log(1-\sigma(c\mathbf{v}^\top\mathbf{x}_i)) \to 0$.

So $\ell(\mathbf{w}) \to 0$ from below, but never reaches 0. The supremum is not attained at any finite $\mathbf{w}$, so the MLE does not exist. $\square$

(c) The regularized objective is:

$$\mathcal{L}_{\text{reg}} = -\ell(\mathbf{w}) + \lambda\|\mathbf{w}\|^2 = -\ell(\mathbf{w}) - \log \mathcal{N}(\mathbf{w} \mid \mathbf{0}, \frac{1}{2\lambda}\mathbf{I}) + \text{const}$$

This is $-\log p(\mathbf{w} \mid \text{data})$ with Gaussian prior $\mathbf{w} \sim \mathcal{N}(\mathbf{0}, \frac{1}{2\lambda}\mathbf{I})$. The Hessian of $\mathcal{L}_{\text{reg}}$ is $\mathbf{H} + 2\lambda\mathbf{I} \succ 0$ (strictly PD), so the objective is strictly convex and has a unique finite minimizer. $\square$

(d) L1 regularization: $\lambda\|\mathbf{w}\|_1 = -\log \prod_j \frac{\lambda}{2}e^{-\lambda|w_j|} + \text{const}$. This corresponds to independent Laplace priors $w_j \sim \text{Laplace}(0, 1/\lambda)$.

The Laplace prior has a sharp peak at zero, which assigns high probability to $w_j = 0$. Geometrically, the L1 ball has corners along the axes, so the contours of the loss function are more likely to first touch the constraint set at a corner (where some $w_j = 0$), producing sparse solutions.

Key Concept

Maximum likelihood estimation picks the parameter value $\hat{\theta}$ that makes the observed data most probable: $\hat{\theta} = \arg\max_\theta L(\theta)$.

Prerequisites

  • Calculus (derivatives, optimization)
  • Random variables & distributions
  • Common distributions (Normal, Poisson, Exponential)

Key Formulas

$L(\theta) = \prod_{i=1}^n f(x_i \mid \theta)$

$\ell(\theta) = \sum_{i=1}^n \log f(x_i \mid \theta)$

$S(\theta) = \frac{\partial \ell}{\partial \theta} = 0$

$I(\theta) = -E\!\left[\frac{\partial^2 \ell}{\partial \theta^2}\right]$

$\text{Var}(\hat{\theta}) \geq \frac{1}{nI(\theta)}$
