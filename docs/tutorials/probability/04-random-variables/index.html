04. Random Variables

A random variable is one of the most important abstractions in all of probability and statistics. It is a function that assigns a numerical value to each outcome in a sample space, allowing us to use the full power of calculus and algebra to analyze random phenomena. This tutorial develops the concept from first principles, covers discrete and continuous random variables, their characterizing functions (PMF, PDF, CDF), transformations, and connects everything to machine learning.

What is a Random Variable?

At its core, a random variable is a function from the sample space $\Omega$ to the real numbers $\mathbb{R}$:

$$\boxed{X : \Omega \to \mathbb{R}}$$

For each outcome $\omega \in \Omega$, the random variable assigns a real number $X(\omega)$. Despite the name, there is nothing "variable" about it in the algebraic sense—it is a deterministic function. The randomness comes from the underlying probability measure on $\Omega$.

Why do we need this abstraction? Consider tossing two coins. The sample space is $\Omega = \{HH, HT, TH, TT\}$. These outcomes are abstract symbols. But if we define $X(\omega) =$ "number of heads," then:

  • $X(TT) = 0$
  • $X(TH) = 1, \quad X(HT) = 1$
  • $X(HH) = 2$

Now we have numbers, and we can compute averages, variances, plot distributions, and apply all the machinery of analysis. The random variable is the bridge between the abstract world of sample spaces and the concrete world of numerical analysis.

Formal requirement (measurability): Technically, $X$ must be a measurable function, meaning that for every real number $x$, the set $\{\omega \in \Omega : X(\omega) \le x\}$ must be an event (i.e., belong to the $\sigma$-algebra). This ensures we can assign probabilities to statements like "$X \le 3$." In practice, every function you will encounter in applications satisfies this condition, so we mention it for completeness but will not dwell on it.

Notation conventions:

  • Capital letters ($X$, $Y$, $Z$) denote random variables.
  • Lowercase letters ($x$, $y$, $z$) denote specific realized values.
  • $P(X = x)$ means $P(\{\omega \in \Omega : X(\omega) = x\})$.
  • $P(X \le x)$ means $P(\{\omega \in \Omega : X(\omega) \le x\})$.

This distinction between the random variable $X$ and a particular value $x$ is fundamental and will be used throughout.

Discrete Random Variables

A random variable $X$ is discrete if it takes values in a countable set $\{x_1, x_2, x_3, \ldots\}$. The set of possible values is called the support of $X$.

$$\text{Support}(X) = \{x \in \mathbb{R} : P(X = x) > 0\}$$

Discrete random variables arise whenever we are counting things or selecting from a finite collection. Here are the most important examples:

Distribution Notation Support Meaning
Bernoulli $\text{Bern}(p)$ $\{0, 1\}$ Single yes/no trial with success probability $p$
Binomial $\text{Bin}(n, p)$ $\{0, 1, \ldots, n\}$ Number of successes in $n$ independent trials
Poisson $\text{Pois}(\lambda)$ $\{0, 1, 2, \ldots\}$ Count of events in a fixed interval (rate $\lambda$)
Geometric $\text{Geom}(p)$ $\{1, 2, 3, \ldots\}$ Number of trials until the first success

Example (Binomial): Flip a fair coin $n = 5$ times. Let $X$ = number of heads. Then $X \sim \text{Bin}(5, 0.5)$ and the support is $\{0, 1, 2, 3, 4, 5\}$. There are $\binom{5}{k}$ ways to get exactly $k$ heads, each with probability $(0.5)^5$, so:

$$P(X = k) = \binom{5}{k} (0.5)^k (0.5)^{5-k} = \binom{5}{k} \cdot \frac{1}{32}$$

Probability Mass Function (PMF)

The probability mass function (PMF) of a discrete random variable $X$ is the function $p_X : \mathbb{R} \to [0, 1]$ defined by:

$$\boxed{p_X(x) = P(X = x)}$$

The PMF must satisfy two properties:

  1. Non-negativity: $p_X(x) \ge 0$ for all $x$.
  2. Normalization: $\displaystyle\sum_{x \in \text{Support}(X)} p_X(x) = 1$.

These two properties are necessary and sufficient: any function satisfying them is a valid PMF for some discrete random variable. The PMF gives a complete description of the distribution of $X$—every probability question about $X$ can be answered from the PMF.

Example (Poisson PMF): If $X \sim \text{Pois}(\lambda)$, then:

$$p_X(k) = \frac{e^{-\lambda} \lambda^k}{k!}, \quad k = 0, 1, 2, \ldots$$

We can verify normalization: $\sum_{k=0}^{\infty} \frac{e^{-\lambda} \lambda^k}{k!} = e^{-\lambda} \cdot e^{\lambda} = 1$, using the Taylor series for $e^{\lambda}$.

Computing probabilities from the PMF: For any set $A \subseteq \mathbb{R}$:

$$P(X \in A) = \sum_{x \in A \cap \text{Support}(X)} p_X(x)$$

Continuous Random Variables

A random variable $X$ is continuous if there exists a non-negative function $f_X$ such that for every interval $[a, b]$:

$$P(a \le X \le b) = \int_a^b f_X(x) \, dx$$

The crucial difference from the discrete case is that for a continuous random variable, the probability of any single point is zero:

$$P(X = x) = \int_x^x f_X(t) \, dt = 0$$

This is deeply counterintuitive at first. If you throw a dart at a number line, the probability of hitting exactly $\pi$ is zero, yet the dart must land somewhere. The resolution is that probability is spread continuously over the real line—individual points have zero probability, but intervals have positive probability.

A practical consequence: for continuous random variables, it does not matter whether inequalities are strict or not:

$$P(a \le X \le b) = P(a < X < b) = P(a \le X < b) = P(a < X \le b)$$

This is not true for discrete random variables, where $P(X = a)$ can be positive.

Probability Density Function (PDF)

The function $f_X$ introduced above is called the probability density function (PDF). It is the continuous analogue of the PMF, but with a critical distinction: $f_X(x)$ is not a probability. It is a density—probability per unit length. It can even exceed 1.

$$\boxed{P(a \le X \le b) = \int_a^b f_X(x) \, dx}$$

The PDF must satisfy two properties:

  1. Non-negativity: $f_X(x) \ge 0$ for all $x$.
  2. Normalization: $\displaystyle\int_{-\infty}^{\infty} f_X(x) \, dx = 1$.

Intuitive meaning: For a small interval $[x, x + dx]$:

$$P(x \le X \le x + dx) \approx f_X(x) \cdot dx$$

The density $f_X(x)$ tells you how "concentrated" the probability is near $x$. Higher density means more probability packed into a small neighborhood.

Example (Exponential PDF): If $X \sim \text{Exp}(\lambda)$ with rate $\lambda > 0$, then:

$$f_X(x) = \begin{cases} \lambda e^{-\lambda x} & x \ge 0 \\ 0 & x < 0 \end{cases}$$

Verification: $\int_0^{\infty} \lambda e^{-\lambda x} \, dx = \left[-e^{-\lambda x}\right]_0^{\infty} = 0 - (-1) = 1$. ✓

Example (Standard Normal PDF): If $Z \sim \mathcal{N}(0, 1)$:

$$f_Z(z) = \frac{1}{\sqrt{2\pi}} e^{-z^2/2}, \quad z \in \mathbb{R}$$

Note that $f_Z(0) = 1/\sqrt{2\pi} \approx 0.399$. The density is highest at the mean, reflecting the bell-shaped concentration of probability.

Cumulative Distribution Function (CDF)

The cumulative distribution function (CDF) of a random variable $X$ is defined for all $x \in \mathbb{R}$ by:

$$\boxed{F_X(x) = P(X \le x)}$$

The CDF is the single most universal characterization of a random variable. Unlike the PMF (only for discrete) or PDF (only for continuous), the CDF is defined for every random variable.

Properties of the CDF (these hold for any random variable):

  1. Non-decreasing: If $a < b$, then $F_X(a) \le F_X(b)$.
  2. Right-continuous: $\lim_{x \to a^+} F_X(x) = F_X(a)$.
  3. Limits: $\lim_{x \to -\infty} F_X(x) = 0$ and $\lim_{x \to +\infty} F_X(x) = 1$.
  4. Interval probability: $P(a < X \le b) = F_X(b) - F_X(a)$.

Relationship to PMF (discrete case):

$$F_X(x) = \sum_{x_i \le x} p_X(x_i)$$

The CDF of a discrete random variable is a step function that jumps at each value in the support. The size of the jump at $x_i$ equals $p_X(x_i)$.

Relationship to PDF (continuous case):

$$F_X(x) = \int_{-\infty}^{x} f_X(t) \, dt \qquad \text{and} \qquad f_X(x) = \frac{d}{dx} F_X(x)$$

The CDF is the integral of the PDF, and the PDF is the derivative of the CDF. This is the Fundamental Theorem of Calculus applied to probability.

Example (Exponential CDF): For $X \sim \text{Exp}(\lambda)$:

$$F_X(x) = \int_0^x \lambda e^{-\lambda t} \, dt = 1 - e^{-\lambda x}, \quad x \ge 0$$

and $F_X(x) = 0$ for $x < 0$. We can verify: $P(X > t) = 1 - F_X(t) = e^{-\lambda t}$, the survival function, which decays exponentially.

Figure 2. Left: CDF of $\text{Bin}(5, 0.5)$—a step function with jumps at each integer. Right: CDF of $\text{Exp}(1)$—a smooth curve rising from 0 to 1.

Using the CDF to compute probabilities:

$$P(a < X \le b) = F_X(b) - F_X(a)$$
$$P(X > x) = 1 - F_X(x)$$

For discrete random variables, if you need $P(X \ge a)$ (with $\ge$ rather than $>$), be careful: $P(X \ge a) = 1 - F_X(a) + P(X = a) = 1 - F_X(a^-)$ where $F_X(a^-)$ is the left-hand limit.

Functions of Random Variables

Given a random variable $X$ and a function $g : \mathbb{R} \to \mathbb{R}$, we can define a new random variable $Y = g(X)$. But what is the distribution of $Y$?

Discrete case: This is straightforward. If $X$ is discrete with PMF $p_X$, then:

$$p_Y(y) = P(Y = y) = P(g(X) = y) = \sum_{x : g(x) = y} p_X(x)$$

We simply sum the probabilities of all $x$ values that map to $y$.

Example: Let $X$ be uniform on $\{-2, -1, 0, 1, 2\}$ and $Y = X^2$. Then $Y$ takes values $\{0, 1, 4\}$ with:

  • $P(Y = 0) = P(X = 0) = 1/5$
  • $P(Y = 1) = P(X = -1) + P(X = 1) = 2/5$
  • $P(Y = 4) = P(X = -2) + P(X = 2) = 2/5$

Continuous case (CDF method): For continuous $X$, we use a two-step approach:

  1. Find the CDF of $Y$: $F_Y(y) = P(Y \le y) = P(g(X) \le y)$.
  2. Differentiate to get the PDF: $f_Y(y) = F_Y'(y)$.

Example: Let $X \sim \text{Uniform}(0, 1)$ and $Y = X^2$. For $0 \le y \le 1$:

$$F_Y(y) = P(X^2 \le y) = P(X \le \sqrt{y}) = \sqrt{y}$$

since $X \ge 0$. Differentiating:

$$f_Y(y) = \frac{d}{dy} \sqrt{y} = \frac{1}{2\sqrt{y}}, \quad 0 < y < 1$$

Note that $f_Y(y) \to \infty$ as $y \to 0^+$. The density is unbounded, yet it integrates to 1. This illustrates that a PDF need not be bounded.

Change of Variables Formula

When $g$ is a monotonic, differentiable function, there is a direct formula for the PDF of $Y = g(X)$ that avoids going through the CDF:

$$\boxed{f_Y(y) = f_X\!\left(g^{-1}(y)\right) \cdot \left|\frac{d}{dy} g^{-1}(y)\right|}$$

The factor $|d g^{-1}(y)/dy|$ is the Jacobian (in one dimension, the absolute value of the derivative of the inverse transformation). It accounts for how the transformation stretches or compresses the probability density.

Intuition: If $g$ stretches the $x$-axis by a factor of 2, then the probability that was packed into an interval of width $dx$ is now spread over width $2\,dx$, so the density must be halved. The Jacobian factor captures exactly this stretching/compression effect.

Example 1 (Linear transformation): Let $X$ have PDF $f_X$ and $Y = aX + b$ where $a \neq 0$.

  • Inverse: $g^{-1}(y) = (y - b)/a$
  • Jacobian: $|d g^{-1}/dy| = 1/|a|$
$$f_Y(y) = \frac{1}{|a|} f_X\!\left(\frac{y - b}{a}\right)$$

This is the general formula for linear transformations. In particular, if $X \sim \mathcal{N}(\mu, \sigma^2)$, then $Z = (X - \mu)/\sigma$ has the standard normal distribution $\mathcal{N}(0, 1)$.

Example 2 (Exponential from Uniform): Let $U \sim \text{Uniform}(0, 1)$ and $Y = -\frac{1}{\lambda} \ln(U)$.

  • Inverse: $g^{-1}(y) = e^{-\lambda y}$
  • Jacobian: $|d g^{-1}/dy| = \lambda e^{-\lambda y}$
  • Since $f_U(u) = 1$ on $(0, 1)$ and $g^{-1}(y) = e^{-\lambda y} \in (0, 1)$ for $y > 0$:
$$f_Y(y) = 1 \cdot \lambda e^{-\lambda y} = \lambda e^{-\lambda y}, \quad y > 0$$

So $Y \sim \text{Exp}(\lambda)$. This is the inverse transform method—a fundamental technique for generating random samples from any distribution using uniform random numbers.

Example 3 (Square transformation): Let $X \sim \mathcal{N}(0, 1)$ and $Y = X^2$.

Here $g(x) = x^2$ is not monotonic (it maps both $x$ and $-x$ to the same $y$). We must split the domain. For $y > 0$, the two inverse branches are $x = +\sqrt{y}$ and $x = -\sqrt{y}$:

$$f_Y(y) = f_X(\sqrt{y}) \cdot \frac{1}{2\sqrt{y}} + f_X(-\sqrt{y}) \cdot \frac{1}{2\sqrt{y}}$$

Since the standard normal is symmetric, $f_X(\sqrt{y}) = f_X(-\sqrt{y})$:

$$f_Y(y) = \frac{2}{2\sqrt{y}} \cdot \frac{1}{\sqrt{2\pi}} e^{-y/2} = \frac{1}{\sqrt{2\pi y}} e^{-y/2}, \quad y > 0$$

This is the PDF of the chi-squared distribution with 1 degree of freedom, $\chi^2(1)$. This result is fundamental in statistics for hypothesis testing.

Mixed Random Variables

Not every random variable is purely discrete or purely continuous. A mixed random variable has a CDF that contains both jumps (discrete part) and continuous segments.

Example: A customer's waiting time $T$ at a service counter:

  • With probability $p = 0.3$, the counter is free and $T = 0$ (no wait).
  • With probability $1 - p = 0.7$, the customer must wait, and the wait time follows $\text{Exp}(\lambda)$ for $t > 0$.

The CDF of $T$ is:

$$F_T(t) = \begin{cases} 0 & t < 0 \\ p + (1 - p)(1 - e^{-\lambda t}) & t \ge 0 \end{cases}$$
$$= \begin{cases} 0 & t < 0 \\ 1 - (1-p) e^{-\lambda t} & t \ge 0 \end{cases}$$

At $t = 0$, there is a jump of size $p = 0.3$ (the discrete component). For $t > 0$, the CDF increases smoothly (the continuous component). The mixed distribution cannot be described by a PMF alone or a PDF alone, but the CDF handles it seamlessly.

In general, any CDF can be decomposed as:

$$F_X(x) = \alpha \, F_d(x) + (1 - \alpha) \, F_c(x)$$

where $F_d$ is a discrete CDF, $F_c$ is a continuous CDF, and $\alpha \in [0, 1]$ is the weight on the discrete component. This is the Lebesgue decomposition of a distribution.

Mixed distributions appear naturally in machine learning when models combine discrete decisions (e.g., "do nothing" vs. "take action") with continuous outcomes (e.g., "how much action to take"). The zero-inflated Poisson model and hurdle models in economics are examples.

Indicator Random Variables

An indicator random variable for an event $A$ is defined as:

$$\boxed{I_A(\omega) = \begin{cases} 1 & \text{if } \omega \in A \\ 0 & \text{if } \omega \notin A \end{cases}}$$

Despite its simplicity, the indicator random variable is one of the most powerful tools in probability. Its key property is:

$$E[I_A] = 1 \cdot P(A) + 0 \cdot P(A^c) = P(A)$$

This means every probability is an expectation of an indicator. This connection allows us to compute probabilities using expectation techniques, which are often simpler.

Useful algebraic properties:

  • $I_A^2 = I_A$ (since $0^2 = 0$ and $1^2 = 1$)
  • $I_{A \cap B} = I_A \cdot I_B$
  • $I_{A^c} = 1 - I_A$
  • $I_{A \cup B} = I_A + I_B - I_A I_B$

Example (Counting via indicators): Let $X_1, X_2, \ldots, X_n$ be the outcomes of $n$ coin flips. The total number of heads is:

$$S = \sum_{i=1}^{n} I_{\{X_i = H\}}$$

By linearity of expectation:

$$E[S] = \sum_{i=1}^{n} E[I_{\{X_i = H\}}] = \sum_{i=1}^{n} P(X_i = H) = np$$

This derivation works regardless of whether the coin flips are independent, because linearity of expectation holds always. This is far simpler than working with the binomial PMF directly.

The birthday problem via indicators: With $n$ people in a room, the expected number of pairs sharing a birthday is:

$$E\!\left[\sum_{i < j} I_{\{B_i = B_j\}}\right] = \binom{n}{2} \cdot \frac{1}{365}$$

For $n = 23$: $\binom{23}{2}/365 = 253/365 \approx 0.693$. On average, fewer than one pair shares a birthday, yet the probability that at least one pair does is about 50.7%. The indicator technique makes the expectation calculation trivial.

Connections to Machine Learning

Random variables are the mathematical language of machine learning. Here is how the concepts in this tutorial connect to core ML ideas:

Probability Concept ML Application
Features as RVs Each feature $X_j$ in a dataset is modeled as a random variable. A data point $(x_1, \ldots, x_d)$ is a realization of the random vector $(X_1, \ldots, X_d)$.
PMF and classification In classification, the label $Y \in \{0, 1, \ldots, K-1\}$ is a discrete RV. A classifier estimates its PMF: $P(Y = k \mid X = x)$. Softmax outputs are estimated PMFs.
PDF and generative models Generative models (VAEs, normalizing flows, diffusion models) learn the PDF $f_X(x)$ of the data distribution. Sampling from the model generates new data.
CDF and quantile regression Quantile regression estimates specific percentiles of the conditional distribution $F_{Y|X}$, useful for prediction intervals.
Change of variables Normalizing flows use invertible transformations with tractable Jacobians to transform simple distributions (Gaussian) into complex data distributions.
Indicator RVs Loss functions like 0-1 loss, cross-entropy with binary labels, and attention masks all use indicator variables. $E[I_{\{\hat{Y} \neq Y\}}]$ is the misclassification rate.

Generative vs. discriminative models:

  • Generative models learn the joint distribution $P(X, Y)$ or $P(X)$. They model the full random variable(s) and can generate new data. Examples: Naive Bayes, GANs, VAEs.
  • Discriminative models learn only the conditional $P(Y \mid X)$. They model how the label RV depends on the features. Examples: logistic regression, neural networks for classification.

The reparameterization trick in variational autoencoders is a direct application of the change of variables formula. Instead of sampling $Z \sim \mathcal{N}(\mu, \sigma^2)$ (which is not differentiable), we write $Z = \mu + \sigma \cdot \epsilon$ where $\epsilon \sim \mathcal{N}(0, 1)$. This is the linear change of variables $Z = g(\epsilon) = \mu + \sigma\epsilon$, and it allows gradients to flow through $\mu$ and $\sigma$.

Normalizing flows compose a sequence of invertible transformations:

$$z_0 \sim p_0(z_0), \quad z_k = g_k(z_{k-1}), \quad k = 1, \ldots, K$$

By the change of variables formula applied repeatedly:

$$\log p_K(z_K) = \log p_0(z_0) - \sum_{k=1}^{K} \log \left|\det \frac{\partial g_k}{\partial z_{k-1}}\right|$$

This is exactly the multivariate generalization of the change of variables formula from this tutorial, with the Jacobian determinant replacing the univariate derivative.

1. Simulating Discrete Random Variables and Plotting PMF

import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

# --- Simulate a Binomial(n=10, p=0.3) random variable ---
n, p = 10, 0.3
num_samples = 10_000

# Method 1: Direct sampling
samples = np.random.binomial(n, p, size=num_samples)

# Method 2: Sum of Bernoulli trials (from first principles)
bernoulli_trials = np.random.binomial(1, p, size=(num_samples, n))
samples_manual = bernoulli_trials.sum(axis=1)

# --- Compute empirical PMF ---
values, counts = np.unique(samples, return_counts=True)
empirical_pmf = counts / num_samples

# --- Theoretical PMF ---
from math import comb, factorial
k_values = np.arange(0, n + 1)
theoretical_pmf = np.array([
    comb(n, k) * p**k * (1 - p)**(n - k)
    for k in k_values
])

# --- Verify normalization ---
print(f"Sum of empirical PMF:    {empirical_pmf.sum():.6f}")
print(f"Sum of theoretical PMF:  {theoretical_pmf.sum():.6f}")

# --- Plot ---
fig, ax = plt.subplots(figsize=(8, 5))
width = 0.35
ax.bar(k_values - width/2, theoretical_pmf, width, label='Theoretical PMF',
       color='steelblue', alpha=0.8, edgecolor='white')
ax.bar(values + width/2, empirical_pmf, width, label=f'Empirical PMF (n={num_samples})',
       color='coral', alpha=0.8, edgecolor='white')
ax.set_xlabel('k (number of successes)')
ax.set_ylabel('P(X = k)')
ax.set_title('PMF of Binomial(10, 0.3)')
ax.set_xticks(k_values)
ax.legend()
ax.grid(axis='y', alpha=0.3)
plt.tight_layout()
plt.savefig('binomial_pmf.png', dpi=150)
plt.show()

2. Continuous Random Variable: PDF and CDF Visualization

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# --- Define distributions ---
distributions = {
    'Exponential(1)':   stats.expon(scale=1),
    'Normal(0, 1)':     stats.norm(0, 1),
    'Uniform(0, 1)':    stats.uniform(0, 1),
    'Gamma(2, 1)':      stats.gamma(a=2, scale=1),
}

fig, axes = plt.subplots(2, 2, figsize=(12, 10))

for ax, (name, dist) in zip(axes.flat, distributions.items()):
    # Define x range based on distribution support
    if 'Uniform' in name:
        x = np.linspace(-0.5, 1.5, 500)
    elif 'Exponential' in name or 'Gamma' in name:
        x = np.linspace(0, 8, 500)
    else:
        x = np.linspace(-4, 4, 500)

    # Plot PDF
    pdf_vals = dist.pdf(x)
    ax.plot(x, pdf_vals, 'b-', linewidth=2, label='PDF f(x)')
    ax.fill_between(x, pdf_vals, alpha=0.15, color='blue')

    # Plot CDF on twin axis
    ax2 = ax.twinx()
    cdf_vals = dist.cdf(x)
    ax2.plot(x, cdf_vals, 'r--', linewidth=2, label='CDF F(x)')
    ax2.set_ylim(-0.05, 1.15)
    ax2.set_ylabel('F(x)', color='red')
    ax2.tick_params(axis='y', labelcolor='red')

    ax.set_title(name, fontweight='bold')
    ax.set_xlabel('x')
    ax.set_ylabel('f(x)', color='blue')
    ax.tick_params(axis='y', labelcolor='blue')

    # Combined legend
    lines_1, labels_1 = ax.get_legend_handles_labels()
    lines_2, labels_2 = ax2.get_legend_handles_labels()
    ax.legend(lines_1 + lines_2, labels_1 + labels_2, loc='upper right')

    # Verify: integral of PDF = 1
    from scipy.integrate import quad
    integral, _ = quad(dist.pdf, dist.ppf(0.0001), dist.ppf(0.9999))
    ax.text(0.02, 0.95, f'integral of PDF approx {integral:.4f}',
            transform=ax.transAxes, fontsize=9, va='top',
            bbox=dict(boxstyle='round', facecolor='wheat', alpha=0.5))

plt.suptitle('PDF and CDF for Common Continuous Distributions', fontsize=14, y=1.02)
plt.tight_layout()
plt.savefig('pdf_cdf_comparison.png', dpi=150, bbox_inches='tight')
plt.show()

3. Change of Variables: Uniform to Exponential

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

np.random.seed(42)

lam = 2.0  # rate parameter
n_samples = 50_000

# --- Inverse transform method ---
# If U ~ Uniform(0,1), then Y = -ln(U)/lambda ~ Exp(lambda)
U = np.random.uniform(0, 1, n_samples)
Y = -np.log(U) / lam

# --- Verify via change of variables formula ---
# g(u) = -ln(u)/lambda, so g^{-1}(y) = e^{-lambda*y}
# |d/dy g^{-1}(y)| = lambda * e^{-lambda*y}
# f_Y(y) = f_U(g^{-1}(y)) * |d/dy g^{-1}(y)|
#         = 1 * lambda * e^{-lambda*y}  (for 0 < g^{-1}(y) < 1, i.e., y > 0)
#         = lambda * e^{-lambda*y}      which is Exp(lambda)!

fig, axes = plt.subplots(1, 3, figsize=(15, 4.5))

# Panel 1: Original uniform samples
axes[0].hist(U, bins=50, density=True, alpha=0.7, color='steelblue', edgecolor='white')
axes[0].axhline(y=1, color='red', linewidth=2, linestyle='--', label='f_U(u) = 1')
axes[0].set_title('U ~ Uniform(0, 1)', fontweight='bold')

Exercise 1: Computing a PMF

A fair die is rolled once. Let $X$ be the remainder when the result is divided by 3 (i.e., $X = \text{result} \mod 3$). Find the PMF of $X$ and verify it sums to 1.

Show Solution

The possible die outcomes are $\{1, 2, 3, 4, 5, 6\}$, each with probability $1/6$.

  • $1 \mod 3 = 1$, $2 \mod 3 = 2$, $3 \mod 3 = 0$
  • $4 \mod 3 = 1$, $5 \mod 3 = 2$, $6 \mod 3 = 0$

So:

  • $P(X = 0) = P(\{3, 6\}) = 2/6 = 1/3$
  • $P(X = 1) = P(\{1, 4\}) = 2/6 = 1/3$
  • $P(X = 2) = P(\{2, 5\}) = 2/6 = 1/3$

Verification: $1/3 + 1/3 + 1/3 = 1$. ✓

Answer: $X$ is uniformly distributed on $\{0, 1, 2\}$ with $p_X(x) = 1/3$ for $x \in \{0, 1, 2\}$.

Exercise 2: CDF from PMF

A random variable $X$ has PMF: $P(X = 1) = 0.2$, $P(X = 3) = 0.5$, $P(X = 7) = 0.3$. Find $F_X(x)$ for all $x$ and compute $P(2 < X \le 5)$.

The CDF is a step function:

Exercise 3: Verifying a PDF

Show that $f(x) = 3x^2$ for $0 \le x \le 1$ (and 0 otherwise) is a valid PDF. Then compute $P(X > 0.5)$.

Show Solution

Non-negativity: $3x^2 \ge 0$ for all $x$. ✓

Normalization:

$$\int_0^1 3x^2 \, dx = \left[x^3\right]_0^1 = 1 - 0 = 1 \quad \checkmark$$

Computing $P(X > 0.5)$:

$$P(X > 0.5) = \int_{0.5}^{1} 3x^2 \, dx = \left[x^3\right]_{0.5}^{1} = 1 - 0.125 = 0.875$$

Exercise 4: CDF to PDF

A continuous random variable has CDF $F_X(x) = 1 - e^{-2x}$ for $x \ge 0$ (and 0 for $x < 0$). Find the PDF and identify the distribution.

Show Solution

The PDF is the derivative of the CDF:

$$f_X(x) = \frac{d}{dx} F_X(x) = \frac{d}{dx}(1 - e^{-2x}) = 2e^{-2x}, \quad x \ge 0$$

and $f_X(x) = 0$ for $x < 0$.

This is the PDF of the Exponential distribution with rate $\lambda = 2$, i.e., $X \sim \text{Exp}(2)$.

Verification: $\int_0^\infty 2e^{-2x} \, dx = \left[-e^{-2x}\right]_0^\infty = 0 - (-1) = 1$. ✓

Answer: $f_X(x) = 2e^{-2x}$ for $x \ge 0$. This is $\text{Exp}(2)$.

Show Solution

Exercise 11: Change of Variables (Linear)

Let $X \sim \text{Exp}(1)$ with PDF $f_X(x) = e^{-x}$ for $x > 0$. Find the PDF of $Y = 3X + 2$ using the change of variables formula.

Exercise 14: Indicator Variables for Counting

$$X = \sum_{i=1}^{52} I_i$$

By linearity of expectation:

Answer: $E[X] = 1$.

Let $G(n) = P(X > n)$. The memoryless property gives $G(m+n) = G(m)G(n)$.

This is Cauchy's functional equation on the non-negative integers. Let $G(1) = q$ where $0 < q < 1$.

Then $G(n) = G(1)^n = q^n$ by induction: $G(n) = G(n-1+1) = G(n-1)G(1) = q^{n-1} \cdot q = q^n$.

So $P(X = k) = G(k-1) - G(k) = q^{k-1} - q^k = q^{k-1}(1-q)$ for $k = 1, 2, \ldots$

This is exactly the Geometric($p = 1-q$) PMF. $\square$

Exercise 20: Mixture Distribution

Let $X$ be a mixture: with probability $1/3$, $X \sim \text{Exp}(1)$; with probability $2/3$, $X \sim \text{Exp}(3)$. (a) Find the PDF of $X$. (b) Find $E[X]$. (c) Find $\text{Var}(X)$.

Show Solution

(a) $f_X(x) = \frac{1}{3}e^{-x} + \frac{2}{3} \cdot 3e^{-3x} = \frac{1}{3}e^{-x} + 2e^{-3x}$ for $x \geq 0$.

(b) $E[X] = \frac{1}{3} \cdot 1 + \frac{2}{3} \cdot \frac{1}{3} = \frac{1}{3} + \frac{2}{9} = \frac{5}{9} \approx 0.556$.

(c) Using the law of total variance: $\text{Var}(X) = E[\text{Var}(X \mid C)] + \text{Var}(E[X \mid C])$

$E[\text{Var}(X \mid C)] = \frac{1}{3}(1) + \frac{2}{3}(\frac{1}{9}) = \frac{1}{3} + \frac{2}{27} = \frac{11}{27}$

$\text{Var}(E[X \mid C]) = E[(E[X \mid C])^2] - (E[X])^2 = \frac{1}{3}(1) + \frac{2}{3}(\frac{1}{9}) - (\frac{5}{9})^2 = \frac{11}{27} - \frac{25}{81} = \frac{8}{81}$

$\text{Var}(X) = \frac{11}{27} + \frac{8}{81} = \frac{33}{81} + \frac{8}{81} = \frac{41}{81} \approx 0.506$

Exercise 21: Distribution of a Non-Monotonic Function

Let $X \sim \mathcal{N}(0, 1)$ and $Y = |X|$. (a) Find the CDF of $Y$. (b) Find the PDF of $Y$. (c) Identify the distribution by name.

Show Solution

(a) CDF of $Y$: For $y \ge 0$:

$$F_Y(y) = P(|X| \le y) = P(-y \le X \le y) = \Phi(y) - \Phi(-y) = 2\Phi(y) - 1$$

where $\Phi$ is the standard normal CDF, using the symmetry $\Phi(-y) = 1 - \Phi(y)$.

For $y < 0$: $F_Y(y) = 0$ since $|X| \ge 0$.

(b) PDF of $Y$: Differentiating for $y > 0$:

$$f_Y(y) = \frac{d}{dy}[2\Phi(y) - 1] = 2\phi(y) = 2 \cdot \frac{1}{\sqrt{2\pi}} e^{-y^2/2} = \sqrt{\frac{2}{\pi}} \, e^{-y^2/2}$$

and $f_Y(y) = 0$ for $y < 0$.

(c) This is the half-normal distribution. It is also $\chi(1)$, the chi distribution with 1 degree of freedom (not to be confused with chi-squared).

Verification: $\int_0^{\infty} \sqrt{2/\pi} \, e^{-y^2/2} \, dy = \sqrt{2/\pi} \cdot \sqrt{\pi/2} = 1$. ✓

Answer: $f_Y(y) = \sqrt{2/\pi} \, e^{-y^2/2}$ for $y \ge 0$ (half-normal distribution).

Exercise 22: Proving CDF Properties

Prove that if $F$ is a valid CDF, then: (a) $F$ is non-decreasing. (b) $F$ is right-continuous. (c) $P(X = a) = F(a) - \lim_{x \to a^-} F(x)$.

Show Solution

(a) Non-decreasing: Let $a < b$. Then $\{X \le a\} \subseteq \{X \le b\}$ since any outcome with $X(\omega) \le a$ also has $X(\omega) \le b$. By monotonicity of probability:

$$F(a) = P(X \le a) \le P(X \le b) = F(b) \quad \square$$

(b) Right-continuity: Let $x_n \downarrow a$ (decreasing to $a$). Define $A_n = \{X \le x_n\}$. Then $A_1 \supseteq A_2 \supseteq \cdots$ and $\bigcap_n A_n = \{X \le a\}$. By continuity of probability:

$$\lim_{n \to \infty} F(x_n) = \lim_{n \to \infty} P(A_n) = P\!\left(\bigcap_n A_n\right) = P(X \le a) = F(a) \quad \square$$

(c) Point mass formula: Let $x_n \uparrow a$ (increasing to $a$, with $x_n < a$). Define $B_n = \{X \le x_n\}$. Then $B_1 \subseteq B_2 \subseteq \cdots$ and $\bigcup_n B_n = \{X < a\}$. By continuity of probability:

$$\lim_{x \to a^-} F(x) = P(X < a)$$

Therefore:

$$P(X = a) = P(X \le a) - P(X < a) = F(a) - \lim_{x \to a^-} F(x) = F(a) - F(a^-) \quad \square$$

This shows that jumps in the CDF correspond exactly to point masses (atoms) of the distribution.

Exercise 23: Multi-Step Transformation

Let $U \sim \text{Uniform}(0, 1)$. Find the PDF of $Y = \tan(\pi(U - 1/2))$. Name the resulting distribution and explain why it has no finite mean.

Show Solution

CDF method: For $y \in \mathbb{R}$:

$$F_Y(y) = P(Y \le y) = P(\tan(\pi(U - 1/2)) \le y)$$

Since $U \in (0, 1)$, we have $\pi(U - 1/2) \in (-\pi/2, \pi/2)$, and $\tan$ is strictly increasing on this interval. So:

$$F_Y(y) = P\!\left(U \le \frac{1}{\pi}\arctan(y) + \frac{1}{2}\right) = \frac{1}{\pi}\arctan(y) + \frac{1}{2}$$

PDF: Differentiating:

$$f_Y(y) = \frac{1}{\pi} \cdot \frac{1}{1 + y^2}, \quad y \in \mathbb{R}$$

This is the standard Cauchy distribution (also $t$-distribution with 1 degree of freedom).

Why no finite mean: The integral $\int_{-\infty}^{\infty} \frac{|y|}{\pi(1+y^2)} \, dy$ diverges because for large $|y|$, the integrand behaves like $\frac{1}{\pi|y|}$, and $\int_1^{\infty} \frac{1}{y} dy = \infty$. The tails are too heavy—extreme values are sufficiently likely that the average does not converge.

Answer: $f_Y(y) = \frac{1}{\pi(1 + y^2)}$, the standard Cauchy distribution. $E[Y]$ is undefined.

Exercise 24: Log-Normal Distribution via Change of Variables

Let $Z \sim \mathcal{N}(\mu, \sigma^2)$ and $Y = e^Z$. (a) Derive the PDF of $Y$ (the log-normal distribution). (b) Show that $E[Y] = e^{\mu + \sigma^2/2}$. (c) Show that the median of $Y$ is $e^{\mu}$, which is strictly less than the mean for $\sigma > 0$.

Show Solution

(a) PDF of $Y$: $g(z) = e^z$ is strictly increasing, so $g^{-1}(y) = \ln(y)$ for $y > 0$.

Jacobian: $|dg^{-1}/dy| = 1/y$.

$$f_Y(y) = f_Z(\ln y) \cdot \frac{1}{y} = \frac{1}{y\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(\ln y - \mu)^2}{2\sigma^2}\right), \quad y > 0$$

(b) Mean of $Y$:

$$E[Y] = E[e^Z] = \int_{-\infty}^{\infty} e^z \cdot \frac{1}{\sigma\sqrt{2\pi}} e^{-(z-\mu)^2/(2\sigma^2)} \, dz$$

Complete the square in the exponent:

$$z - \frac{(z-\mu)^2}{2\sigma^2} = -\frac{z^2 - 2z(\mu + \sigma^2)}{2\sigma^2} + \mu = -\frac{(z - (\mu + \sigma^2))^2}{2\sigma^2} + \mu + \frac{\sigma^2}{2}$$

So:

$$E[Y] = e^{\mu + \sigma^2/2} \int_{-\infty}^{\infty} \frac{1}{\sigma\sqrt{2\pi}} e^{-(z-\mu-\sigma^2)^2/(2\sigma^2)} \, dz = e^{\mu + \sigma^2/2} \cdot 1$$

(c) Median: The median $m$ satisfies $P(Y \le m) = 1/2$, i.e., $P(e^Z \le m) = P(Z \le \ln m) = 1/2$. Since $Z \sim \mathcal{N}(\mu, \sigma^2)$, its median is $\mu$, so $\ln m = \mu$, giving $m = e^{\mu}$.

Since $e^{\mu} < e^{\mu + \sigma^2/2}$ for $\sigma > 0$, the median is strictly less than the mean. This right-skewness is characteristic of log-normal distributions and is important in modeling income, stock prices, and other positive quantities. $\square$

Answer: $f_Y(y) = \frac{1}{y\sigma\sqrt{2\pi}} e^{-(\ln y - \mu)^2/(2\sigma^2)}$ for $y > 0$. Mean $= e^{\mu + \sigma^2/2}$, median $= e^{\mu}$.

Exercise 25: Probability Integral Transform

(a) Prove the probability integral transform: if $X$ is a continuous random variable with CDF $F_X$, then $U = F_X(X) \sim \text{Uniform}(0, 1)$. (b) Prove the converse: if $U \sim \text{Uniform}(0, 1)$, then $X = F_X^{-1}(U)$ has CDF $F_X$. (c) Explain why this result is foundational for simulation, and give an example where the inverse CDF does not exist in closed form.

Show Solution

(a) Forward direction: Let $U = F_X(X)$. Since $F_X$ is continuous and non-decreasing, for $u \in [0, 1]$:

$$P(U \le u) = P(F_X(X) \le u)$$

Since $F_X$ is strictly increasing (for a continuous RV), $F_X(X) \le u \iff X \le F_X^{-1}(u)$. Therefore:

$$P(U \le u) = P(X \le F_X^{-1}(u)) = F_X(F_X^{-1}(u)) = u$$

This is the CDF of $\text{Uniform}(0, 1)$. $\square$

(b) Converse: Let $X = F_X^{-1}(U)$ where $U \sim \text{Uniform}(0, 1)$. Then:

$$P(X \le x) = P(F_X^{-1}(U) \le x) = P(U \le F_X(x)) = F_X(x)$$

since $P(U \le u) = u$ for $U \sim \text{Uniform}(0, 1)$. $\square$

(c) Significance for simulation: This result means we can generate samples from any continuous distribution using only a uniform random number generator:

  1. Generate $U \sim \text{Uniform}(0, 1)$ (computers can do this).
  2. Compute $X = F_X^{-1}(U)$ (the inverse CDF, also called the quantile function).

Then $X$ has the desired distribution $F_X$. This is the inverse transform method.

When inverse CDF lacks closed form: The standard normal distribution $\Phi(x) = \int_{-\infty}^{x} \frac{1}{\sqrt{2\pi}} e^{-t^2/2} \, dt$ has no closed-form inverse. In practice, we use either (1) numerical approximations of $\Phi^{-1}$ (e.g., the rational approximation used in software), or (2) alternative sampling methods like the Box-Muller transform: if $U_1, U_2 \sim \text{Uniform}(0, 1)$, then $Z = \sqrt{-2\ln U_1} \cos(2\pi U_2) \sim \mathcal{N}(0, 1)$.

Exercise 26: Characteristic Function Basics

The characteristic function of a random variable $X$ is $\varphi_X(t) = E[e^{itX}]$. (a) Compute $\varphi_X(t)$ for $X \sim \text{Bernoulli}(p)$. (b) Compute $\varphi_X(t)$ for $X \sim \mathcal{N}(0,1)$. (c) Show that $|\varphi_X(t)| \leq 1$ for all $t$.

Show Solution

(a) $\varphi_X(t) = (1-p)e^{i \cdot 0 \cdot t} + pe^{it} = (1 - p) + pe^{it}$.

(b) $\varphi_X(t) = \int_{-\infty}^{\infty} e^{itx} \frac{1}{\sqrt{2\pi}} e^{-x^2/2} \, dx$. Complete the square in the exponent: $itx - x^2/2 = -(x - it)^2/2 - t^2/2$. So $\varphi_X(t) = e^{-t^2/2}$.

(c) $|\varphi_X(t)| = |E[e^{itX}]| \leq E[|e^{itX}|] = E[1] = 1$, since $|e^{i\theta}| = 1$ for all real $\theta$. $\square$

Exercise 27: Distribution of $X^2$ for Normal

Let $X \sim \mathcal{N}(0,1)$. Find the distribution of $Y = X^2$. (a) Find the CDF of $Y$. (b) Find the PDF of $Y$. (c) Identify this as a $\chi^2$ distribution with 1 degree of freedom and verify its mean and variance.

Show Solution

(a) For $y > 0$: $F_Y(y) = P(X^2 \leq y) = P(-\sqrt{y} \leq X \leq \sqrt{y}) = \Phi(\sqrt{y}) - \Phi(-\sqrt{y}) = 2\Phi(\sqrt{y}) - 1$.

(b) $f_Y(y) = F_Y'(y) = 2\phi(\sqrt{y}) \cdot \frac{1}{2\sqrt{y}} = \frac{\phi(\sqrt{y})}{\sqrt{y}} = \frac{1}{\sqrt{2\pi y}} e^{-y/2}$ for $y > 0$.

(c) This is the $\text{Gamma}(1/2, 1/2)$ distribution, also written $\chi^2_1$. The PDF matches $\frac{1}{2^{1/2}\Gamma(1/2)} y^{-1/2} e^{-y/2}$ since $\Gamma(1/2) = \sqrt{\pi}$.

$E[Y] = E[X^2] = \text{Var}(X) + (E[X])^2 = 1 + 0 = 1$. $\text{Var}(Y) = E[X^4] - (E[X^2])^2 = 3 - 1 = 2$.

Exercise 28: Stieltjes Integration

A random variable $X$ has CDF $F(x) = 0$ for $x < 0$, $F(x) = x/2$ for $0 \leq x < 1$, and $F(x) = (x+1)/3$ for $1 \leq x < 2$, $F(x) = 1$ for $x \geq 2$. Note the jump at $x = 1$. (a) Is $X$ continuous, discrete, or mixed? (b) Find $P(X = 1)$. (c) Compute $E[X]$ using the Riemann-Stieltjes integral.

Show Solution

(a) $F$ is continuous on $[0,1)$ and $[1,2)$ but has a jump at $x=1$: $F(1^-) = 1/2$ and $F(1) = 2/3$. So $X$ is mixed.

(b) $P(X = 1) = F(1) - F(1^-) = 2/3 - 1/2 = 1/6$.

(c) $E[X] = \int x \, dF(x)$. Split into continuous and discrete parts:

$$E[X] = \int_0^1 x \cdot \frac{1}{2} \, dx + 1 \cdot \frac{1}{6} + \int_1^2 x \cdot \frac{1}{3} \, dx$$ $$= \frac{1}{2} \cdot \frac{1}{2} + \frac{1}{6} + \frac{1}{3}\left[\frac{x^2}{2}\right]_1^2 = \frac{1}{4} + \frac{1}{6} + \frac{1}{3} \cdot \frac{3}{2} = \frac{1}{4} + \frac{1}{6} + \frac{1}{2} = \frac{11}{12}$$

Exercise 29: Order Statistics

Let $X_1, \ldots, X_n$ be i.i.d. $\text{Uniform}(0,1)$. Let $X_{(k)}$ denote the $k$-th order statistic (the $k$-th smallest value). (a) Derive the PDF of $X_{(k)}$. (b) Show that $E[X_{(k)}] = k/(n+1)$. (c) For $n = 4$, find $\text{Var}(X_{(2)})$.

Show Solution

(a) The PDF of $X_{(k)}$ for $\text{Uniform}(0,1)$: $f_{X_{(k)}}(x) = \frac{n!}{(k-1)!(n-k)!} x^{k-1}(1-x)^{n-k}$ for $x \in (0,1)$.

This is a $\text{Beta}(k, n-k+1)$ distribution.

(b) $E[X_{(k)}] = \frac{k}{k + (n-k+1)} = \frac{k}{n+1}$, using the mean of $\text{Beta}(\alpha, \beta) = \frac{\alpha}{\alpha + \beta}$.

(c) $X_{(2)} \sim \text{Beta}(2, 3)$. $\text{Var} = \frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)} = \frac{2 \cdot 3}{25 \cdot 6} = \frac{6}{150} = \frac{1}{25} = 0.04$.

Exercise 30: Multivariate Change of Variables

Let $(X, Y)$ be jointly uniform on the unit disk $\{(x,y): x^2 + y^2 \leq 1\}$, so $f_{X,Y}(x,y) = 1/\pi$. Transform to polar coordinates $(R, \Theta)$ where $X = R\cos\Theta$, $Y = R\sin\Theta$. (a) Compute the Jacobian. (b) Find the joint PDF $f_{R,\Theta}(r,\theta)$. (c) Show that $R$ and $\Theta$ are independent and find their marginal distributions.

Show Solution

(a) The Jacobian of the transformation $(r,\theta) \to (x,y)$:

$$|J| = \left|\det \begin{pmatrix} \cos\theta & -r\sin\theta \\ \sin\theta & r\cos\theta \end{pmatrix}\right| = |r\cos^2\theta + r\sin^2\theta| = r$$

(b) $f_{R,\Theta}(r,\theta) = f_{X,Y}(r\cos\theta, r\sin\theta) \cdot |J| = \frac{1}{\pi} \cdot r = \frac{r}{\pi}$ for $0 \leq r \leq 1$, $0 \leq \theta < 2\pi$.

(c) $f_{R,\Theta}(r,\theta) = \frac{r}{\pi} = (2r) \cdot \frac{1}{2\pi} = f_R(r) \cdot f_\Theta(\theta)$.

So $R$ and $\Theta$ are independent with $f_R(r) = 2r$ for $r \in [0,1]$ (triangular) and $\Theta \sim \text{Uniform}(0, 2\pi)$.
