The Stoichiometric Matrix as a Linear Map

A metabolic network with $m$ metabolites and $n$ reactions is fully encoded by its stoichiometric matrix $S \in \mathbb{R}^{m \times n}$. Each column $S_{\cdot j}$ represents reaction $j$: negative entries for consumed metabolites, positive entries for produced metabolites. Each row $S_{i \cdot}$ encodes all the reactions that affect metabolite $i$.

The matrix $S$ defines a linear transformation from flux space $\mathbb{R}^n$ to metabolite production-rate space $\mathbb{R}^m$:

$$S : \mathbb{R}^n \to \mathbb{R}^m, \qquad \mathbf{v} \mapsto S\mathbf{v}$$

Given a flux vector $\mathbf{v} = (v_1, v_2, \ldots, v_n)^T$, where $v_j$ is the rate of reaction $j$, the product $S\mathbf{v}$ gives the net production rate of every metabolite. The $i$-th component is:

$$(S\mathbf{v})_i = \sum_{j=1}^{n} S_{ij} v_j = \frac{d[\text{metabolite } i]}{dt}$$

This is nothing other than a linear combination of the columns of $S$, weighted by the fluxes — a fact that connects metabolic modeling directly to the core machinery of linear algebra.

The Stoichiometric Matrix as a Linear Map

The stoichiometric matrix $S \in \mathbb{R}^{m \times n}$ defines a linear map $S : \mathbb{R}^n \to \mathbb{R}^m$ that sends a flux vector $\mathbf{v}$ to the vector of net metabolite production rates $S\mathbf{v}$. The steady-state assumption of constraint-based modeling requires:

$$S\mathbf{v} = \mathbf{0}$$

This states that every internal metabolite is produced and consumed at equal rates — the cell is in metabolic homeostasis. The set of all flux vectors satisfying this constraint is precisely the null space of $S$.

From the perspective of linear algebra, $S$ is just an $m \times n$ matrix, and everything we know about the structure of linear maps applies directly. The rank of $S$ tells us the dimension of the image (achievable production rates), and the dimension of the null space tells us the degrees of freedom remaining after imposing the steady-state constraint. The next section makes this connection precise by examining all four fundamental subspaces of $S$.

The Four Fundamental Subspaces

Gilbert Strang's four fundamental subspaces of a matrix provide the complete geometric picture of any linear map. Applied to the stoichiometric matrix $S \in \mathbb{R}^{m \times n}$ of rank $r$, each subspace has a direct biological meaning.

Column Space of $S$: Achievable Production Rates

The column space $\text{Col}(S) \subseteq \mathbb{R}^m$ is the set of all vectors that can be expressed as $S\mathbf{v}$ for some flux vector $\mathbf{v}$:

$$\text{Col}(S) = \{S\mathbf{v} : \mathbf{v} \in \mathbb{R}^n\} \subseteq \mathbb{R}^m, \qquad \dim = r$$

Biological meaning: This is the set of all achievable metabolite production-rate vectors. If a particular production pattern $\mathbf{b}$ lies outside $\text{Col}(S)$, there exists no flux distribution that can produce it — the network simply cannot achieve that metabolic state. The dimension $r$ tells us how many independent production modes the network supports.

Null Space of $S$: Steady-State Flux Distributions

The null space $\text{Null}(S) \subseteq \mathbb{R}^n$ is the kernel of the linear map:

$$\text{Null}(S) = \{\mathbf{v} \in \mathbb{R}^n : S\mathbf{v} = \mathbf{0}\} \subseteq \mathbb{R}^n, \qquad \dim = n - r$$

Biological meaning: This is the feasible space for Flux Balance Analysis. Every vector in $\text{Null}(S)$ is a flux distribution that maintains steady state — no metabolite accumulates or depletes. The dimension $n - r$ counts the degrees of freedom in the network: the number of independent flux values that can be freely chosen. In genome-scale metabolic models, $n \gg r$ typically, so the null space is high-dimensional and FBA uses an objective function to select a single optimal point within it.

Row Space of $S$: Thermodynamic Driving Forces

The row space $\text{Row}(S) = \text{Col}(S^T) \subseteq \mathbb{R}^n$ consists of all linear combinations of the rows of $S$:

$$\text{Row}(S) = \{S^T\mathbf{y} : \mathbf{y} \in \mathbb{R}^m\} \subseteq \mathbb{R}^n, \qquad \dim = r$$

Biological meaning: The row space encodes thermodynamic driving forces. A vector of chemical potentials $\boldsymbol{\mu} \in \mathbb{R}^m$ (one per metabolite) induces a vector of reaction Gibbs free energies via $\Delta \mathbf{G} = S^T \boldsymbol{\mu}$. The row space is exactly the set of reaction-energy vectors that arise from some assignment of chemical potentials. The fact that $\text{Row}(S) \perp \text{Null}(S)$ means: at steady state, the net thermodynamic work done by the system through internal cycles is zero — internal loops cannot extract free energy.

Left Null Space of $S$: Conservation Relations

The left null space $\text{Null}(S^T) \subseteq \mathbb{R}^m$ is the null space of $S^T$:

$$\text{Null}(S^T) = \{\mathbf{l} \in \mathbb{R}^m : S^T\mathbf{l} = \mathbf{0}\} = \{\mathbf{l} \in \mathbb{R}^m : \mathbf{l}^T S = \mathbf{0}^T\} \subseteq \mathbb{R}^m, \qquad \dim = m - r$$

Biological meaning: Each vector $\mathbf{l}$ in the left null space defines a conserved moiety — a linear combination of metabolite concentrations that remains constant over time regardless of the fluxes. For example, if $\mathbf{l}$ has nonzero entries only for ATP, ADP, and AMP, then $\mathbf{l}^T \mathbf{x}(t) = \text{const}$ means the total adenine nucleotide pool is conserved. We derive this formally in Section 4.

Orthogonal Complement Structure

The four subspaces come in two orthogonal pairs, and their dimensions satisfy the rank-nullity theorem in both directions:

  • $\text{Null}(S) \oplus \text{Row}(S) = \mathbb{R}^n$, with $\dim\text{Null}(S) + \dim\text{Row}(S) = (n - r) + r = n$
  • $\text{Null}(S^T) \oplus \text{Col}(S) = \mathbb{R}^m$, with $\dim\text{Null}(S^T) + \dim\text{Col}(S) = (m - r) + r = m$

Null Space Analysis

The null space of $S$ is the central object in constraint-based metabolic modeling. Its dimension controls the degrees of freedom in the network, and its basis vectors represent independent steady-state flux modes. We now develop the theory for computing and interpreting $\text{Null}(S)$.

Dimension and Degrees of Freedom

By the rank-nullity theorem, the dimension of the null space is:

$$\dim\text{Null}(S) = n - \text{rank}(S) = n - r$$

For a typical genome-scale model like E. coli iJO1366, we have $m = 1805$ metabolites and $n = 2583$ reactions, with $\text{rank}(S) \approx 1780$. This gives a null space of dimension $\approx 803$ — meaning there are roughly 800 independent degrees of freedom in the flux distribution. This is why FBA needs an objective function: without it, the steady-state constraint alone admits an enormous family of solutions.

Computing the Null Space via SVD

The most numerically stable method for computing the null space uses the Singular Value Decomposition. Factor $S$ as:

$$S = U \Sigma V^T$$

where $U \in \mathbb{R}^{m \times m}$ and $V \in \mathbb{R}^{n \times n}$ are orthogonal matrices, and $\Sigma \in \mathbb{R}^{m \times n}$ is diagonal with singular values $\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > \sigma_{r+1} = \cdots = \sigma_{\min(m,n)} = 0$.

Derivation: Null Space from SVD
1
Start with $S\mathbf{v} = \mathbf{0}$ and substitute the SVD: $$U \Sigma V^T \mathbf{v} = \mathbf{0}$$
2
Left-multiply by $U^T$ (orthogonal, so $U^T U = I$): $$\Sigma V^T \mathbf{v} = \mathbf{0}$$

Let $\mathbf{w} = V^T \mathbf{v}$, so we need $\Sigma \mathbf{w} = \mathbf{0}$.

3
Since $\Sigma$ is diagonal, $\Sigma \mathbf{w} = \mathbf{0}$ requires $\sigma_i w_i = 0$ for each $i$:

For $i \leq r$: $\sigma_i > 0$, so $w_i = 0$.

For $i > r$: $\sigma_i = 0$, so $w_i$ can be anything.

4
Transform back: $\mathbf{v} = V\mathbf{w}$.

Since only $w_{r+1}, \ldots, w_n$ are free, $\mathbf{v}$ is a linear combination of columns $r+1, \ldots, n$ of $V$:

$$\boxed{\text{Null}(S) = \text{span}\{\mathbf{v}_{r+1}, \mathbf{v}_{r+2}, \ldots, \mathbf{v}_n\}}$$

where $\mathbf{v}_j$ denotes the $j$-th column of $V$. These are the right singular vectors corresponding to the zero singular values.

Example: A Small Metabolic Network

Consider a toy network with 3 metabolites ($A$, $B$, $C$) and 4 reactions:

  • $v_1$: $\to A$  (uptake of A)
  • $v_2$: $A \to B$  (conversion)
  • $v_3$: $A \to C$  (branching)
  • $v_4$: $B + C \to$  (export)

The stoichiometric matrix is:

$$S = \begin{pmatrix} 1 & -1 & -1 & 0 \\ 0 & 1 & 0 & -1 \\ 0 & 0 & 1 & -1 \end{pmatrix}$$

Row reduction reveals $\text{rank}(S) = 3$, so $\dim\text{Null}(S) = 4 - 3 = 1$. There is exactly one degree of freedom: once you fix the uptake flux $v_1$, the remaining fluxes are determined. Solving $S\mathbf{v} = \mathbf{0}$: from row 3, $v_3 = v_4$; from row 2, $v_2 = v_4$; from row 1, $v_1 = v_2 + v_3 = 2v_4$. Setting $v_4 = t$:

$$\text{Null}(S) = \text{span}\left\{\begin{pmatrix} 2 \\ 1 \\ 1 \\ 1 \end{pmatrix}\right\}$$

The single null space basis vector tells us: at steady state, the uptake flux is twice the export flux, and the branching pathway splits equally. Every feasible flux distribution is a scalar multiple of this vector (subject to non-negativity constraints in the full FBA problem).

Conservation Relations

The left null space of $S$ reveals conservation laws — quantities that remain invariant over time regardless of the flux distribution. These are among the most powerful structural insights that linear algebra provides about a metabolic network.

Mathematical Derivation

Derivation: Conservation Relations from the Left Null Space
1
The metabolite dynamics are governed by: $$\frac{d\mathbf{x}}{dt} = S\mathbf{v}(t)$$

where $\mathbf{x}(t) \in \mathbb{R}^m$ is the vector of metabolite concentrations and $\mathbf{v}(t)$ is the flux vector (which may change over time).

2
Let $\mathbf{l} \in \text{Null}(S^T)$, meaning $\mathbf{l}^T S = \mathbf{0}^T$.

Consider the quantity $\mathbf{l}^T \mathbf{x}(t)$ — a weighted sum of metabolite concentrations.

3
Differentiate with respect to time: $$\frac{d}{dt}\left(\mathbf{l}^T \mathbf{x}(t)\right) = \mathbf{l}^T \frac{d\mathbf{x}}{dt} = \mathbf{l}^T S \mathbf{v}(t) = \mathbf{0}^T \mathbf{v}(t) = 0$$
4
Therefore $\mathbf{l}^T \mathbf{x}(t) = \text{constant}$ for all time: $$\boxed{\mathbf{l}^T \mathbf{x}(t) = \mathbf{l}^T \mathbf{x}(0) \quad \forall\, t \geq 0, \quad \forall\, \mathbf{v}(t)}$$

This holds for any flux distribution — not just steady-state fluxes. The conservation is a structural property of the network topology, independent of kinetics or regulation.

Biological Significance: Conserved Moieties

Each linearly independent vector in $\text{Null}(S^T)$ identifies a conserved moiety (also called a conserved pool). Classic biological examples include:

  • Adenine nucleotide pool: $[\text{ATP}] + [\text{ADP}] + [\text{AMP}] = \text{const}$. The corresponding left null space vector $\mathbf{l}$ has entry 1 for ATP, ADP, and AMP, and 0 elsewhere.
  • NAD/NADH pool: $[\text{NAD}^+] + [\text{NADH}] = \text{const}$. The cell interconverts these cofactors but cannot create or destroy the total pool through internal reactions.
  • CoA pool: $[\text{CoA}] + [\text{Acetyl\text{-}CoA}] + [\text{Succinyl\text{-}CoA}] + \cdots = \text{const}$.

Computing Conservation Relations via SVD

From the SVD $S = U\Sigma V^T$, the left null space is spanned by the columns of $U$ corresponding to zero singular values:

$$\text{Null}(S^T) = \text{span}\{\mathbf{u}_{r+1}, \mathbf{u}_{r+2}, \ldots, \mathbf{u}_m\}$$

The dimension $m - r$ tells us the number of independent conservation relations. In our toy example above, $m = 3$ and $r = 3$, so $m - r = 0$ — there are no conservation relations, which makes sense because none of the metabolites share a conserved moiety.

LP Duality and Shadow Prices

Flux Balance Analysis solves a linear program (LP). Every LP has a dual, and the dual variables have profound biological meaning: they are the shadow prices of metabolites, measuring each metabolite's marginal contribution to the cellular objective.

The Primal FBA Problem

The standard FBA formulation is:

$$\max_{\mathbf{v}} \quad \mathbf{c}^T \mathbf{v}$$ $$\text{s.t.} \quad S\mathbf{v} = \mathbf{0}$$ $$\mathbf{v}_{lb} \leq \mathbf{v} \leq \mathbf{v}_{ub}$$

where $\mathbf{c}$ is the objective vector (typically the biomass reaction), and $\mathbf{v}_{lb}, \mathbf{v}_{ub}$ are lower and upper bounds on each flux.

Deriving the Dual LP

Derivation: Dual of the FBA Linear Program
1
Rewrite the bound constraints as standard form.

Introduce slack: $\mathbf{v} - \mathbf{v}_{lb} \geq \mathbf{0}$ and $\mathbf{v}_{ub} - \mathbf{v} \geq \mathbf{0}$. Associate dual variables $\boldsymbol{\mu}^- \geq \mathbf{0}$ with the lower bounds and $\boldsymbol{\mu}^+ \geq \mathbf{0}$ with the upper bounds. Let $\mathbf{y} \in \mathbb{R}^m$ (unrestricted) be the dual variables for the equality constraints $S\mathbf{v} = \mathbf{0}$.

2
Form the Lagrangian: $$L(\mathbf{v}, \mathbf{y}, \boldsymbol{\mu}^+, \boldsymbol{\mu}^-) = \mathbf{c}^T \mathbf{v} - \mathbf{y}^T(S\mathbf{v}) - (\boldsymbol{\mu}^+)^T(\mathbf{v} - \mathbf{v}_{ub}) + (\boldsymbol{\mu}^-)^T(\mathbf{v} - \mathbf{v}_{lb})$$

Wait — we need to be careful with signs. Since the primal is a max, the dual is a min. The Lagrangian for a maximization problem subtracts the constraint violations:

$$L = \mathbf{c}^T\mathbf{v} - \mathbf{y}^T S\mathbf{v} + (\boldsymbol{\mu}^-)^T(\mathbf{v} - \mathbf{v}_{lb}) - (\boldsymbol{\mu}^+)^T(\mathbf{v} - \mathbf{v}_{ub})$$
3
Collect terms in $\mathbf{v}$: $$L = \left(\mathbf{c} - S^T\mathbf{y} + \boldsymbol{\mu}^- - \boldsymbol{\mu}^+\right)^T \mathbf{v} - (\boldsymbol{\mu}^-)^T \mathbf{v}_{lb} + (\boldsymbol{\mu}^+)^T \mathbf{v}_{ub}$$
4
For the dual, maximize over $\mathbf{v}$ (unconstrained).

The Lagrangian is linear in $\mathbf{v}$, so it is bounded above only if the coefficient of $\mathbf{v}$ is zero:

$$S^T\mathbf{y} + \boldsymbol{\mu}^+ - \boldsymbol{\mu}^- = \mathbf{c}$$
5
The dual LP is therefore: $$\boxed{\min_{\mathbf{y},\, \boldsymbol{\mu}^+,\, \boldsymbol{\mu}^-} \quad \mathbf{v}_{ub}^T \boldsymbol{\mu}^+ - \mathbf{v}_{lb}^T \boldsymbol{\mu}^-}$$ $$\text{s.t.} \quad S^T\mathbf{y} + \boldsymbol{\mu}^+ - \boldsymbol{\mu}^- = \mathbf{c}$$ $$\boldsymbol{\mu}^+ \geq \mathbf{0}, \quad \boldsymbol{\mu}^- \geq \mathbf{0}, \quad \mathbf{y} \text{ free}$$

By strong duality (which holds because the primal is feasible and bounded), the optimal dual objective equals the optimal primal objective: $\mathbf{c}^T \mathbf{v}^* = \mathbf{v}_{ub}^T (\boldsymbol{\mu}^+)^* - \mathbf{v}_{lb}^T (\boldsymbol{\mu}^-)^*$.

Interpreting Shadow Prices

The dual variable $y_i$ associated with the steady-state constraint for metabolite $i$ is called the shadow price of that metabolite. Its interpretation:

$$y_i = \frac{\partial z^*}{\partial b_i} \bigg|_{b_i = 0}$$

where $z^* = \mathbf{c}^T \mathbf{v}^*$ is the optimal objective value and $b_i$ is the right-hand side of the $i$-th equality constraint (which is zero at steady state). In biological terms:

  • $y_i > 0$: Metabolite $i$ is a bottleneck. If we could supply a small amount of metabolite $i$ exogenously (increasing $b_i$ from 0 to $\epsilon$), the optimal growth rate would increase by approximately $y_i \cdot \epsilon$.
  • $y_i = 0$: Metabolite $i$ is not limiting. The cell has sufficient capacity to produce or acquire it.
  • $y_i < 0$: Metabolite $i$ is in excess. Forcing its accumulation (increasing $b_i$) would actually decrease growth.
Shadow Prices as Metabolic Currency

Shadow prices define an internal pricing system for the cell. Just as a market assigns prices reflecting scarcity, the dual solution assigns to each metabolite a "price" reflecting its marginal value for growth. High shadow prices indicate metabolic bottlenecks — these are prime targets for metabolic engineering. A reaction's contribution to growth equals the dot product of its stoichiometric column with the shadow price vector: $\Delta z_j = \mathbf{y}^T S_{\cdot j} \cdot v_j$.

Reduced Costs

While shadow prices quantify the value of metabolites, reduced costs quantify the value of reactions. The reduced cost of reaction $j$ is defined as:

$$\bar{c}_j = c_j - \mathbf{y}^T S_{\cdot j}$$

where $c_j$ is the objective coefficient of reaction $j$, $\mathbf{y}$ is the vector of shadow prices, and $S_{\cdot j}$ is the $j$-th column of $S$ (the stoichiometry of reaction $j$).

Interpretation via the Dual Constraint

The dual constraint $S^T\mathbf{y} + \boldsymbol{\mu}^+ - \boldsymbol{\mu}^- = \mathbf{c}$ can be rewritten component-wise as:

$$\bar{c}_j = c_j - (S^T\mathbf{y})_j = \mu_j^+ - \mu_j^-$$

By complementary slackness, at optimality:

  • If $v_j^{lb} < v_j^* < v_j^{ub}$ (reaction $j$ is interior), then $\mu_j^+ = \mu_j^- = 0$, so $\bar{c}_j = 0$. The reaction is at its optimal level.
  • If $v_j^* = v_j^{ub}$ (reaction $j$ is at its upper bound), then $\mu_j^+ \geq 0$, so $\bar{c}_j \geq 0$. Increasing the upper bound would improve the objective by $\mu_j^+$ per unit.
  • If $v_j^* = v_j^{lb}$ (reaction $j$ is at its lower bound), then $\mu_j^- \geq 0$, so $\bar{c}_j \leq 0$. The reaction is being "held back" because using it more would hurt the objective.

Economic Interpretation

The term $\mathbf{y}^T S_{\cdot j}$ measures the total value of the metabolites produced by reaction $j$ (where "value" is determined by the shadow prices). The reduced cost $\bar{c}_j = c_j - \mathbf{y}^T S_{\cdot j}$ is then:

$$\bar{c}_j = \underbrace{c_j}_{\substack{\text{direct contribution} \\ \text{to objective}}} - \underbrace{\mathbf{y}^T S_{\cdot j}}_{\substack{\text{value of metabolites} \\ \text{produced by reaction } j}}$$

If $\bar{c}_j > 0$ for a reaction at its upper bound, the reaction is "profitable" — its direct contribution exceeds the metabolic cost of running it, and the cell would benefit from increasing its capacity. If $\bar{c}_j < 0$ for a reaction at its lower bound, running it more would be a net loss. Reactions with $\bar{c}_j = 0$ are in metabolic equilibrium: their direct benefit exactly equals their metabolic cost.

Singular Value Decomposition of $S$

The SVD goes beyond computing null spaces — it reveals the hierarchical structure of the metabolic network. The singular values and singular vectors decompose the network into independent metabolic "modes" ordered by importance.

The Decomposition

Writing $S = U\Sigma V^T$ component-wise as a sum of rank-1 matrices:

$$S = \sum_{k=1}^{r} \sigma_k \, \mathbf{u}_k \mathbf{v}_k^T$$

Each term $\sigma_k \, \mathbf{u}_k \mathbf{v}_k^T$ is a metabolic mode:

  • $\mathbf{v}_k \in \mathbb{R}^n$ is the flux pattern (a profile over all reactions)
  • $\mathbf{u}_k \in \mathbb{R}^m$ is the corresponding metabolite pattern (a profile over all metabolites)
  • $\sigma_k$ is the strength (importance) of this mode

Interpreting the Singular Values

The singular values are ordered $\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > 0$. The largest singular values correspond to the dominant metabolic pathways — the reactions and metabolites that carry the most "flux capacity" in the network. A rapid decay in the singular value spectrum indicates that the network's behavior is well-approximated by a small number of dominant modes:

$$S \approx S_k = \sum_{i=1}^{k} \sigma_i \, \mathbf{u}_i \mathbf{v}_i^T, \qquad \|S - S_k\|_F = \sqrt{\sum_{i=k+1}^{r} \sigma_i^2}$$

By the Eckart-Young theorem, $S_k$ is the best rank-$k$ approximation to $S$ in both the Frobenius and operator norms. This means: the first $k$ metabolic modes capture as much of the network's structure as possible with $k$ dimensions.

Connection to Principal Component Analysis of Flux Distributions

If we sample many feasible flux distributions $\mathbf{v}_1, \ldots, \mathbf{v}_N$ (e.g., from flux sampling) and form the data matrix $X \in \mathbb{R}^{N \times n}$, then PCA of $X$ reveals the dominant directions of variation in flux space. These principal components live in $\text{Null}(S)$ (since all sampled fluxes are steady-state) and describe how the cell can redistribute fluxes while maintaining homeostasis.

The SVD of $S$ itself is complementary: it reveals the structure of the map (which flux patterns produce which metabolite patterns), while PCA of sampled fluxes reveals the structure of the feasible set (how the null space is explored in practice).

Practical Use of SVD in Systems Biology

SVD of stoichiometric matrices has been used to (1) identify metabolic bottlenecks by examining the largest singular values, (2) reduce model complexity by truncating small singular values, (3) detect modeling errors (unexpected zero or near-zero singular values suggest redundant constraints), and (4) compare metabolic network architectures across species by comparing their singular value spectra.

Null Space from SVD with NumPy

import numpy as np
from scipy.linalg import svd, null_space
from scipy.sparse.linalg import svds

# ---------------------------------------------------------
# Example: Toy metabolic network
# Reactions: v1: -> A, v2: A -> B, v3: A -> C, v4: B + C ->
# ---------------------------------------------------------
S = np.array([
    [ 1, -1, -1,  0],   # A
    [ 0,  1,  0, -1],   # B
    [ 0,  0,  1, -1],   # C
])

m, n = S.shape
print(f"Stoichiometric matrix S: {m} metabolites x {n} reactions")

# --- Method 1: scipy.linalg.null_space (convenience) ---
ns = null_space(S)
print(f"\nNull space dimension (scipy): {ns.shape[1]}")
print(f"Null space basis vector:\n{ns.round(6)}")

# Verify: S @ ns should be zero
residual = S @ ns
print(f"Verification ||S @ ns|| = {np.linalg.norm(residual):.2e}")

# --- Method 2: Full SVD (educational) ---
U, sigma, Vt = svd(S, full_matrices=True)
print(f"\nSingular values: {sigma.round(6)}")

# Rank = number of singular values above threshold
tol = max(m, n) * np.finfo(float).eps * sigma[0]
rank = np.sum(sigma > tol)
print(f"Numerical rank: {rank}")
print(f"Null space dimension = n - rank = {n} - {rank} = {n - rank}")

# Null space = last (n - rank) columns of V = last (n - rank) rows of Vt
null_basis = Vt[rank:, :].T
print(f"\nNull space basis (from SVD):\n{null_basis.round(6)}")

# Verify orthogonality between null space and row space
row_basis = Vt[:rank, :]
print(f"\nRow space basis (first {rank} rows of Vt):")
print(row_basis.round(6))
print(f"Dot product (should be ~0): {(row_basis @ null_basis).round(10)}")

Left Null Space and Conservation Relations

def find_conservation_relations(S, metabolite_names=None, tol=1e-10):
    """
    Find conservation relations from the left null space of S.

    The left null space of S consists of vectors l such that l^T S = 0.
    Each such vector defines a conserved quantity: l^T x(t) = const.

    Parameters
    ----------
    S : ndarray, shape (m, n)
        Stoichiometric matrix.
    metabolite_names : list of str, optional
        Names for each metabolite row.
    tol : float
        Threshold for singular values considered zero.

    Returns
    -------
    L : ndarray, shape (m - r, m)
        Rows are the conservation relation vectors.
    """
    U, sigma, Vt = svd(S, full_matrices=True)
    rank = np.sum(sigma > tol * sigma[0] * max(S.shape))

    # Left null space = columns of U beyond rank
    left_null = U[:, rank:]       # shape (m, m-r)
    L = left_null.T               # shape (m-r, m), each row is a conservation vector

    print(f"Matrix rank: {rank}")
    print(f"Number of conservation relations: {S.shape[0] - rank}")

    if metabolite_names and L.shape[0] > 0:
        print("\nConservation relations:")
        for i, row in enumerate(L):
            nonzero = np.where(np.abs(row) > tol)[0]
            terms = []
            for idx in nonzero:
                coeff = row[idx]
                name = metabolite_names[idx]
                if abs(coeff - 1.0) < tol:
                    terms.append(name)
                elif abs(coeff + 1.0) < tol:
                    terms.append(f"-{name}")
                else:
                    terms.append(f"{coeff:.3f}*{name}")
            print(f"  Relation {i+1}: {' + '.join(terms)} = const")

    return L


# --- Example with conserved moieties ---
# Network with ATP/ADP cycling:
#   v1: ATP -> ADP + Pi  (hydrolysis)
#   v2: ADP + Pi -> ATP  (synthesis)
#   v3: -> Pi            (phosphate uptake)
#   v4: Pi ->            (phosphate export)
S_atp = np.array([
    [-1,  1,  0,  0],   # ATP
    [ 1, -1,  0,  0],   # ADP
    [ 1, -1,  1, -1],   # Pi
])

metabolites = ['ATP', 'ADP', 'Pi']
L = find_conservation_relations(S_atp, metabolites)
print(f"\nConservation matrix L:\n{L.round(6)}")

# Verify: L @ S should be zero
print(f"Verification ||L @ S|| = {np.linalg.norm(L @ S_atp):.2e}")

SVD Analysis of a Stoichiometric Matrix

import matplotlib.pyplot as plt

def analyze_svd(S, reaction_names=None, metabolite_names=None):
    """
    Perform SVD analysis of a stoichiometric matrix.

    Plots the singular value spectrum and identifies dominant
    metabolic modes.

    Parameters
    ----------
    S : ndarray, shape (m, n)
        Stoichiometric matrix.
    reaction_names : list of str, optional
    metabolite_names : list of str, optional
    """
    U, sigma, Vt = svd(S, full_matrices=False)
    m, n = S.shape
    r = np.sum(sigma > 1e-10 * sigma[0] * max(m, n))

    # --- Singular value spectrum ---
    fig, axes = plt.subplots(1, 3, figsize=(16, 5))

    # Plot 1: Singular values
    ax = axes[0]
    ax.semilogy(range(1, len(sigma) + 1), sigma, 'o-', color='#3498db',
                markersize=4, linewidth=1.5)
    ax.axhline(y=sigma[0] * 1e-10 * max(m, n), color='#e74c3c',
               linestyle='--', alpha=0.7, label='Numerical zero threshold')
    ax.set_xlabel('Index $k$')
    ax.set_ylabel('$\\sigma_k$ (log scale)')
    ax.set_title('Singular Value Spectrum')
    ax.legend()
    ax.grid(True, alpha=0.3)

    # Plot 2: Cumulative energy
    ax = axes[1]
    energy = np.cumsum(sigma**2) / np.sum(sigma**2)
    ax.plot(range(1, len(sigma) + 1), energy, 's-', color='#27ae60',
            markersize=4, linewidth=1.5)
    ax.axhline(y=0.99, color='#e74c3c', linestyle='--', alpha=0.7,
               label='99% energy')
    k99 = np.searchsorted(energy, 0.99) + 1
    ax.axvline(x=k99, color='#f39c12', linestyle=':', alpha=0.7,
               label=f'k={k99} modes for 99%')
    ax.set_xlabel('Number of modes $k$')
    ax.set_ylabel('Cumulative fraction of $\\|S\\|_F^2$')
    ax.set_title('Cumulative Energy')
    ax.legend()
    ax.grid(True, alpha=0.3)

    # Plot 3: Dominant mode structure
    ax = axes[2]
    if reaction_names:
        # Show top flux contributions for the first mode
        v1 = Vt[0, :]
        top_idx = np.argsort(np.abs(v1))[-10:]
        names = [reaction_names[i] for i in top_idx]
        vals = v1[top_idx]
        colors = ['#27ae60' if v > 0 else '#e74c3c' for v in vals]
        ax.barh(range(len(top_idx)), vals, color=colors, alpha=0.8)
        ax.set_yticks(range(len(top_idx)))
        ax.set_yticklabels(names, fontsize=8)
        ax.set_xlabel('Weight in $\\mathbf{v}_1$')
        ax.set_title('Dominant Flux Mode')
    else:
        v1 = Vt[0, :]
        ax.bar(range(n), v1, color='#3498db', alpha=0.7)
        ax.set_xlabel('Reaction index')
        ax.set_ylabel('Weight in $\\mathbf{v}_1$')
        ax.set_title('Dominant Flux Mode')

    plt.tight_layout()
    plt.show()

    # --- Summary statistics ---
    print(f"Matrix dimensions: {m} x {n}")
    print(f"Rank: {r}")
    print(f"Null space dimension: {n - r}")
    print(f"Left null space dimension: {m - r}")
    print(f"Condition number: {sigma[0]/sigma[r-1]:.2f}")
    print(f"Modes for 99% energy: {k99}")

    return U, sigma, Vt

Shadow Prices with COBRApy

import cobra
from cobra.io import load_model

def analyze_shadow_prices(model_name='textbook'):
    """
    Compute and analyze shadow prices from FBA using COBRApy.

    Shadow prices are the dual variables associated with the
    steady-state constraints Sv = 0. They indicate the marginal
    value of each metabolite to the objective.

    Parameters
    ----------
    model_name : str
        Name of the model to load ('textbook' for E. coli core).
    """
    # Load model
    model = load_model(model_name)
    print(f"Model: {model.id}")
    print(f"  Metabolites: {len(model.metabolites)}")
    print(f"  Reactions:   {len(model.reactions)}")
    print(f"  Genes:       {len(model.genes)}")

    # Solve FBA
    solution = model.optimize()
    print(f"\nObjective value (growth rate): {solution.objective_value:.6f}")

    # --- Shadow prices ---
    shadow_prices = solution.shadow_prices
    print(f"\nShadow prices (top 10 by absolute value):")
    top_sp = shadow_prices.abs().nlargest(10)
    for met_id, val in top_sp.items():
        met = model.metabolites.get_by_id(met_id)
        sp = shadow_prices[met_id]
        print(f"  {met_id:20s} ({met.name:30s}): {sp:+.6f}")

    # --- Reduced costs ---
    reduced_costs = solution.reduced_costs
    nonzero_rc = reduced_costs[reduced_costs.abs() > 1e-8]
    print(f"\nReactions with nonzero reduced costs: {len(nonzero_rc)}")
    print("Top 10 by absolute value:")
    top_rc = reduced_costs.abs().nlargest(10)
    for rxn_id, val in top_rc.items():
        rxn = model.reactions.get_by_id(rxn_id)
        rc = reduced_costs[rxn_id]
        flux = solution.fluxes[rxn_id]
        print(f"  {rxn_id:20s}: rc={rc:+.6f}, flux={flux:.6f}, "
              f"bounds=[{rxn.lower_bound}, {rxn.upper_bound}]")

    # --- Visualization ---
    fig, axes = plt.subplots(1, 2, figsize=(14, 5))

    # Shadow prices distribution
    ax = axes[0]
    sp_vals = shadow_prices.values
    ax.hist(sp_vals[sp_vals != 0], bins=30, color='#3498db',
            alpha=0.7, edgecolor='white')
    ax.set_xlabel('Shadow Price')
    ax.set_ylabel('Count')
    ax.set_title('Distribution of Nonzero Shadow Prices')
    ax.axvline(x=0, color='#e74c3c', linestyle='--', alpha=0.5)

    # Reduced costs distribution
    ax = axes[1]
    rc_vals = reduced_costs.values
    ax.hist(rc_vals[rc_vals != 0], bins=30, color='#27ae60',
            alpha=0.7, edgecolor='white')
    ax.set_xlabel('Reduced Cost')
    ax.set_ylabel('Count')
    ax.set_title('Distribution of Nonzero Reduced Costs')
    ax.axvline(x=0, color='#e74c3c', linestyle='--', alpha=0.5)

    plt.tight_layout()
    plt.show()

    return solution

Full Subspace Analysis of a COBRA Model

import cobra
from cobra.io import load_model

def full_subspace_analysis(model_name='textbook'):
    """
    Complete four-subspace analysis of a COBRA metabolic model.

    Extracts the stoichiometric matrix, computes the SVD, and
    reports dimensions and biological content of each subspace.
    """
    model = load_model(model_name)

    # Extract stoichiometric matrix
    met_ids = [m.id for m in model.metabolites]
    rxn_ids = [r.id for r in model.reactions]
    m_count = len(met_ids)
    n_count = len(rxn_ids)

    S = cobra.util.create_stoichiometric_matrix(model, array_type='dense')
    print(f"Stoichiometric matrix: {m_count} metabolites x {n_count} reactions")

    # SVD
    U, sigma, Vt = svd(S, full_matrices=True)
    tol = max(m_count, n_count) * np.finfo(float).eps * sigma[0]
    rank = np.sum(sigma > tol)

    print(f"\n=== Four Fundamental Subspaces ===")
    print(f"Rank of S: {rank}")
    print(f"  Column space dim:     {rank}  (achievable production rates)")
    print(f"  Null space dim:       {n_count - rank}  (steady-state degrees of freedom)")
    print(f"  Row space dim:        {rank}  (thermodynamic constraints)")
    print(f"  Left null space dim:  {m_count - rank}  (conservation relations)")

    # Conservation relations
    if m_count - rank > 0:
        left_null = U[:, rank:]
        print(f"\n=== Conservation Relations ({m_count - rank} found) ===")
        for i in range(min(5, m_count - rank)):
            vec = left_null[:, i]
            nonzero = np.where(np.abs(vec) > 0.1)[0]
            involved = [(met_ids[j], vec[j]) for j in nonzero]
            involved.sort(key=lambda x: -abs(x[1]))
            terms = [f"{c:.2f}*{name}" for name, c in involved[:6]]
            print(f"  Relation {i+1}: {' + '.join(terms)}")

    # Singular value spectrum summary
    print(f"\n=== Singular Value Spectrum ===")
    print(f"  Largest:  {sigma[0]:.4f}")
    print(f"  Smallest nonzero: {sigma[rank-1]:.6f}")
    print(f"  Condition number: {sigma[0]/sigma[rank-1]:.2f}")

    cumulative = np.cumsum(sigma[:rank]**2) / np.sum(sigma[:rank]**2)
    for threshold in [0.5, 0.9, 0.95, 0.99]:
        k = np.searchsorted(cumulative, threshold) + 1
        print(f"  Modes for {threshold*100:.0f}% energy: {k}/{rank}")

    return S, U, sigma, Vt, rank


S, U, sigma, Vt, rank = full_subspace_analysis('textbook')

Exercises & Progress

Test your understanding of the linear algebra underlying metabolic network analysis, from null space computation to LP duality.

Easy 1. Null Space of a Small Stoichiometric Matrix

Consider a metabolic network with 3 metabolites ($X$, $Y$, $Z$) and 5 reactions:

  • $v_1$: $\to X$  (uptake)
  • $v_2$: $X \to Y$
  • $v_3$: $X \to Z$
  • $v_4$: $Y \to Z$
  • $v_5$: $Z \to$  (export)
  1. Write down the stoichiometric matrix $S \in \mathbb{R}^{3 \times 5}$.
  2. Compute $\text{rank}(S)$ by row reduction. How many degrees of freedom does the network have?
  3. Find a basis for $\text{Null}(S)$ by solving $S\mathbf{v} = \mathbf{0}$ explicitly. Express your answer as $n - r$ linearly independent vectors.
  4. Verify your answer: for each basis vector $\mathbf{v}$, confirm that $S\mathbf{v} = \mathbf{0}$ and give a biological interpretation (which pathway does each null space vector represent?).
Easy 2. Conservation Relations from the Left Null Space

Consider a simplified cofactor cycling network with 4 metabolites (ATP, ADP, NAD$^+$, NADH) and 3 reactions:

  • $v_1$: ATP + NAD$^+$ $\to$ ADP + NADH  (coupled oxidation)
  • $v_2$: ADP $\to$ ATP  (phosphorylation)
  • $v_3$: NADH $\to$ NAD$^+$  (electron transport)
  1. Write down the stoichiometric matrix $S \in \mathbb{R}^{4 \times 3}$.
  2. Compute $\text{rank}(S)$. What is $\dim\text{Null}(S^T) = m - r$?
  3. Find a basis for the left null space by solving $\mathbf{l}^T S = \mathbf{0}^T$.
  4. Interpret each conservation relation biologically. Which pools are conserved, and why does this make biochemical sense?
Medium 3. Shadow Prices: Computation and Interpretation

Consider the following small FBA problem with stoichiometric matrix, bounds, and objective:

$$S = \begin{pmatrix} 1 & -1 & -1 & 0 \\ 0 & 1 & 0 & -1 \end{pmatrix}, \qquad \mathbf{c} = \begin{pmatrix} 0 \\ 0 \\ 0 \\ 1 \end{pmatrix}$$

Bounds: $0 \leq v_1 \leq 10$, $0 \leq v_2 \leq 8$, $0 \leq v_3 \leq 6$, $0 \leq v_4 \leq \infty$.

  1. Solve the primal FBA problem: $\max \mathbf{c}^T\mathbf{v}$ subject to $S\mathbf{v} = 0$ and the bounds. Find the optimal $\mathbf{v}^*$ and $z^* = \mathbf{c}^T\mathbf{v}^*$.
  2. Write down the dual LP explicitly. Identify the dual variables $\mathbf{y}$, $\boldsymbol{\mu}^+$, and $\boldsymbol{\mu}^-$.
  3. Solve the dual and find the shadow prices $y_1$ and $y_2$. Verify strong duality: the dual objective equals $z^*$.
  4. Interpret the shadow prices: which metabolite is the bottleneck for the objective? If the upper bound on $v_1$ were increased to 11, what would the new optimal objective be (approximately)?
  5. Compute the reduced costs $\bar{c}_j$ for each reaction and verify complementary slackness.
Medium 4. Rank-Nullity Theorem for Metabolic Networks

This exercise connects the abstract rank-nullity theorem to concrete properties of metabolic networks.

  1. Prove from first principles: For any matrix $S \in \mathbb{R}^{m \times n}$, $\text{rank}(S) + \dim\text{Null}(S) = n$. Hint: Use the fact that if $\{\mathbf{v}_1, \ldots, \mathbf{v}_k\}$ is a basis for $\text{Null}(S)$, you can extend it to a basis for $\mathbb{R}^n$, and show that $S$ maps the remaining basis vectors to a linearly independent set of size $n - k$ in $\mathbb{R}^m$.
  2. Verify numerically for the E. coli core model (use COBRApy's textbook model): extract $S$, compute $\text{rank}(S)$ via SVD, compute $\dim\text{Null}(S)$ via SVD, and check that $\text{rank}(S) + \dim\text{Null}(S) = n$.
  3. Similarly verify for the left null space: $\text{rank}(S) + \dim\text{Null}(S^T) = m$. Report the number of conservation relations.
  4. Explain biologically why $\dim\text{Null}(S)$ is typically much larger than zero in genome-scale models. What does a large null space mean for the cell's metabolic flexibility?
Hard 5. Derive the Dual LP and Prove Strong Duality

This exercise requires a complete, rigorous derivation of LP duality for the FBA problem and a proof that strong duality holds.

  1. Standard form conversion: Rewrite the FBA problem $\max\{\mathbf{c}^T\mathbf{v} : S\mathbf{v} = \mathbf{0},\; \mathbf{v}_{lb} \leq \mathbf{v} \leq \mathbf{v}_{ub}\}$ in standard LP form: $\max\{\mathbf{c}^T\mathbf{x} : A\mathbf{x} = \mathbf{b},\; \mathbf{x} \geq \mathbf{0}\}$ by introducing slack variables and/or variable shifts $\mathbf{v}' = \mathbf{v} - \mathbf{v}_{lb}$.
  2. Derive the dual: Starting from the standard-form primal, derive the dual LP using the Lagrangian method. Show all steps: form the Lagrangian $L(\mathbf{x}, \boldsymbol{\lambda}, \boldsymbol{\mu})$, minimize over $\mathbf{x}$, and state the resulting dual problem.
  3. Weak duality: Prove that for any primal-feasible $\mathbf{v}$ and dual-feasible $(\mathbf{y}, \boldsymbol{\mu}^+, \boldsymbol{\mu}^-)$: $$\mathbf{c}^T\mathbf{v} \leq \mathbf{v}_{ub}^T\boldsymbol{\mu}^+ - \mathbf{v}_{lb}^T\boldsymbol{\mu}^-$$ Hint: Substitute the dual constraint into $\mathbf{c}^T\mathbf{v}$ and use the primal constraints and sign conditions.
  4. Strong duality: State the conditions under which strong duality holds (i.e., primal optimal = dual optimal with no gap). For FBA, argue that these conditions are satisfied when the flux bounds define a nonempty bounded polytope. Hint: Use the LP strong duality theorem (Slater's condition for LP, which requires only feasibility).
  5. Complementary slackness: Derive the complementary slackness conditions for FBA: $\mu_j^+(v_j^{ub} - v_j^*) = 0$ and $\mu_j^-(v_j^* - v_j^{lb}) = 0$ for all $j$. Explain why these conditions mean that $\bar{c}_j = 0$ whenever reaction $j$ is strictly between its bounds.
