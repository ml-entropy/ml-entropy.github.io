The Generative Model

A VAE defines a generative process:

$$\mathbf{z} \sim p(\mathbf{z}) = \mathcal{N}(\mathbf{0}, \mathbf{I})$$ $$\mathbf{x} \sim p_\theta(\mathbf{x}|\mathbf{z})$$

Sample a latent code $\mathbf{z}$ from a simple prior, then decode it into an observation $\mathbf{x}$ using a neural network.

The Goal

Learn parameters $\theta$ that maximize the likelihood of the data: $$\log p_\theta(\mathbf{x}) = \log \int p_\theta(\mathbf{x}|\mathbf{z}) p(\mathbf{z}) d\mathbf{z}$$ This integral is intractable—we can't compute it directly.

The Evidence Lower Bound (ELBO)

We can't maximize $\log p_\theta(\mathbf{x})$ directly, but we can maximize a lower bound. Introduce an approximate posterior $q_\phi(\mathbf{z}|\mathbf{x})$:

Deriving the ELBO
1
Start with: $\log p_\theta(\mathbf{x}) = \log \int p_\theta(\mathbf{x}, \mathbf{z}) d\mathbf{z}$
2
Multiply by 1: $= \log \int \frac{q_\phi(\mathbf{z}|\mathbf{x})}{q_\phi(\mathbf{z}|\mathbf{x})} p_\theta(\mathbf{x}, \mathbf{z}) d\mathbf{z}$
3
Jensen's inequality: $\geq \int q_\phi(\mathbf{z}|\mathbf{x}) \log \frac{p_\theta(\mathbf{x}, \mathbf{z})}{q_\phi(\mathbf{z}|\mathbf{x})} d\mathbf{z}$
4
Rearrange to get the ELBO: $$\mathcal{L}(\theta, \phi; \mathbf{x}) = \mathbb{E}_{q_\phi(\mathbf{z}|\mathbf{x})}[\log p_\theta(\mathbf{x}|\mathbf{z})] - D_{KL}(q_\phi(\mathbf{z}|\mathbf{x}) \| p(\mathbf{z}))$$
Interpretation

First term: Reconstruction quality—can the decoder reconstruct $\mathbf{x}$ from $\mathbf{z}$?
Second term: Regularization—keep the posterior close to the prior.

The Reparameterization Trick

We need gradients through $\mathbb{E}_{q_\phi}[\cdot]$, but sampling breaks backpropagation. The solution: reparameterize the sampling.

Let $q_\phi(\mathbf{z}|\mathbf{x}) = \mathcal{N}(\boldsymbol{\mu}_\phi(\mathbf{x}), \text{diag}(\boldsymbol{\sigma}_\phi^2(\mathbf{x})))$. Then:

$$\mathbf{z} = \boldsymbol{\mu}_\phi(\mathbf{x}) + \boldsymbol{\sigma}_\phi(\mathbf{x}) \odot \boldsymbol{\epsilon}, \quad \boldsymbol{\epsilon} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$$

Now $\mathbf{z}$ is a deterministic function of $\phi$ and $\boldsymbol{\epsilon}$. Gradients flow through $\boldsymbol{\mu}$ and $\boldsymbol{\sigma}$!

VAE Architecture

The VAE has two networks:

  • Encoder $q_\phi(\mathbf{z}|\mathbf{x})$: Maps input to latent distribution parameters $(\boldsymbol{\mu}, \boldsymbol{\sigma})$
  • Decoder $p_\theta(\mathbf{x}|\mathbf{z})$: Maps latent code to reconstruction
Training Algorithm
1
Encode: $\boldsymbol{\mu}, \boldsymbol{\sigma} = \text{Encoder}_\phi(\mathbf{x})$
2
Sample: $\boldsymbol{\epsilon} \sim \mathcal{N}(\mathbf{0}, \mathbf{I})$, $\mathbf{z} = \boldsymbol{\mu} + \boldsymbol{\sigma} \odot \boldsymbol{\epsilon}$
3
Decode: $\hat{\mathbf{x}} = \text{Decoder}_\theta(\mathbf{z})$
4
Loss: $\mathcal{L} = \|\mathbf{x} - \hat{\mathbf{x}}\|^2 + D_{KL}(q_\phi \| p)$

KL Divergence Closed Form

For Gaussian encoder and standard normal prior, the KL term has a closed form:

$$D_{KL}(\mathcal{N}(\boldsymbol{\mu}, \boldsymbol{\sigma}^2) \| \mathcal{N}(\mathbf{0}, \mathbf{I})) = \frac{1}{2}\sum_{j=1}^{J}\left(\sigma_j^2 + \mu_j^2 - 1 - \log\sigma_j^2\right)$$
Posterior Collapse

Sometimes the encoder learns to ignore the input and output the prior ($\boldsymbol{\mu} = \mathbf{0}$, $\boldsymbol{\sigma} = \mathbf{1}$). The decoder then ignores $\mathbf{z}$ entirely. Solutions: KL annealing, free bits, or stronger decoders.
