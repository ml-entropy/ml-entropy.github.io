11. Recurrent Neural Networks

RNNs process sequences by maintaining a hidden state that evolves over time. Let's understand their dynamics, limitations, and how LSTMs overcome them.

Theory Code Exercises

Why Sequences Need Special Treatment

Standard neural networks assume fixed-size inputs. But sequences (text, audio, time series) have variable length, and order matters. RNNs handle this through recurrence—feeding outputs back as inputs.

The Key Idea

Maintain a hidden state $\mathbf{h}_t$ that summarizes all information seen up to time $t$. Update it at each step based on the new input.

Vanilla RNN

The basic RNN equations:

$$\mathbf{h}_t = \tanh(\mathbf{W}_{hh}\mathbf{h}_{t-1} + \mathbf{W}_{xh}\mathbf{x}_t + \mathbf{b}_h)$$ $$\mathbf{y}_t = \mathbf{W}_{hy}\mathbf{h}_t + \mathbf{b}_y$$

The same weights $\mathbf{W}_{hh}$, $\mathbf{W}_{xh}$, $\mathbf{W}_{hy}$ are used at every time step—this is parameter sharing across time.

Unrolling

An RNN can be "unrolled" into a deep feedforward network where each layer corresponds to a time step. This reveals why long sequences are hard— gradients must flow through many layers.

Backpropagation Through Time (BPTT)

The total loss is the sum over time steps: $L = \sum_t L_t$. The gradient with respect to parameters must account for all paths through time:

Gradient Flow Through Time
1
$\frac{\partial L}{\partial \mathbf{W}_{hh}} = \sum_t \frac{\partial L_t}{\partial \mathbf{W}_{hh}}$
2
Each $\frac{\partial L_t}{\partial \mathbf{W}_{hh}}$ involves: $\frac{\partial L_t}{\partial \mathbf{h}_t} \frac{\partial \mathbf{h}_t}{\partial \mathbf{h}_{t-1}} \cdots \frac{\partial \mathbf{h}_1}{\partial \mathbf{W}_{hh}}$
3
The key term: $\frac{\partial \mathbf{h}_t}{\partial \mathbf{h}_{t-1}} = \mathbf{W}_{hh}^T \text{diag}(\tanh'(\cdot))$

The Vanishing Gradient Problem

Gradients flowing back through time get multiplied by $\mathbf{W}_{hh}$ repeatedly:

$$\frac{\partial \mathbf{h}_T}{\partial \mathbf{h}_1} = \prod_{t=2}^{T} \frac{\partial \mathbf{h}_t}{\partial \mathbf{h}_{t-1}}$$
The Problem

If $\|\mathbf{W}_{hh}\| < 1$: gradients vanish exponentially
If $\|\mathbf{W}_{hh}\| > 1$: gradients explode exponentially
Either way, learning long-term dependencies is nearly impossible.

LSTM: Long Short-Term Memory

LSTMs solve vanishing gradients with gating mechanisms and a separate cell state $\mathbf{c}_t$ that can persist unchanged.

LSTM Equations
1
Forget gate: $\mathbf{f}_t = \sigma(\mathbf{W}_f[\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_f)$
2
Input gate: $\mathbf{i}_t = \sigma(\mathbf{W}_i[\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_i)$
3
Candidate: $\tilde{\mathbf{c}}_t = \tanh(\mathbf{W}_c[\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_c)$
4
Cell update: $\mathbf{c}_t = \mathbf{f}_t \odot \mathbf{c}_{t-1} + \mathbf{i}_t \odot \tilde{\mathbf{c}}_t$
5
Output gate: $\mathbf{o}_t = \sigma(\mathbf{W}_o[\mathbf{h}_{t-1}, \mathbf{x}_t] + \mathbf{b}_o)$
6
Hidden state: $\mathbf{h}_t = \mathbf{o}_t \odot \tanh(\mathbf{c}_t)$
Why LSTMs Work

The cell state has an additive update: $\mathbf{c}_t = \mathbf{f}_t \odot \mathbf{c}_{t-1} + \ldots$ When $\mathbf{f}_t \approx 1$, gradients flow directly through without multiplication. This is like a "gradient highway."
