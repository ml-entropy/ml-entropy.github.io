Random Variables

A random variable is a function that maps outcomes from a sample space to real numbers. It's how we mathematically describe uncertain quantities.

Definition: Random Variable

A random variable $X: \Omega \to \mathbb{R}$ assigns a numerical value to each outcome in the sample space $\Omega$. We write $P(X = x)$ for the probability that $X$ takes value $x$.

Random variables come in two flavors:

  • Discrete: Takes countable values (coin flips, dice rolls, word counts)
  • Continuous: Takes values in an interval (height, temperature, neural network weights)

The Rules of Probability

The Sum Rule (Marginalization)

To find the probability of $X$ alone, sum over all possible values of $Y$:

$$P(X = x) = \sum_{y} P(X = x, Y = y)$$

This is called marginalization—we "marginalize out" the variable $Y$.

The Product Rule (Chain Rule)

Joint probabilities decompose into conditionals:

$$P(X, Y) = P(X | Y) \cdot P(Y) = P(Y | X) \cdot P(X)$$
Chain Rule Extension

For multiple variables: $P(X_1, X_2, \ldots, X_n) = P(X_1) \prod_{i=2}^{n} P(X_i | X_1, \ldots, X_{i-1})$

Bayes' Theorem: The Heart of ML

From the product rule, we can derive the most important equation in machine learning:

Deriving Bayes' Theorem
1
Start with product rule: $P(A, B) = P(A|B)P(B) = P(B|A)P(A)$
2
Equate the two expressions: $P(A|B)P(B) = P(B|A)P(A)$
3
Solve for $P(A|B)$: $$\boxed{P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}}$$

In machine learning terms:

$$\underbrace{P(\theta | \mathcal{D})}_{\text{Posterior}} = \frac{\overbrace{P(\mathcal{D} | \theta)}^{\text{Likelihood}} \cdot \overbrace{P(\theta)}^{\text{Prior}}}{\underbrace{P(\mathcal{D})}_{\text{Evidence}}}$$
  • Prior $P(\theta)$: What we believe before seeing data
  • Likelihood $P(\mathcal{D}|\theta)$: How probable is the data given parameters
  • Posterior $P(\theta|\mathcal{D})$: Updated belief after seeing data
  • Evidence $P(\mathcal{D})$: Normalizing constant

Expectation and Variance

The expected value is the probability-weighted average:

$$\mathbb{E}[X] = \sum_{x} x \cdot P(X = x) \quad \text{or} \quad \mathbb{E}[X] = \int x \cdot p(x) \, dx$$

The variance measures spread around the mean:

$$\text{Var}(X) = \mathbb{E}[(X - \mathbb{E}[X])^2] = \mathbb{E}[X^2] - \mathbb{E}[X]^2$$
Linearity of Expectation

$\mathbb{E}[aX + bY] = a\mathbb{E}[X] + b\mathbb{E}[Y]$, always—even when $X$ and $Y$ are dependent. This is one of the most useful properties in probability.

Independence

Two events are independent if knowing one tells you nothing about the other:

$$P(A \cap B) = P(A) \cdot P(B) \quad \Leftrightarrow \quad P(A|B) = P(A)$$

Conditional independence is crucial for graphical models:

$$X \perp Y | Z \quad \Leftrightarrow \quad P(X, Y | Z) = P(X|Z) \cdot P(Y|Z)$$
Common Pitfall

Independence and conditional independence are different! $X \perp Y$ does NOT imply $X \perp Y | Z$, and vice versa. This subtlety is essential for understanding Bayesian networks.
