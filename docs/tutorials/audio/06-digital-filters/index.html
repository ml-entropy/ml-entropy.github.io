What is a Digital Filter?

A digital filter is a system that transforms one discrete-time signal into another, modifying its frequency content in a controlled way. Every audio equalizer, noise-reduction algorithm, reverb tail, and crossover network is, at its core, a digital filter.

In its most general form, a filter maps an input sequence $x[n]$ to an output sequence $y[n]$. The most important class — linear time-invariant (LTI) filters — is completely characterized by its impulse response $h[n]$, and governed by the general difference equation:

$$y[n] = \sum_{k=0}^{M} b_k\, x[n-k] \;-\; \sum_{k=1}^{N} a_k\, y[n-k]$$

The $b_k$ coefficients multiply past and present inputs (feedforward path); the $a_k$ coefficients multiply past outputs (feedback path). When all $a_k = 0$ we have an FIR filter; when at least one $a_k \neq 0$ we have an IIR filter.

LTI Systems & Convolution

For any LTI filter with impulse response $h[n]$, the output is the convolution of the input with $h[n]$:

$$y[n] = (x * h)[n] = \sum_{k=-\infty}^{\infty} x[k]\,h[n-k]$$

Linearity means superposition holds; time-invariance means a delayed input produces an equally delayed output. These two properties together make filter analysis tractable.

FIR Filters

A Finite Impulse Response (FIR) filter has no feedback: the output depends only on current and past inputs. With $M+1$ coefficients $b_0, b_1, \ldots, b_M$:

$$y[n] = \sum_{k=0}^{M} b_k\, x[n-k] = b_0 x[n] + b_1 x[n-1] + \cdots + b_M x[n-M]$$

The impulse response is simply $h[n] = b_n$ for $0 \le n \le M$, zero elsewhere — finite by definition. Key properties:

  • Always stable — no poles (other than at the origin), so bounded input always produces bounded output (BIBO stable).
  • Linear phase possible — symmetric coefficients ($b_k = b_{M-k}$) give exactly linear phase, meaning all frequencies are delayed by the same amount (pure group delay $M/2$ samples).
  • High order required — sharp cutoff filters may need hundreds or thousands of coefficients, making them computationally heavier than IIR.
FIR Design Methods

Window method: multiply the ideal (sinc) impulse response by a window (Hamming, Kaiser, Blackman) to control sidelobe levels.
Frequency sampling: specify the desired $H[k]$ at $N$ frequency points and IDFT.
Parks-McClellan (equiripple): optimal Chebyshev approximation via the Remez exchange algorithm — minimizes the maximum error in both passband and stopband.

IIR Filters

An Infinite Impulse Response (IIR) filter includes feedback terms. Its impulse response never strictly reaches zero — it decays (exponentially if stable) but theoretically continues forever. The general difference equation is:

$$y[n] = \sum_{k=0}^{M} b_k\, x[n-k] \;-\; \sum_{k=1}^{N} a_k\, y[n-k]$$

Advantages over FIR for many audio tasks:

  • Efficiency — a 2nd-order IIR can approximate a filter that would require a 100th-order FIR.
  • Analog prototype available — classic designs (Butterworth, Chebyshev I & II, elliptic, Bessel) from analog filter theory map directly via the bilinear transform.
IIR Stability Warning

An IIR filter is BIBO stable if and only if all poles of its transfer function lie strictly inside the unit circle in the z-plane. If any pole is on or outside the unit circle, the output will grow without bound. Always check poles when designing IIR filters, and use second-order sections (SOS) to reduce numerical issues.

Transfer Function & Z-Transform

The Z-transform converts a discrete-time sequence $x[n]$ into a function of the complex variable $z$:

$$X(z) = \sum_{n=-\infty}^{\infty} x[n]\, z^{-n}$$

For a delay of $k$ samples, $z^{-k}$ acts as the delay operator. Applying the Z-transform to the difference equation, the transfer function is the ratio of output to input in the z-domain:

$$H(z) = \frac{Y(z)}{X(z)} = \frac{B(z)}{A(z)} = \frac{\sum_{k=0}^{M} b_k z^{-k}}{1 + \sum_{k=1}^{N} a_k z^{-k}}$$

Factoring numerator and denominator reveals zeros (values of $z$ where $H(z) = 0$) and poles (values of $z$ where $H(z) \to \infty$):

$$H(z) = b_0 \frac{\prod_{k=1}^{M}(1 - z_k z^{-1})}{\prod_{k=1}^{N}(1 - p_k z^{-1})}$$

The frequency response is obtained by evaluating on the unit circle, $z = e^{j\omega}$:

$$H(e^{j\omega}) = H(z)\big|_{z=e^{j\omega}}$$

Geometrically: $|H(e^{j\omega})|$ is the product of distances from $e^{j\omega}$ to each zero divided by distances to each pole. A zero near the unit circle creates a notch; a pole near the unit circle creates a peak.

Common Filter Types

Filter specifications use several key parameters:

  • Cutoff frequency $\omega_c$ — the boundary between passband and stopband, usually defined at the −3 dB point.
  • Q factor — for resonant filters, $Q = f_0 / \text{BW}$ where BW is the −3 dB bandwidth. High Q = narrow, resonant peak.
  • Rolloff slope — how steeply the filter attenuates outside the passband, expressed in dB/octave or dB/decade.
  • Ripple — allowed variation in the passband (Chebyshev filters trade ripple for steeper rolloff).

Biquad Filter

The biquad (bi-quadratic) is a second-order IIR section — the most important building block in audio DSP. Its transfer function has a quadratic numerator and denominator:

$$H(z) = \frac{b_0 + b_1 z^{-1} + b_2 z^{-2}}{1 + a_1 z^{-1} + a_2 z^{-2}}$$

The corresponding difference equation is:

$$y[n] = b_0 x[n] + b_1 x[n-1] + b_2 x[n-2] \;-\; a_1 y[n-1] \;-\; a_2 y[n-2]$$

Five coefficients $(b_0, b_1, b_2, a_1, a_2)$ fully characterize the filter. Robert Bristow-Johnson's Audio EQ Cookbook provides closed-form formulas for all common EQ types from three audio parameters: sample rate $f_s$, center/cutoff frequency $f_0$, and Q (or bandwidth BW, or shelf slope S).

Biquad Cookbook: Lowpass Filter

Given $f_0$ (cutoff) and $Q$, let $\omega_0 = 2\pi f_0/f_s$ and $\alpha = \sin(\omega_0)/(2Q)$:

$$b_0 = \frac{1 - \cos\omega_0}{2}, \quad b_1 = 1 - \cos\omega_0, \quad b_2 = \frac{1-\cos\omega_0}{2}$$ $$a_0 = 1 + \alpha, \quad a_1 = -2\cos\omega_0, \quad a_2 = 1 - \alpha$$

Normalize all coefficients by $a_0$. The same parameterization covers highpass, bandpass, notch, peaking EQ, low shelf, and high shelf.

For higher-order filters, cascade multiple biquad sections. An $N$th-order filter becomes $\lceil N/2 \rceil$ biquad sections in series. This second-order sections (SOS) form is numerically more stable than a single high-order difference equation, because coefficient quantization errors in each biquad only affect that section's two poles, not all poles simultaneously.

Linear Phase

A filter has linear phase when its phase response is a linear function of frequency:

$$\angle H(e^{j\omega}) = -\tau \omega$$

This means every frequency component is delayed by exactly $\tau$ samples — the filter preserves the shape of transients and the relative timing of spectral components. The group delay $\tau_g(\omega) = -d\angle H/d\omega$ is constant for all $\omega$.

An FIR filter has exactly linear phase if and only if its impulse response is symmetric (Type I/II) or antisymmetric (Type III/IV):

$$h[n] = h[M - n] \quad \text{(symmetric, linear phase)}$$

IIR filters cannot have linear phase. Their recursive structure inherently introduces frequency-dependent phase distortion (non-constant group delay).

When Linear Phase Matters

Mastering & crossovers: Phase misalignment between low and high frequency bands causes comb filtering and transient smearing — linear phase FIR crossovers avoid this.
Image processing & speech: Phase coherence across frequency is critical for preserving click/transient shapes.
Most real-time EQ: Small IIR phase non-linearity is inaudible for modest EQ corrections — IIR is preferred for its efficiency.

Bilinear Transform & Digital Filter Design

The standard method for designing IIR filters starts from a known analog prototype $H_a(s)$ (Butterworth, Chebyshev, etc.) and maps it to a digital filter $H(z)$ using the bilinear transform:

$$s = \frac{2}{T_s} \cdot \frac{1 - z^{-1}}{1 + z^{-1}}$$

This substitution maps the entire left half of the $s$-plane (stable analog region) to the interior of the unit circle in the $z$-plane — guaranteeing stability is preserved. The frequency axis $j\Omega$ maps to the unit circle $e^{j\omega}$ via the warping relation:

$$\Omega = \frac{2}{T_s}\tan\!\left(\frac{\omega}{2}\right)$$

Because of this frequency warping, the cutoff frequency must be pre-warped before designing the analog prototype:

$$\Omega_c = \frac{2}{T_s}\tan\!\left(\frac{\omega_c}{2}\right)$$

The bilinear transform introduces no aliasing (unlike impulse invariance), making it the preferred method for lowpass, highpass, bandpass, and bandstop digital filter design.

Exercises: Digital Filters

Easy (1–10)

1Identify Filter TypeEasy

A filter passes frequencies below 1 kHz and attenuates frequencies above 1 kHz. What type of filter is it? What is the cutoff frequency?

This is a lowpass filter with cutoff frequency $f_c = 1$ kHz. It passes low frequencies and attenuates high frequencies, which is the defining characteristic of a lowpass filter.

2FIR or IIR?Easy

A filter is described by the equation $y[n] = 0.5x[n] + 0.3x[n-1] + 0.2x[n-2]$. Is this an FIR or IIR filter? How many taps does it have?

This is an FIR filter — there are no feedback terms (no $y[n-k]$ terms on the right side). It has 3 taps (coefficients $b_0 = 0.5$, $b_1 = 0.3$, $b_2 = 0.2$), so it is a 2nd-order (length-3) FIR filter.

3Simple ConvolutionEasy

An FIR filter has impulse response $h = [1, 2, 1]$ (values at $n=0,1,2$). Apply it to the input $x = [3, 1, 0, 0, \ldots]$ (i.e., $x[0]=3, x[1]=1$, rest zero). Compute $y[0], y[1], y[2], y[3]$.

$y[n] = \sum_{k=0}^{2} h[k]\,x[n-k]$

$y[0] = h[0]x[0] = 1\cdot3 = 3$

$y[1] = h[0]x[1] + h[1]x[0] = 1\cdot1 + 2\cdot3 = 7$

$y[2] = h[0]x[2] + h[1]x[1] + h[2]x[0] = 0 + 2\cdot1 + 1\cdot3 = 5$

$y[3] = h[1]x[2] + h[2]x[1] = 0 + 1\cdot1 = 1$

Output: $y = [3, 7, 5, 1, 0, \ldots]$

4Cutoff FrequencyEasy

A lowpass filter has its −3 dB point at normalized frequency $\omega_c = \pi/4$ rad/sample. If the sample rate is 44,100 Hz, what is the cutoff frequency in Hz?

Normalized frequency $\omega = 2\pi f / f_s$, so $f = \omega f_s / (2\pi)$.

$f_c = \frac{\pi/4}{2\pi} \times 44100 = \frac{1}{8} \times 44100 = 5512.5$ Hz.

5Filter Type from Pole-Zero PlotEasy

A filter has zeros at $z = +1$ and $z = -1$ (i.e., at DC and Nyquist), and two poles at $z = \pm j\cdot 0.9$. What type of filter would you expect this to be?

The zeros at $z=+1$ (DC, $\omega=0$) and $z=-1$ (Nyquist, $\omega=\pi$) null out both DC and the highest frequency. The poles at $z = \pm j0.9$ are close to the unit circle at $\omega = \pm\pi/2$ (quarter-Nyquist, i.e., $f_s/4$), creating gain there. This is a bandpass filter centered at $f_s/4$.

6Stability CheckEasy

A first-order IIR filter is $y[n] = x[n] + a\,y[n-1]$. For what values of $a$ is this filter stable?

The transfer function is $H(z) = 1/(1 - az^{-1}) = z/(z - a)$. The single pole is at $z = a$. For BIBO stability, all poles must be strictly inside the unit circle: $|a| < 1$.

So the filter is stable when $-1 < a < 1$.

7Impulse Response LengthEasy

An FIR filter has 128 coefficients. What is the length of its impulse response? If you convolve this filter with an input of 512 samples, how many output samples result?

Impulse response length = 128 samples (the coefficients are the impulse response).

Convolution output length = $N + M - 1 = 512 + 128 - 1 = 639$ samples (linear convolution).

8Notch Filter ZerosEasy

To create a notch filter that completely removes 60 Hz hum at sample rate 48,000 Hz, where should you place the zeros in the z-plane?

Place zeros exactly on the unit circle at the frequency corresponding to 60 Hz:

$\omega_0 = 2\pi \times 60 / 48000 = 2\pi/800 \approx 0.00785\pi$ rad/sample.

Zeros at $z = e^{\pm j\omega_0} = \cos\omega_0 \pm j\sin\omega_0 \approx 0.99997 \pm j0.00785$.

For real coefficients, zeros must come in conjugate pairs. Placing zeros exactly on the unit circle guarantees $H(e^{j\omega_0}) = 0$ — complete nulling at 60 Hz.

9Biquad State VariablesEasy

How many multiply-accumulate (MAC) operations does a single biquad section require per output sample? How many state variables does it maintain?

From $y[n] = b_0 x[n] + b_1 x[n-1] + b_2 x[n-2] - a_1 y[n-1] - a_2 y[n-2]$:

5 multiplications + 4 additions = 5 MACs per output sample (one multiply per coefficient; $b_0$ can sometimes be combined).

4 state variables: $x[n-1]$, $x[n-2]$, $y[n-1]$, $y[n-2]$. (In the transposed direct form II, only 2 state variables are needed, which is preferred for numerical accuracy.)

10Filter Cascade OrderEasy

You want an 8th-order Butterworth lowpass filter implemented as cascaded biquad sections. How many biquad sections do you need?

Each biquad is a 2nd-order section. For an 8th-order filter: $\lceil 8/2 \rceil = 4$ biquad sections.

The overall transfer function is $H(z) = H_1(z) \cdot H_2(z) \cdot H_3(z) \cdot H_4(z)$, with each $H_k(z)$ implementing one conjugate pole pair of the Butterworth prototype.

Medium (11–20)

11FIR Output ComputationMedium

A 3-tap FIR filter has coefficients $b = [0.25, 0.5, 0.25]$ (a simple lowpass). Compute the output for input $x[n] = \cos(\pi n / 2)$ (a sinusoid at $\omega = \pi/2$). What does the output tell you about the filter's gain at this frequency?

The frequency response at $\omega = \pi/2$:

$H(e^{j\pi/2}) = b_0 + b_1 e^{-j\pi/2} + b_2 e^{-j\pi} = 0.25 + 0.5(-j) + 0.25(-1) = 0.25 - 0.5j - 0.25 = -0.5j$

$|H(e^{j\pi/2})| = 0.5$

The output is $y[n] = 0.5\cos(\pi n/2 - \pi/2) = 0.5\sin(\pi n/2)$ — the sinusoid at $\omega=\pi/2$ is attenuated by half (−6 dB), confirming the filter rolls off significantly at this frequency (it's a lowpass).

12Z-Transform of FIRMedium

Find the transfer function $H(z)$ and zeros of the 3-tap averaging filter $h = [1/3, 1/3, 1/3]$. Where are the zeros?

$H(z) = \frac{1}{3}(1 + z^{-1} + z^{-2}) = \frac{1}{3}\cdot\frac{z^2 + z + 1}{z^2}$

Zeros: solve $z^2 + z + 1 = 0 \Rightarrow z = \frac{-1 \pm \sqrt{-3}}{2} = e^{\pm j2\pi/3}$.

These are the complex cube roots of unity (excluding $z=1$). On the unit circle at angles $\pm 120°$, i.e., at $\omega = \pm 2\pi/3$. These correspond to $f = f_s/3$ — so the filter completely nulls the frequency $f_s/3$ and its mirror. There are also two poles at $z = 0$ (trivial, don't affect frequency response magnitude).

13First-Order IIR as Exponential SmootherMedium

The filter $y[n] = (1-\alpha)x[n] + \alpha\, y[n-1]$ with $0 < \alpha < 1$ is an exponential smoother. (a) Find its transfer function. (b) At what normalized frequency is the gain $1/\sqrt{2}$ (the −3 dB point)? Express in terms of $\alpha$.

(a) $H(z) = \frac{(1-\alpha)}{1 - \alpha z^{-1}}$. Single pole at $z = \alpha$.

(b) The frequency response: $H(e^{j\omega}) = \frac{1-\alpha}{1 - \alpha e^{-j\omega}}$.

$|H(e^{j\omega})|^2 = \frac{(1-\alpha)^2}{1 - 2\alpha\cos\omega + \alpha^2}$

At DC ($\omega=0$): $|H| = (1-\alpha)/(1-\alpha) = 1$. Set $|H|^2 = 1/2$:

$(1-\alpha)^2 = \frac{1}{2}(1 - 2\alpha\cos\omega + \alpha^2)$

$\cos\omega = \frac{(1-\alpha)^2/2 - (1+\alpha^2)/2 + 1}{-\alpha} \cdot \frac{1}{-1}$. Simplified: $\cos\omega_c = \frac{2\alpha^2 - 4\alpha + 2}{2\alpha} = \frac{\alpha^2 - 2\alpha + 1}{\alpha} \cdot \frac{1}{2}$... Exact: $\omega_c = \cos^{-1}\!\left(\frac{2\alpha^2 - 4\alpha + 2}{2\alpha}\right)$. For $\alpha \approx 1$, $\omega_c \approx 2(1-\alpha)$ (low cutoff = smooth filter).

14Biquad Lowpass CoefficientsMedium

Using the Audio EQ Cookbook formulas, compute the biquad coefficients for a lowpass filter with $f_0 = 1000$ Hz, $Q = 0.707$ (Butterworth), $f_s = 44100$ Hz. Give the normalized coefficients.

$\omega_0 = 2\pi \times 1000/44100 = 0.14247$ rad

$\alpha = \sin(\omega_0)/(2Q) = \sin(0.14247)/(2 \times 0.707) = 0.14199/1.414 = 0.10042$

$\cos\omega_0 = \cos(0.14247) = 0.98985$

Raw coefficients:

$b_0 = (1-\cos\omega_0)/2 = 0.00508$, $b_1 = 1-\cos\omega_0 = 0.01015$, $b_2 = 0.00508$

$a_0 = 1+\alpha = 1.10042$, $a_1 = -2\cos\omega_0 = -1.97970$, $a_2 = 1-\alpha = 0.89958$

Divide all by $a_0$: $b_0 = 0.00461$, $b_1 = 0.00922$, $b_2 = 0.00461$, $a_1 = -1.79902$, $a_2 = 0.81757$.

15Linear Phase VerificationMedium

Show that the FIR filter with coefficients $h = [1, 2, 3, 2, 1]$ has linear phase by computing $H(e^{j\omega})$ and showing the phase is $-2\omega$.

$H(z) = 1 + 2z^{-1} + 3z^{-2} + 2z^{-3} + z^{-4}$

Factor out $z^{-2}$ (the center of symmetry):

$H(e^{j\omega}) = e^{-j2\omega}(e^{j2\omega} + 2e^{j\omega} + 3 + 2e^{-j\omega} + e^{-j2\omega})$

$= e^{-j2\omega}(2\cos 2\omega + 4\cos\omega + 3)$

The term in parentheses is real-valued (a real trigonometric polynomial). Therefore:

$\angle H(e^{j\omega}) = -2\omega + \begin{cases}0 & \text{if } (2\cos2\omega+4\cos\omega+3)>0 \\ \pi & \text{otherwise}\end{cases}$

In the passband the real term is positive, so $\angle H = -2\omega$ — exactly linear phase with group delay $\tau = 2$ samples (half the filter length $M=4$).

16Bilinear Transform Frequency WarpingMedium

You want a digital highpass filter with cutoff at $f_c = 8000$ Hz and $f_s = 44100$ Hz. What analog prototype cutoff frequency $\Omega_c$ should you use (in rad/s) when designing via the bilinear transform?

$\omega_c = 2\pi f_c / f_s = 2\pi \times 8000/44100 = 1.13939$ rad/sample.

Pre-warped analog cutoff:

$\Omega_c = \frac{2}{T_s}\tan\!\left(\frac{\omega_c}{2}\right) = 2f_s\tan(\omega_c/2) = 2 \times 44100 \times \tan(0.56970)$

$= 88200 \times 0.63227 = 55{,}766$ rad/s

$\approx 8877$ Hz (the analog prototype must be designed for this slightly higher frequency to compensate for warping).

17Allpass Filter PhaseMedium

The first-order allpass filter $H(z) = \frac{z^{-1} - a}{1 - az^{-1}}$ has $|H(e^{j\omega})| = 1$ for all $\omega$. Verify this property and find the phase response $\angle H(e^{j\omega})$.

$H(e^{j\omega}) = \frac{e^{-j\omega} - a}{1 - ae^{-j\omega}}$

Note the numerator is the complex conjugate of the denominator: $e^{-j\omega} - a = e^{-j\omega}(1 - ae^{j\omega}) = e^{-j\omega}\overline{(1 - ae^{-j\omega})}$ (for real $a$).

Therefore $|H(e^{j\omega})| = |e^{-j\omega}| \cdot \left|\frac{\overline{(1-ae^{-j\omega})}}{1-ae^{-j\omega}}\right| = 1 \times 1 = 1$. ✓

Phase: $\angle H(e^{j\omega}) = -\omega + 2\angle(e^{-j\omega} - a) \cdot (-1) \ldots$ More directly: $\angle H = \angle(e^{-j\omega}-a) - \angle(1-ae^{-j\omega}) = -\omega - 2\arctan\!\left(\frac{a\sin\omega}{1-a\cos\omega}\right)$.

18Filter Order for Stopband AttenuationMedium

A Butterworth lowpass filter needs to attenuate frequencies above $\omega_s$ by at least 60 dB, while having no more than 3 dB ripple below $\omega_c$, with $\omega_s/\omega_c = 2$. What minimum filter order is required?

Butterworth order formula: $n \geq \frac{\log\left(\frac{10^{A_s/10}-1}{10^{A_p/10}-1}\right)}{2\log(\omega_s/\omega_c)}$

$A_s = 60$ dB, $A_p = 3$ dB, $\omega_s/\omega_c = 2$.

Numerator: $\log\!\left(\frac{10^6-1}{10^{0.3}-1}\right) = \log\!\left(\frac{999999}{0.9953}\right) \approx \log(1004511) \approx 6.002$

Denominator: $2\log(2) = 0.6021$

$n \geq 6.002/0.6021 \approx 9.97 \Rightarrow n = 10$.

A 10th-order Butterworth filter (5 cascaded biquads) meets the spec.

You need a lowpass filter with 80 dB stopband attenuation at $\omega_s = 1.5\omega_c$. Hard (21–30)

21Stability via Jury TestHard

A second-order IIR has denominator $A(z) = 1 + a_1 z^{-1} + a_2 z^{-2}$. Derive the Jury stability conditions — the necessary and sufficient conditions on $a_1$ and $a_2$ for both poles to lie inside the unit circle.

Poles are roots of $z^2 + a_1 z + a_2 = 0$. We need $|p_{1,2}| < 1$. Apply the Jury stability test to the polynomial $z^2 + a_1 z + a_2$.

The three necessary and sufficient conditions are:

1. $A(1) > 0$: $1 + a_1 + a_2 > 0$

2. $A(-1) > 0$: $1 - a_1 + a_2 > 0$

3. $|a_2| < 1$: the constant term satisfies $-1 < a_2 < 1$

These define a triangle in the $(a_1, a_2)$ plane (the "stability triangle"). Geometrically: both poles inside unit circle $\Leftrightarrow$ the point $(a_1, a_2)$ is inside the triangle with vertices $(2, 1)$, $(-2, 1)$, $(0, -1)$.

Proof sketch: Vieta's formulas give $p_1 p_2 = a_2$ and $p_1+p_2 = -a_1$. Condition 3 ensures $|p_1 p_2|<1$; conditions 1 and 2 ensure the real parts and product together keep both poles inside.

22Prove Linear Phase from SymmetryHard

Prove rigorously that a causal $M$-th order FIR with symmetric impulse response $h[n] = h[M-n]$ has exactly linear phase $\angle H(e^{j\omega}) = -M\omega/2$.

$H(e^{j\omega}) = \sum_{n=0}^{M} h[n] e^{-j\omega n}$. Factor out $e^{-j\omega M/2}$:

$H(e^{j\omega}) = e^{-j\omega M/2} \sum_{n=0}^{M} h[n] e^{-j\omega(n - M/2)}$

Let $k = n - M/2$, so $k$ ranges from $-M/2$ to $M/2$:

$H(e^{j\omega}) = e^{-j\omega M/2} \sum_{k=-M/2}^{M/2} h[k+M/2] e^{-j\omega k}$

Using symmetry $h[M-n] = h[n]$, i.e., $h[M/2-k] = h[M/2+k]$, so $h[k+M/2]$ is an even function of $k$. Therefore the sum $\tilde{H}(\omega) = \sum_k h[k+M/2]e^{-j\omega k}$ is a real-valued function of $\omega$ (imaginary parts cancel). Hence:

$H(e^{j\omega}) = e^{-j\omega M/2} \tilde{H}(\omega)$ where $\tilde{H}(\omega) \in \mathbb{R}$.

Therefore $\angle H(e^{j\omega}) = -\omega M/2 + \angle\tilde{H}(\omega)$ where $\angle\tilde{H} \in \{0, \pi\}$. In the passband $\tilde{H} > 0$, so $\angle H = -\omega M/2$ — exactly linear. $\blacksquare$

23IIR Cannot Have Linear PhaseHard

Prove that a causal, stable IIR filter cannot have exactly linear phase (except for the trivial delay).

Suppose $H(z)$ is a causal stable IIR with linear phase $\angle H(e^{j\omega}) = -\tau\omega$ for integer $\tau$. Then $H(e^{j\omega}) = |H(e^{j\omega})| e^{-j\omega\tau}$, so $H(z) = z^{-\tau} G(z)$ where $G(z)$ has real non-negative frequency response $|H(e^{j\omega})|$.

A stable causal system with real non-negative frequency response is a non-negative real function — this means $G(z)$ is minimum phase. But $H(z) = z^{-\tau} G(z)$; the only way $H(z)$ is also causal and stable is if $G(z)$ has no poles (i.e., $G$ is FIR, a polynomial in $z^{-1}$). Any IIR $H(z)$ has at least one pole at $z = p \neq 0$, which would require $G(z) = H(z)/z^{-\tau} = z^\tau H(z)$ to have that same pole — but then $G$ would not be a finite polynomial.

Therefore, no causal stable IIR with non-trivial recursive structure can have exactly linear phase. $\blacksquare$

24Design a Notch FilterHard

Design a biquad notch filter to remove 50 Hz at $f_s = 44100$ Hz with a bandwidth of 5 Hz (−3 dB width). Give exact coefficients and verify the notch depth is theoretically infinite.

Notch frequency: $\omega_0 = 2\pi\times50/44100 = 0.007117$ rad/sample.

Q for bandwidth BW = 5 Hz: $Q = f_0/\text{BW} = 50/5 = 10$.

$\alpha = \sin\omega_0/(2Q) = 0.007116/(20) = 0.0003558$, $\cos\omega_0 = 0.999975$.

Notch biquad (cookbook): $b_0 = 1$, $b_1 = -2\cos\omega_0 = -1.999949$, $b_2 = 1$, $a_0 = 1+\alpha = 1.0003558$, $a_1 = -2\cos\omega_0 = -1.999949$, $a_2 = 1-\alpha = 0.9996442$.

Normalize: $b_0=0.999644$, $b_1=-1.999238$, $b_2=0.999644$, $a_1=-1.999238$, $a_2=0.999289$.

The numerator is $b_0+b_1 z^{-1}+b_2 z^{-2}$. At $z=e^{j\omega_0}$: numerator $= b_0+b_1 e^{-j\omega_0}+b_2 e^{-2j\omega_0}$. With the original (pre-normalization) $b_0=b_2=1$, $b_1=-2\cos\omega_0$: this equals $e^{-j\omega_0}(e^{j\omega_0}-2\cos\omega_0+e^{-j\omega_0})=e^{-j\omega_0}(2\cos\omega_0-2\cos\omega_0)=0$. The zeros are exactly on the unit circle at $\pm\omega_0$ — infinite notch depth. $\blacksquare$

26Coefficient Quantization EffectHard

A biquad has poles at $p = 0.99 e^{\pm j\pi/4}$. The coefficient $a_2 = |p|^2 = 0.9801$ must be stored as a 16-bit fixed-point number with range $[-2, 2)$ (1 sign bit, 1 integer bit, 14 fractional bits). What is the quantization step for $a_2$, and how much does this shift the pole radius? Could this destabilize the filter?

Step size: $\Delta = 2^{1-14} \cdot 2 / 2^{15} \cdot 2^{14}$... More directly: range $[-2,2)$ over $2^{16}$ levels: $\Delta = 4/65536 = 6.1 \times 10^{-5}$.

True $a_2 = 0.9801$. Quantized: $\hat{a}_2 = \text{round}(0.9801/\Delta)\times\Delta$. $0.9801/6.1\times10^{-5} = 16067.2 \to 16067 \times 6.1\times10^{-5} = 0.98009$. Error $\approx 10^{-5}$.

Pole radius: $|p| = \sqrt{a_2}$. $\delta|p| \approx \delta a_2/(2|p|) = 10^{-5}/1.98 \approx 5\times10^{-6}$. Negligible — pole stays at radius $\approx0.99$, well inside unit circle.

However, near $a_2 \to 1$ (very narrow resonance), quantization can push poles outside the unit circle. For $a_2 = 0.9999$, only 1 quantization step separates stable from unstable. This is why high-order direct-form IIR filters should be implemented as SOS, and poles should not be placed very close to the unit circle in fixed-point systems.

27Lattice Filter CoefficientsHard

The lattice (PARCOR) form of an FIR filter is parameterized by reflection coefficients $\{k_m\}$. For a 2nd-order FIR with $h = [1, a_1, a_2]$, derive the reflection coefficients $k_1$ and $k_2$ from the Levinson-Durbin recursion.

Levinson-Durbin maps polynomial coefficients to reflection (PARCOR) coefficients recursively. For the polynomial $A_2(z) = 1 + a_1 z^{-1} + a_2 z^{-2}$:

Step 1: $k_2 = a_2^{(2)} = a_2$ (the last coefficient of the order-2 polynomial).

Step 2: Order-1 polynomial by back-step recursion:

$a_1^{(1)} = \frac{a_1 - k_2 a_1^{*}}{1 - |k_2|^2} = \frac{a_1 - a_2 a_1}{1 - a_2^2} = \frac{a_1(1-a_2)}{1-a_2^2} = \frac{a_1}{1+a_2}$

$k_1 = a_1^{(1)} = a_1/(1+a_2)$.

Stability requires $|k_1| < 1$ and $|k_2| < 1$. Lattice form is preferred for spectral analysis (LPC) because each $k_m$ has direct physical meaning as a reflection coefficient and stability reduces to $|k_m| < 1$ at each stage.

28Halfband Filter PropertyHard

A halfband FIR lowpass filter satisfies $H(e^{j\omega}) + H(e^{j(\pi-\omega)}) = 1$ for all $\omega$. Show that this implies every other coefficient (except the center) must be zero: $h[n] = 0$ for even $n \neq M/2$ (where $M$ is the filter order).

The complementary condition $H(\omega) + H(\pi - \omega) = 1$ means $H(\omega) - 1/2 = -(H(\pi-\omega) - 1/2)$, so $G(\omega) = H(\omega) - 1/2$ is an odd function about $\omega = \pi/2$.

The impulse response of $G$ is $g[n] = h[n] - \frac{1}{2}\delta[n - M/2]$ (the $1/2$ shift in the time domain is a $1/2$ delta at the center).

$G(e^{j\omega})$ being odd about $\pi/2$ means $G(e^{j\omega}) = -G(e^{j(\pi-\omega)})$. In the time domain, multiplication by $e^{j\pi n} = (-1)^n$ corresponds to the shift $\omega \to \omega - \pi$. So $G(e^{j(\pi-\omega)}) = \mathcal{F}\{(-1)^n g[n]\}(-\omega)$...

More directly: the power-complementary condition forces $g[n](-1)^n = -g[n]$, i.e., $g[n](1+(-1)^n) = 0$, so $g[n] = 0$ for even $n$. Therefore $h[n] = 0$ for even $n \neq M/2$, and $h[M/2] = 1/2$. This halves the number of non-zero taps — halfband filters cost only half as many MACs as a general FIR of the same length. $\blacksquare$

29Minimum Phase vs. Linear Phase DelayHard

A linear phase FIR lowpass of order $M = 100$ introduces a delay of 50 samples. A minimum-phase IIR of similar frequency response introduces much less group delay on average. Given the group delay of a 5th-order Butterworth IIR is approximately $\tau_0 \approx N/(2\omega_c)$ samples at DC, compare the two delays for $\omega_c = \pi/4$. When would you prefer each?

FIR delay: constant 50 samples for all frequencies.

IIR Butterworth 5th-order at DC: $\tau_0 \approx N/(2\omega_c) = 5/(2\times\pi/4) = 5\times4/(2\pi) = 10/\pi \approx 3.18$ samples.

The IIR has about 16× less delay at DC, though its group delay varies with frequency (peaks near $\omega_c$, approaches $\sim 2.5/\omega_c$ there).

Prefer linear phase FIR when: Phase coherence across frequency matters (crossovers, mastering, transient-critical audio); offline processing where latency is acceptable; strictly symmetric impulse response is required.

Prefer IIR when: Low latency is critical (real-time effects, hearing aids, control systems); computational resources are limited; the IIR's phase nonlinearity is inaudible for the application (most broadband EQ).

30Oversampling FIR Anti-Aliasing FilterHard

A sigma-delta DAC oversamples at $4\times$ (e.g., 176.4 kHz for a 44.1 kHz signal). The digital interpolation filter must pass $[0, 22.05$ kHz] and attenuate $[22.05, 154.35$ kHz] by 120 dB. Using the Kaiser formula, estimate the FIR order needed. Then explain why a polyphase implementation reduces the actual computational cost by $4\times$.

At 176.4 kHz, the transition band in normalized frequency: passband edge = $22050/176400 = 0.125$, stopband edge = $22050/176400$ (images start at $f_s - f_{pass} = 176400 - 22050$, so in normalized terms the stopband starts just above 0.125)... Actually the transition band is the gap: passband $[0, 0.125]$, stopband $[0.125, 1-0.125] = [0.125, 0.875]$. In normalized $[0,1]$ units: $\Delta f = $ very small if we want sharp cutoff right at 0.125. Transition bandwidth $\Delta\omega = 2\pi\times(22050/176400) = \pi/4$ (very wide — 4× oversampling gives lots of room!).

Kaiser: $N = (120 - 7.95)/(2.285 \times \pi/4 \times 1/\pi) \approx 112.05/(2.285\times0.25) \approx 112/0.571 \approx 196$ taps.

Polyphase decomposition: When upsampling by $L=4$ and filtering, 3 out of every 4 input samples are zero. Rearranging the FIR into $L=4$ sub-filters (polyphase components), each of length $\approx 196/4 = 49$ taps, processes only the non-zero input samples. Computational cost: $196/4 = 49$ MACs/input-sample instead of $196 \times 4 = 784$ MACs/output-sample (since output rate is $4\times$ higher). Net savings: exactly $4\times$ — matching the oversampling ratio.
