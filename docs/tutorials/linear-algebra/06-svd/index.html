The SVD Theorem

Every matrix $A$ (any size, any rank) can be decomposed as:

Singular Value Decomposition

$$A = U\Sigma V^T$$ where:

  • $U$ is $m \times m$ orthogonal (left singular vectors)
  • $\Sigma$ is $m \times n$ diagonal (singular values $\sigma_1 \geq \sigma_2 \geq \cdots \geq 0$)
  • $V$ is $n \times n$ orthogonal (right singular vectors)

Geometric Interpretation

Any linear transformation can be broken into three steps:

$A\mathbf{x} = U\Sigma V^T\mathbf{x}$
1
$V^T\mathbf{x}$: Rotate in the input space (to align with singular directions)
2
$\Sigma(\cdot)$: Scale each coordinate by the singular value
3
$U(\cdot)$: Rotate in the output space
Key Insight

The columns of $V$ are orthogonal directions in the input that get mapped to orthogonal directions (columns of $U$) in the output. The singular values tell you the stretching factor along each direction.

Computing the SVD

The singular values and vectors come from eigendecomposition:

  • Right singular vectors: Eigenvectors of $A^T A$
  • Left singular vectors: Eigenvectors of $AA^T$
  • Singular values: $\sigma_i = \sqrt{\lambda_i}$ where $\lambda_i$ are eigenvalues of $A^TA$
$$A^TA = V\Sigma^T\Sigma V^T, \quad AA^T = U\Sigma\Sigma^T U^T$$

Low-Rank Approximation

The SVD reveals the "importance" of each component via singular values. We can approximate $A$ using only the top $k$ components:

$$A_k = \sum_{i=1}^{k} \sigma_i \mathbf{u}_i \mathbf{v}_i^T$$
Eckart-Young Theorem

$A_k$ is the best rank-$k$ approximation to $A$ in both Frobenius and spectral norms. No other rank-$k$ matrix is closer!

The approximation error is controlled by the discarded singular values:

$$\|A - A_k\|_F = \sqrt{\sigma_{k+1}^2 + \cdots + \sigma_r^2}$$

Applications in ML

Principal Component Analysis

PCA is SVD of the centered data matrix. The right singular vectors are the principal components.

Recommender Systems

Low-rank matrix factorization (like Netflix's algorithm) uses truncated SVD to find latent factors in user-item rating matrices.

Image Compression

An image is a matrix. Keeping only top $k$ singular values compresses the image while preserving major features.

Pseudoinverse

The Moore-Penrose pseudoinverse $A^+ = V\Sigma^+ U^T$ solves least-squares problems and handles non-square or rank-deficient matrices.

06. Singular Value Decomposition

SVD reveals that every linear transformation is just rotation → scaling → rotation. This decomposition is the foundation of dimensionality reduction, recommender systems, and image compression.
