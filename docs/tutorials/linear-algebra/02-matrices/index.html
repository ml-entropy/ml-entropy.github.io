02. Matrices and Linear Transformations

A matrix is not just a grid of numbers—it's a linear transformation in disguise. This perspective unlocks geometric intuition and deep understanding of matrix operations.

Theory Code Exercises

Linear Transformations

A function $T: \mathbb{R}^n \to \mathbb{R}^m$ is a linear transformation if it satisfies two properties:

Linearity Conditions

$$T(\mathbf{u} + \mathbf{v}) = T(\mathbf{u}) + T(\mathbf{v}) \quad \text{(additivity)}$$ $$T(c\mathbf{v}) = cT(\mathbf{v}) \quad \text{(homogeneity)}$$

These conditions mean: straight lines stay straight, the origin stays fixed, and parallel lines remain parallel.

Matrices as Transformations

Every linear transformation from $\mathbb{R}^n$ to $\mathbb{R}^m$ can be represented by an $m \times n$ matrix $A$. The transformation is simply matrix-vector multiplication:

$$T(\mathbf{x}) = A\mathbf{x}$$
Matrix-Vector Multiplication
1
Write the matrix $A$ as columns: $A = [\mathbf{a}_1 | \mathbf{a}_2 | \cdots | \mathbf{a}_n]$
2
Write $\mathbf{x} = (x_1, x_2, \ldots, x_n)^T$
3
Then: $A\mathbf{x} = x_1\mathbf{a}_1 + x_2\mathbf{a}_2 + \cdots + x_n\mathbf{a}_n$
Key Insight

The columns of $A$ are where the standard basis vectors land after transformation. The $j$-th column of $A$ is $T(\mathbf{e}_j)$.

Matrix Multiplication as Composition

If $A$ represents transformation $S$ and $B$ represents transformation $T$, then $AB$ represents the composition $S \circ T$ (apply $T$ first, then $S$).

$$(AB)\mathbf{x} = A(B\mathbf{x})$$

The $(i,j)$ entry of $AB$ is:

$$(AB)_{ij} = \sum_{k=1}^{n} A_{ik} B_{kj} = \text{(row } i \text{ of } A) \cdot \text{(column } j \text{ of } B)$$
Matrix Multiplication is Not Commutative

In general, $AB \neq BA$. Order matters! Rotating then scaling is different from scaling then rotating.

Special Matrices

Identity Matrix

The identity matrix $I$ leaves vectors unchanged: $I\mathbf{x} = \mathbf{x}$.

$$I = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix}$$

Inverse Matrix

If $A^{-1}$ exists, then $A^{-1}A = AA^{-1} = I$. The inverse "undoes" the transformation.

Transpose

The transpose $A^T$ swaps rows and columns: $(A^T)_{ij} = A_{ji}$.

$$(AB)^T = B^T A^T$$

Rank and Null Space

The column space of $A$ is the span of its columns—all possible outputs $A\mathbf{x}$.

The null space of $A$ is the set of all $\mathbf{x}$ where $A\mathbf{x} = \mathbf{0}$.

The rank of $A$ is the dimension of its column space.

Rank-Nullity Theorem

For any $m \times n$ matrix $A$: $$\text{rank}(A) + \dim(\text{null}(A)) = n$$
