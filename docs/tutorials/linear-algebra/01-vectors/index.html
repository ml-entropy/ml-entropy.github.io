What is a Vector?

In physics, a vector is an arrow with magnitude and direction. In computer science, it's an ordered list of numbers. But in linear algebra, a vector is simply any object that can be added to other vectors and scaled by numbers.

Vector in $\mathbb{R}^n$

A vector $\mathbf{v} \in \mathbb{R}^n$ is an ordered tuple of $n$ real numbers: $$\mathbf{v} = \begin{pmatrix} v_1 \\ v_2 \\ \vdots \\ v_n \end{pmatrix}$$

Vector Operations

Two fundamental operations define vector algebra:

Addition: Add component-wise

$$\mathbf{u} + \mathbf{v} = \begin{pmatrix} u_1 + v_1 \\ u_2 + v_2 \\ \vdots \\ u_n + v_n \end{pmatrix}$$

Scalar Multiplication: Scale each component

$$c\mathbf{v} = \begin{pmatrix} cv_1 \\ cv_2 \\ \vdots \\ cv_n \end{pmatrix}$$

Vector Spaces

A vector space is a set $V$ equipped with addition and scalar multiplication that satisfy specific axioms. This abstraction lets us apply the same techniques to polynomials, functions, matrices—anything that "behaves like vectors."

Vector Space Axioms
1
Closure: $\mathbf{u} + \mathbf{v} \in V$ and $c\mathbf{v} \in V$
2
Commutativity: $\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$
3
Associativity: $(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$
4
Zero vector: $\exists \mathbf{0}$ such that $\mathbf{v} + \mathbf{0} = \mathbf{v}$
5
Distributivity: $c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}$

Linear Combinations and Span

A linear combination of vectors $\mathbf{v}_1, \ldots, \mathbf{v}_k$ is:

$$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k$$

The span of these vectors is the set of all possible linear combinations:

$$\text{span}(\mathbf{v}_1, \ldots, \mathbf{v}_k) = \{c_1\mathbf{v}_1 + \cdots + c_k\mathbf{v}_k : c_i \in \mathbb{R}\}$$
Geometric Interpretation

The span of a single vector is a line through the origin. The span of two non-parallel vectors is a plane. Each new independent vector adds a dimension.

Linear Independence

Vectors $\mathbf{v}_1, \ldots, \mathbf{v}_k$ are linearly independent if the only solution to:

$$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k = \mathbf{0}$$

is $c_1 = c_2 = \cdots = c_k = 0$.

Key Insight

Linear independence means no vector in the set can be written as a combination of the others. Each vector contributes something "new" to the space.

Basis and Dimension

A basis for a vector space $V$ is a set of vectors that is:

  • Linearly independent
  • Spanning (every vector in $V$ is a linear combination of basis vectors)

The dimension of $V$ is the number of vectors in any basis.

$$\dim(\mathbb{R}^n) = n$$

The standard basis for $\mathbb{R}^3$ is:

$$\mathbf{e}_1 = \begin{pmatrix} 1 \\ 0 \\ 0 \end{pmatrix}, \quad \mathbf{e}_2 = \begin{pmatrix} 0 \\ 1 \\ 0 \end{pmatrix}, \quad \mathbf{e}_3 = \begin{pmatrix} 0 \\ 0 \\ 1 \end{pmatrix}$$

01. Vectors and Vector Spaces

What is a vector, really? Beyond arrows and lists of numbers, vectors are elements of vector spaces—structures that obey specific axioms. Understanding this abstraction is key to linear algebra.
